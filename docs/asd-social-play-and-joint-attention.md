# 社交游戏与共同注意训练实操手册

> 适用对象：自闭症康复训练师、早期干预老师、家长指导老师、融合教育老师、影子老师。  
> 使用场景：孩子不看人、叫名不应、只自己玩、不分享兴趣、不轮流、不模仿、不会假装游戏、不和同伴玩。  
> 核心目的：帮助新人训练师用游戏和生活互动训练儿童的社交动机、共同注意、轮流、模仿、假装游戏和同伴互动。  
> 重要提醒：社交训练不是逼孩子“看人”和“表演礼貌”，而是让孩子逐渐体验到：人是有趣的、互动是有用的、分享是快乐的。

---

## 一、先用一句话讲清楚社交训练

一句大白话：

> 社交训练不是让孩子机械看人，而是让孩子发现“和人在一起也很好玩”。

很多新人会把社交训练理解为：

- 看老师眼睛；
- 叫人；
- 说你好；
- 回答问题；
- 和小朋友坐一起。

这些只是表面。真正的社交核心是：

- 我愿意注意你；
- 我想让你知道我的想法；
- 我会回应你；
- 我会和你轮流；
- 我会分享快乐；
- 我知道你和我可能想得不一样。

---

## 二、共同注意是什么？

共同注意是ASD干预里非常核心的能力。

一句大白话：

> 共同注意就是孩子看到一个有趣的东西，不只是自己看，还会拉你一起看。

比如：

- 孩子看到飞机，指给妈妈看；
- 孩子搭好积木，拿给老师看；
- 孩子看到泡泡飞起来，先看泡泡，再看老师笑；
- 孩子听到小狗叫，转头看小狗，再看妈妈。

共同注意不是单纯要东西。

### 请求和共同注意的区别

| 行为 | 孩子的意思 |
|---|---|
| 指饼干，看妈妈 | 我要饼干 |
| 指飞机，看妈妈 | 妈妈你看飞机 |
| 递打不开的盒子 | 帮我打开 |
| 拿画给妈妈看 | 看我画的东西 |

请求是为了得到东西。  
共同注意是为了分享兴趣。

ASD孩子常常会请求，但不会分享。  
所以训练师要特别练“分享型互动”。

## 三、社交游戏训练的底层逻辑

孩子为什么不理人？常见原因有：

1. 人的表情、声音、动作太复杂；
2. 玩具比人更稳定、更可控；
3. 他不知道别人能帮他；
4. 他不知道互动有什么乐趣；
5. 他听不懂社交规则；
6. 他曾经在人际互动中反复失败；
7. 感觉过载或焦虑。

所以训练师不是强迫孩子社交，而是设计体验：

> 人能带来好玩的事；人能帮我；人会回应我；我和人可以来回玩。

---

## 四、社交训练的能力阶梯

### 阶段1：允许他人在旁边

目标：孩子不排斥老师靠近、旁观、拿同类玩具。

### 阶段2：回应他人

目标：孩子能回应叫名、接物、看人、跟随简单动作。

### 阶段3：主动找人

目标：孩子能主动拉人、递物、看人、请求、分享。

### 阶段4：来回互动

目标：孩子能轮流、等待、模仿、简单对话、共同完成游戏。

### 阶段5：同伴互动

目标：孩子能加入同伴、邀请同伴、遵守规则、处理冲突。

新人不要一上来要求阶段5。  
如果孩子还在阶段1，你先要让他接受你，而不是逼他和小朋友玩桌游。

---

## 五、社交训练黄金原则

### 原则1：先让人变得有趣

老师要成为“好事发生器”。

孩子喜欢泡泡，泡泡在老师手里；  
孩子喜欢车，老师让车飞出去；  
孩子喜欢挠痒，老师会玩挠痒游戏；  
孩子喜欢荡秋千，老师推他。

目的：

> 孩子开始觉得：找人有用，和人互动有意思。

### 原则2：少问问题，多制造互动

错误：

> “这是什么？这是什么颜色？你看我，你说呀。”

正确：

> “车来了，嘟嘟！”  
> “停！”  
> “哇，倒了！”  
> “轮到你。”

问题太多会让孩子觉得社交像考试。

### 原则3：跟随孩子兴趣

先进入孩子的玩法，再轻轻扩展。

如果孩子转轮子，不要马上抢走。  
可以先说：

> “轮子转转。”

然后加入：

> “我的轮子也转。”  
> “停！”  
> “车开了。”

### 原则4：互动要短、快、成功

初期互动不要太长。

- 看你1秒就强化；
- 轮流1轮就成功；
- 模仿1个动作就玩；
- 分享1次就回应。

### 原则5：不要强行掰脸看眼睛

眼神接触应该来自兴趣和需要，不应靠控制头部。

可以把有趣的东西放在自己眼睛附近，但不要强迫。

### 原则6：用夸张但不过载的表情

有些孩子喜欢夸张表情，有些孩子会被吓到。  
训练师要观察孩子反应。

### 原则7：把社交和自然强化绑在一起

孩子看人 → 游戏继续。  
孩子递物 → 老师打开。  
孩子轮流 → 小车开出去。  
孩子分享 → 大人惊喜回应。

## 六、叫名反应训练

### 1. 目标

孩子听到名字后，能转头、看人或用声音回应。

注意：  
叫名反应不是为了让孩子“服从”，而是为了建立安全和沟通入口。

### 2. SOP

1. 站在孩子侧前方，不要离太远；
2. 手里准备强强化物；
3. 叫名字一次；
4. 等1–2秒；
5. 孩子转头或看你，马上强化；
6. 没反应，用声音玩具或动作吸引，不要重复叫很多遍；
7. 成功后逐渐增加距离、换人、换环境。

### 3. 老师语言

> “小明。”  
> 孩子看。  
> “你看老师了，泡泡！”

### 4. 避坑

- 不要连续叫十几遍；
- 不要每次叫名后都是批评或任务；
- 不要掰脸；
- 不要在孩子专注高强度兴趣时硬叫很多次。

### 5. 进阶

- 安静环境叫名；
- 玩具中叫名；
- 远距离叫名；
- 户外叫名；
- 同伴叫名；
- 危险场景停下回应。

## 七、眼神互动训练

### 1. 目标

孩子愿意自然地看人，尤其是在请求、分享、等待、轮流时看人。

### 2. 不推荐

- “看我看我看我”；
- 用手掰下巴；
- 不看就不给任何回应；
- 长时间要求盯眼睛。

### 3. 推荐方法：物品靠近脸

SOP：

1. 拿孩子喜欢的泡泡；
2. 放在自己眼睛旁边；
3. 等孩子看过来；
4. 孩子看你或看物品靠近脸的方向，马上吹泡泡；
5. 逐渐减少物品靠脸。

### 4. 推荐方法：暂停等待

孩子正在玩喜欢的游戏时，老师突然停下。  
等待孩子看老师。  
一看，游戏继续。

例如：

> 推车——停——孩子看老师——老师说“开！”车继续。

### 5. 推荐方法：惊喜事件

制造有趣事件：

- 积木倒；
- 玩偶摔倒；
- 小车飞出去；
- 老师夸张打喷嚏；
- 泡泡飞到头上。

孩子看老师时，老师立刻回应：

> “哈哈，倒了！”

## 八、共同注意训练

### 1. 共同注意的三种基础形式

#### 看物再看人

孩子看泡泡，再看老师。

#### 指物给人看

孩子指飞机给妈妈看。

#### 拿物给人看

孩子拿作品给老师看。

### 2. SOP：用惊喜事件训练

1. 准备孩子感兴趣的活动，如泡泡或小车；
2. 制造惊喜：泡泡突然很多、小车撞倒积木；
3. 老师夸张说：“哇！”
4. 等孩子看老师；
5. 孩子一看，马上共享：“你也看到了！车撞倒了！”
6. 再重复。

### 3. SOP：户外分享

1. 看到小狗、飞机、灯、车等；
2. 老师先指：“看，小狗！”
3. 等孩子顺着看；
4. 再等孩子看回老师；
5. 孩子看回时，老师回应：“对，小狗跑！”

### 4. SOP：作品分享

1. 孩子搭好积木；
2. 老师不要只问“这是什么”；
3. 可以说：“哇，你搭好了，给妈妈看。”
4. 帮孩子把作品拿给家长；
5. 家长夸张回应：“我看到了，好高！”

## 九、轮流游戏训练

### 1. 目标

孩子能理解“你一次，我一次”。

### 2. 适合活动

- 滚球；
- 推车；
- 敲鼓；
- 搭积木；
- 投篮；
- 插玩具；
- 吹泡泡；
- 按按钮。

### 3. SOP

1. 选孩子喜欢但可以共享的活动；
2. 孩子先玩一次；
3. 老师说：“我一次。”快速玩一下；
4. 马上说：“你一次。”还给孩子；
5. 初期老师回合要非常短；
6. 成功2–3轮后强化；
7. 逐渐加入同伴。

### 4. 老师语言

> “你一次。”  
> “我一次。”  
> “轮到你。”  
> “轮到我。”

### 5. 避坑

- 不要拿孩子最不能分享的物品开始；
- 不要老师拿到后玩太久；
- 不要孩子一抢就完全放弃规则；
- 不要一开始就要求多人轮流。

## 十、模仿游戏训练

### 1. 为什么模仿重要？

大白话：

> 模仿是孩子学习别人的入口。

孩子会模仿，才能更容易学：

- 说话；
- 游戏；
- 生活动作；
- 同伴行为；
- 课堂规则。

### 2. 模仿阶梯

1. 模仿有物动作：推车、敲鼓、喂娃娃；
2. 模仿大动作：拍手、举手、跺脚；
3. 模仿精细动作：指、捏、摸头；
4. 模仿口部动作：张嘴、吹、伸舌；
5. 模仿声音；
6. 模仿同伴游戏。

### 3. SOP

1. 先模仿孩子，让孩子发现“老师在跟我一样”；
2. 孩子注意到后，老师做一个小变化；
3. 说：“这样做”；
4. 等孩子模仿；
5. 成功后立刻让游戏更好玩；
6. 不会就身体或示范提示；
7. 逐渐撤提示。

### 4. 示例：敲鼓

- 孩子敲鼓，老师也敲；
- 老师敲两下停；
- 孩子看老师；
- 老师说：“你敲”；
- 孩子敲，老师夸张回应。

## 十一、假装游戏训练

### 1. 假装游戏是什么？

大白话：

> 假装游戏就是把一个东西当成另一个东西，或者扮演生活里的角色。

例子：

- 假装喂娃娃；
- 假装打电话；
- 假装医生看病；
- 假装做饭；
- 积木当蛋糕；
- 盒子当车库。

### 2. 为什么重要？

假装游戏帮助孩子发展：

- 想象力；
- 语言；
- 社交角色；
- 情绪理解；
- 生活经验；
- 同伴游戏能力。

### 3. 假装游戏阶梯

1. 功能性动作：杯子喝水、勺子吃饭；
2. 对娃娃动作：喂娃娃、给娃娃盖被子；
3. 多步骤脚本：做饭→喂娃娃→擦嘴；
4. 角色扮演：医生、老师、司机；
5. 合作故事：你当医生，我当病人。

### 4. SOP：喂娃娃

1. 准备娃娃、碗、勺子；
2. 老师示范：“娃娃饿了，吃饭。”
3. 老师喂一口；
4. 递勺子给孩子：“你喂。”
5. 孩子喂后，老师让娃娃说：“好吃！”
6. 加入擦嘴、喝水、睡觉。

### 5. SOP：医生游戏

1. 准备听诊器、娃娃、创可贴；
2. 老师说：“娃娃生病了。”
3. 示范听一听；
4. 孩子模仿；
5. 贴创可贴；
6. 说“好了”；
7. 换角色：孩子当医生，老师当病人。

## 十二、同伴互动训练

### 1. 不要一开始就把孩子丢进大群体

很多ASD孩子在大群体中会：

- 躲开；
- 抢东西；
- 跑走；
- 不知道规则；
- 被忽视；
- 崩溃。

同伴训练要从小开始。

### 2. 同伴互动阶梯

1. 同处一个空间；
2. 平行玩；
3. 观察同伴；
4. 模仿同伴；
5. 和同伴轮流；
6. 简单合作；
7. 规则游戏；
8. 处理冲突。

### 3. 选择合适同伴

适合同伴：

- 温和；
- 愿意等待；
- 不抢；
- 能听老师引导；
- 对孩子友好。

不适合初期同伴：

- 攻击性强；
- 规则意识差；
- 爱嘲笑；
- 过度强势；
- 人数太多。

### 4. 两人滚球SOP

1. 两个孩子面对面坐；
2. 老师在旁边辅助；
3. A滚给B；
4. B滚给A；
5. 老师说：“给小明”；
6. 成功后夸双方；
7. 逐渐减少老师提示。

### 5. 合作搭积木SOP

1. 设定共同目标：“一起搭高塔”；
2. A放一块；
3. B放一块；
4. 老师提示“轮到谁”；
5. 塔倒时一起笑；
6. 完成后一起给家长看。

## 十三、社交故事和规则卡

社交故事大白话：

> 用简单图文告诉孩子一个社交场景里会发生什么、别人怎么想、我可以怎么做。

适合：

- 排队；
- 轮流；
- 分享；
- 输赢；
- 生日会；
- 上学；
- 看医生；
- 同伴拒绝。

### 示例：轮流社交故事

```text
我和小朋友一起玩球。
有时候轮到我。
有时候轮到小朋友。
轮到别人时，我可以等一等。
等到我时，我再玩。
我会说：轮到我了吗？
```

### 规则卡示例

```text
玩游戏规则：
1. 轮流
2. 等待
3. 手不抢
4. 说“轮到我”
```

## 十四、不同年龄社交训练重点

### 0–3岁

重点：

- 亲子互动；
- 看人；
- 回应名字；
- 轮流声音；
- 模仿动作；
- 简单游戏。

活动：

- 躲猫猫；
- 挠痒痒；
- 泡泡；
- 滚球；
- 唱儿歌。

### 3–6岁

重点：

- 功能性游戏；
- 假装游戏；
- 轮流；
- 共同注意；
- 同伴平行玩到合作玩。

活动：

- 娃娃家；
- 医生游戏；
- 小车轨道；
- 积木合作；
- 简单桌游。

### 6–12岁

重点：

- 规则游戏；
- 话题轮替；
- 处理输赢；
- 加入同伴；
- 理解别人想法；
- 识别欺负。

活动：

- 桌游；
- 运动；
- 小组任务；
- 角色扮演；
- 社交情景剧。

### 12岁以上

重点：

- 友谊边界；
- 青春期社交；
- 网络安全；
- 拒绝与求助；
- 同伴压力；
- 职业社交。

活动：

- 情景讨论；
- 安全社交脚本；
- 购物问路；
- 面试练习；
- 团体合作任务。

## 十五、社交目标怎么写？

### 错误写法

```text
提高社交能力。
多看人。
会和小朋友玩。
```

### 正确模板

```text
在________活动中，当________发生时，儿童能________，在____次机会中完成____次，连续____节课达到。
```

### 示例1：共同注意

```text
在泡泡游戏中，当老师吹出泡泡并停顿时，儿童能在5次机会中至少3次看向老师或指向泡泡进行分享，连续3节课达到。
```

### 示例2：轮流

```text
在滚球游戏中，儿童能与成人完成至少3轮“你一次我一次”，手势提示不超过2次，连续3节课达到。
```

### 示例3：同伴互动

```text
在双人积木活动中，儿童能在成人提示下与同伴轮流放积木，每轮等待不抢，完成5轮中的4轮，连续3次小组课达到。
```

### 示例4：假装游戏

```text
在娃娃家游戏中，儿童能模仿并独立完成喂娃娃、喝水、盖被子3个假装动作中的2个，连续3节课达到。
```

## 十六、社交游戏数据怎么记？

### 1. 共同注意记录表

## 十七、家长版解释话术

### 1. 解释共同注意

```text
共同注意就是孩子看到有趣的东西，会想让你也看到。
比如看到飞机，他不只是自己看，而是指给你看、看你一眼。
这说明他开始把快乐和别人分享，这是社交很重要的基础。
```

### 2. 解释不要强迫眼神

```text
我们不是用手掰孩子看人。
我们会让互动变有趣，让孩子因为想继续玩、想分享，而自然地看向大人。
这样的眼神才有意义。
```

### 3. 解释同伴训练不能太急

```text
孩子如果还不会轮流、等待、模仿，直接放进一群小朋友里，可能只会更挫败。
我们会先从一对一、短时间、简单游戏开始，再慢慢过渡到小组。
```

### 4. 家庭练习建议

```text
家里每天不用练很久。
您可以抓3个小游戏：泡泡停下来等他看你；滚球时你一次我一次；看到小狗或车时指给他看，再等他看你。
重点是让他体验到：和人一起玩很有意思。
```

## 十八、常见错误

### 错误1：把社交训练变成眼神训练

只盯着看眼睛，忽略互动意义。

改法：  
在请求、分享、轮流中自然诱发看人。

### 错误2：问题太多

一直问“这是什么”，孩子像被考试。

改法：  
多描述、多等待、多制造惊喜。

### 错误3：一开始就要求和同伴玩

孩子基础不够，容易失败。

改法：  
先成人互动，再一个同伴，再小组。

### 错误4：老师主导太强

完全按老师想法玩，孩子没兴趣。

改法：  
先跟随孩子，再扩展。

### 错误5：只练请求，不练分享

孩子会要东西，但不会给你看。

改法：  
加入共同注意目标。

### 错误6：忽略孩子感觉负荷

环境太吵、人太多，孩子无法社交。

改法：  
先调环境，再训练互动。

### 错误7：同伴选择不合适

同伴抢、催、笑，孩子更抗拒社交。

改法：  
选择温和、可引导同伴。

## 二十二、新人必须记住的10句话

1. 社交训练不是逼孩子看人。  
2. 先让人变得有趣，孩子才愿意靠近人。  
3. 共同注意是分享，不是要东西。  
4. 少问考试题，多创造互动。  
5. 先跟随孩子兴趣，再轻轻扩展。  
6. 轮流从一对一、短回合开始。  
7. 模仿是学习别人行为的入口。  
8. 假装游戏是语言和社交的重要桥梁。  
9. 同伴互动不能操之过急，要搭台阶。  
10. 真正的社交目标是让孩子体验到“和人一起也很好”。

## 二十三、最后给新人训练师的话

对很多ASD孩子来说，人是复杂的，玩具是简单的；  
人会变化，玩具更可控；  
人会提要求，玩具不会让他紧张。

所以我们不能一上来就要求孩子“会社交”。  
我们要先成为一个让孩子觉得安全、有趣、可预测的人。

当孩子第一次在泡泡飞起时回头看你，  
第一次把积木塔拿给你看，  
第一次愿意把球滚回来，  
第一次和同伴一起笑，  
那不是一个小动作，  
那是孩子开始把你带进他的世界。

这就是社交训练最珍贵的起点。
