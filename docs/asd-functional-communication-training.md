# 功能沟通训练FCT实操手册

> 适用对象：自闭症康复训练师、语言训练老师、家长指导老师、影子老师、融合教育老师。  
> 使用场景：儿童不会表达需求，或经常用哭闹、尖叫、打人、抢东西、逃跑、躺地、自伤等方式表达“我要、不要、帮我、休息、太吵了”。  
> 核心目的：帮助训练师教会孩子用更安全、更清楚、更有效的方式沟通，从源头减少问题行为。  
> 重要提醒：FCT是行为支持和沟通训练的重要方法，但严重自伤、伤人、逃跑、吞物、癫痫样表现或疑似精神症状，必须同步上报督导并建议专业评估。

---

## 一、先用一句话讲清楚FCT

FCT，全称 Functional Communication Training，常译为“功能沟通训练”。

一句大白话：

> FCT就是教孩子用更合适的方式，说出他原本想用问题行为表达的意思。

比如：

| 以前孩子怎么表达 | 实际想表达 | 我们教的新表达 |
|---|---|---|
| 尖叫 | 我要继续玩 | “还要”/递还要卡 |
| 打人 | 我不想做了 | “休息”/递休息卡 |
| 抢零食 | 我想吃 | “我要饼干”/指图片 |
| 跑出教室 | 这里太吵 | “太吵了”/拿耳机卡 |
| 躺地 | 任务太难 | “帮我”/“简单一点” |

FCT的核心不是让孩子“别闹”，而是：

> 给孩子一条比闹更快、更省力、更安全的沟通路。

---

## 二、为什么问题行为常常是沟通？

很多ASD孩子不是没有需求，而是不会用别人听得懂的方式表达。

当孩子不会说：

- “我想要”；
- “我不要”；
- “我不舒服”；
- “我害怕”；
- “太吵了”；
- “我需要帮忙”；
- “我想休息”；

他就可能用身体和行为表达：

- 哭；
- 叫；
- 打；
- 抢；
- 跑；
- 躺；
- 推；
- 咬；
- 扔。

训练师要记住：

> 问题行为很多时候不是孩子的坏脾气，而是孩子当前最熟练的语言。

我们的任务是帮他换一种语言。

## 三、FCT适合解决哪些问题？

### 1. 为了获得东西

表现：

- 抢玩具；
- 哭着要手机；
- 拉大人去拿零食；
- 尖叫要求继续动画。

教：

- “我要”；
- “还要”；
- “打开”；
- “给我”；
- “我想玩手机5分钟”。

### 2. 为了逃避任务

表现：

- 一写作业就哭；
- 一上桌面就躺地；
- 扔材料；
- 打老师；
- 跑开。

教：

- “休息”；
- “帮我”；
- “太难了”；
- “少一点”；
- “我想等一下”。

### 3. 为了获得关注

表现：

- 大人聊天时尖叫；
- 故意扔东西；
- 跑开让大人追；
- 不停打断。

教：

- “看我”；
- “陪我玩”；
- 轻拍肩膀；
- 举手；
- 递“找妈妈”卡。

### 4. 为了感觉需求或逃避刺激

表现：

- 捂耳尖叫；
- 冲出嘈杂教室；
- 拒绝理发洗头；
- 撞墙寻求压力；
- 不停跳。

教：

- “太吵了”；
- “我要耳机”；
- “我要出去”；
- “我要压一压”；
- “我要跳一跳”。

## 四、FCT的核心逻辑

FCT不是单独教一句话，而是一个完整替换过程。

### 旧路

```text
孩子想要手机 → 尖叫 → 家长给手机 → 尖叫变多
```

### 新路

```text
孩子想要手机 → 递手机卡/说“手机” → 家长给1分钟手机 → 合适表达变多
```

核心规则：

> 旧行为不再直接有效，新表达马上有效。

如果孩子尖叫仍然最快拿到手机，他就没有理由学新表达。

---

## 五、FCT实施五步法

### 第一步：找出问题行为功能

先用ABC记录判断：

- 他是在要东西？
- 逃避任务？
- 要关注？
- 感觉需求？

没有功能分析，FCT容易教错。

例子：孩子打人。

可能意思不同：

| 场景 | 打人的意思可能是 |
|---|---|
| 老师拿出作业 | 我不想做 |
| 同伴拿他车 | 这是我的/我要车 |
| 妈妈打电话 | 看我 |
| 教室很吵 | 我想离开 |

不同功能，要教不同表达。

### 第二步：选择替代表达方式

替代表达必须：

1. 孩子做得到；
2. 比问题行为更省力；
3. 大人能马上理解；
4. 能得到类似结果；
5. 方便在多个场景使用。

表达方式可以是：

- 眼神；
- 伸手；
- 指物；
- 递物；
- 手势；
- 图片卡；
- 沟通板；
- 单词；
- 短句；
- 打字；
- 电子AAC设备。

AAC大白话解释：

> AAC就是辅助沟通工具，比如图片、手势、沟通板、平板语音按钮。它不是放弃说话，而是先给孩子一个能表达的工具。

### 第三步：在孩子刚想要时教

不要等孩子已经爆炸再教。

最好的时机：

- 孩子看向想要物；
- 伸手；
- 拉大人；
- 皱眉；
- 快要哭但还没爆；
- 任务开始有点抗拒；
- 环境刚变吵。

### 第四步：新表达后马上满足

刚开始一定要让新表达非常有效。

孩子递“休息卡” → 马上休息。  
孩子说“帮我” → 马上帮。  
孩子指“耳机” → 马上给耳机。  
孩子说“还要” → 马上继续。

### 第五步：逐渐提高要求

等孩子稳定后，再慢慢提高：

- 从递卡到说词；
- 从一个词到短句；
- 从马上满足到等待；
- 从每次都满足到按规则满足；
- 从一个场景到多个场景。

## 六、替代表达怎么选？

### 1. 无语言或语言很少的孩子

可以先用：

- 指；
- 递图片；
- 拿物品给大人；
- 手势；
- 简单声音。

目标不是一开始说完整句，而是先让孩子知道：

> 我可以用一个动作让别人明白我。

### 2. 有单词的孩子

可以教：

- 要；
- 不要；
- 帮；
- 开；
- 停；
- 走；
- 休息；
- 还要。

### 3. 有短句的孩子

可以教：

- 我要____；
- 我不要____；
- 请帮我____；
- 我要休息；
- 太吵了；
- 我想离开；
- 再等一下。

### 4. 高语言但情绪差时说不出的孩子

可以同时使用：

- 情绪卡；
- 休息卡；
- 文字卡；
- 手机备忘录；
- 手势；
- 约定暗号。

提醒：  
会说话的孩子也可能需要AAC。因为崩溃时语言系统可能暂时“掉线”。

## 八、FCT案例一：孩子尖叫要东西

### 场景

孩子看到饼干，拿不到就尖叫。

### ABC

| A | B | C |
|---|---|---|
| 饼干在柜子上，孩子够不到 | 尖叫、拉家长 | 家长给饼干 |

功能：获得饼干。

### 替代表达

根据能力选择：

- 指饼干；
- 递“饼干”图片；
- 说“饼”；
- 说“我要饼干”。

### SOP

1. 饼干放在看得见够不着的位置；
2. 孩子看或伸手时，马上提示：“饼干”；
3. 孩子指/递卡/说词；
4. 立刻给一小块；
5. 重复5–10次；
6. 逐渐减少提示。

### 注意

不要孩子尖叫很久才教。  
要在尖叫前或刚开始时抓住机会。

---

## 九、FCT案例二：孩子打人逃避任务

### 场景

一拿出作业，孩子就打老师。

### ABC

| A | B | C |
|---|---|---|
| 老师拿出配对卡 | 孩子打老师手 | 老师把卡片收走，让孩子休息 |

功能：逃避任务。

### 替代表达

- 递“休息”卡；
- 说“休息”；
- 说“帮我”；
- 说“太难了”；
- 举手表示暂停。

### SOP

1. 任务开始前准备休息卡；
2. 任务量降到很小，比如做2个；
3. 孩子开始烦躁时，提示递休息卡；
4. 递卡后马上休息30秒；
5. 休息后回来做1个简单任务；
6. 再给休息或强化；
7. 后续逐渐增加任务量。

### 注意

不能让“打人”直接取消全部任务。  
但可以让“休息卡”有效。

---

## 十、FCT案例三：孩子跑出教室

### 场景

教室吵或任务难时，孩子冲出门。

### ABC

| A | B | C |
|---|---|---|
| 教室声音大/老师要求做题 | 孩子冲向门口 | 老师追出去，孩子离开教室 |

可能功能：逃避环境、逃避任务，或获得追逐关注。

### 替代表达

- “出去”；
- “休息”；
- “太吵了”；
- “我要安静角”；
- 递离开卡。

### SOP

1. 安全优先，门口设安全管理；
2. 在孩子平静时教“出去/休息”卡；
3. 发现孩子看门、站起、烦躁时，马上提示递卡；
4. 递卡后带去指定安静区1–2分钟；
5. 安静区结束后回到任务或较低要求；
6. 记录跑离次数和递卡次数。

### 注意

如果孩子冲马路、冲出机构，需要安全计划和人员分工，不是单靠FCT即可。

---

## 十一、FCT案例四：孩子捂耳尖叫

### 场景

商场、教室、吹风机声音大时尖叫。

### 功能

逃避声音刺激。

### 替代表达

- “太吵”；
- “耳机”；
- “停”；
- “我要出去”；
- 捂耳+递耳机卡。

### SOP

1. 提前准备耳机和“太吵”卡；
2. 进入噪音环境前预告；
3. 孩子刚捂耳时，提示“太吵”；
4. 孩子说/指后，马上给耳机或带到安静处；
5. 等稳定后短时间再进入；
6. 逐步增加停留时间。

### 注意

不要在孩子明显痛苦时逼他忍。  
先让他有表达和保护自己的工具。

---

## 十二、FCT案例五：孩子通过扔东西要关注

### 场景

妈妈和别人说话时，孩子扔杯子。

### 功能

获得关注。

### 替代表达

- 轻拍妈妈；
- 递“妈妈看我”卡；
- 说“看我”；
- 举手；
- 说“我有事”。

### SOP

1. 妈妈聊天前给孩子一个任务或玩具；
2. 提前说：“有事轻拍妈妈”；
3. 练习轻拍后妈妈马上回应；
4. 孩子扔东西时低反应，安全处理；
5. 引导孩子轻拍或递卡；
6. 用合适方式叫妈妈后，妈妈马上给关注。

### 注意

孩子安静等待时，要主动给关注。  
不要只有闹的时候才看他。

## 十三、图片沟通怎么做？

### 1. 准备图片

先准备最常用的：

- 我要；
- 还要；
- 打开；
- 帮我；
- 休息；
- 不要；
- 太吵；
- 耳机；
- 厕所；
- 喝水；
- 妈妈；
- 老师。

图片要：

- 清楚；
- 大小合适；
- 方便拿；
- 可以贴魔术贴；
- 放在孩子拿得到的位置。

### 2. 初学阶段不要放太多

一次给1–3张即可。  
太多会让孩子不知道选什么。

### 3. 递卡教学SOP

1. 孩子想要泡泡；
2. 老师把“泡泡”卡放在孩子面前；
3. 轻提示孩子拿卡递给老师；
4. 老师接卡，马上说“泡泡”；
5. 立刻吹泡泡；
6. 逐渐减少手部提示；
7. 后续增加“我要+泡泡”。

### 4. 图片不是终点

图片可以逐步升级为：

- 图片+声音；
- 图片+单词；
- 两张图片组合；
- 短句条；
- 口语；
- 电子AAC。

## 十四、沟通板怎么设计？

### 1. 初级沟通板

```text
[我要] [不要] [帮我]
[休息] [喝水] [厕所]
[还要] [打开] [太吵]
```

### 2. 家庭常用沟通板

```text
[吃] [喝水] [手机]
[玩具] [出去] [睡觉]
[洗澡] [厕所] [妈妈]
[不要] [帮我] [休息]
```

### 3. 学校常用沟通板

```text
[老师] [同学] [铅笔]
[书] [厕所] [喝水]
[休息] [帮我] [太吵]
[我不会] [等一下] [完成]
```

### 4. 沟通板使用原则

- 放在孩子随时能拿到的地方；
- 大人要示范使用；
- 孩子使用后必须回应；
- 不要只在训练时拿出来；
- 不要把沟通板当考试题。

## 十五、AAC会不会影响说话？

很多家长担心：

> 用图片或设备，孩子是不是更不说话？

训练师可以这样解释：

```text
AAC不是放弃说话，而是先给孩子一个能表达的工具。
当孩子发现沟通有用，他的沟通动机会增加。
我们使用图片时，也会同步说出口语，比如孩子递“水”卡，我们马上说“水”，再给水。
这样图片反而可以帮助语言发展。
```

重点：

- 图片和口语一起用；
- 不逼孩子在崩溃时说话；
- 有声音就鼓励声音；
- 逐渐从图片过渡到词语和句子；
- 对长期口语困难者，AAC本身也是重要沟通权利。

## 十六、FCT训练中的强化原则

FCT中，新表达刚开始必须非常有效。

### 1. 立即回应

孩子递“休息卡”，马上休息。  
孩子说“打开”，马上打开。

### 2. 高成功率

初期不要频繁拒绝孩子的新表达。  
否则他会觉得：

> 说了也没用，那我还是哭吧。

### 3. 后期再加入规则

等新表达稳定后，再教：

- 等待；
- 轮流；
- 不是每次都可以；
- 可以换一个；
- 先做后要；
- 延迟满足。

### 4. 问题行为不直接有效

如果孩子尖叫，尽量不要直接给最终物。  
等稍微平静，提示合适表达，再给。

## 十七、如何从“马上满足”过渡到“按规则满足”

FCT初期：

```text
孩子说休息 → 马上休息
```

后期要逐步变成：

```text
孩子说休息 → 做完1个 → 休息
孩子说手机 → 等1分钟 → 手机
孩子说还要 → 轮到你时再玩
```

### 过渡阶梯

#### 阶段1：每次表达都满足

目标：让孩子相信新表达有效。

#### 阶段2：短等待

表达后等1–5秒。

#### 阶段3：先小任务后满足

表达后说：

> “先放1个，再休息。”

#### 阶段4：选择替代

不能给手机时：

> “手机没有，可以选小车或积木。”

#### 阶段5：规则化

例如：

> “每天手机10分钟。”  
> “作业后玩。”  
> “超市只买1样。”

注意：  
不要在孩子刚学会表达时马上大量拒绝。否则旧行为会回来。

## 十九、FCT目标怎么写？

### 模板

```text
在________场景中，当________发生时，儿童能通过________表达________，在____次机会中独立完成____次，连续____天/节课达到，同时目标问题行为下降到____。
```

### 示例1：请求

```text
在零食场景中，当儿童想继续吃但零食袋关闭时，儿童能通过递“还要”卡或说“还要”请求继续，5次机会中独立完成4次，连续3天达到，同时尖叫次数下降到每天不超过1次。
```

### 示例2：休息

```text
在桌面任务中，当儿童出现烦躁、推材料等早期信号时，儿童能递“休息”卡或说“休息”，5次机会中完成4次，连续3节课达到，同时打人行为下降50%。
```

### 示例3：感觉逃避

```text
在嘈杂环境中，当儿童感到声音不适时，儿童能指“太吵”卡或说“太吵了”，并请求耳机或离开，3次机会中完成2次，连续2周达到。
```

## 二十、FCT常见错误

### 错误1：没有先分析功能

孩子打人是为了逃避任务，你却教他说“我要玩具”，当然没用。

改法：  
先做ABC，再选表达。

### 错误2：替代表达太难

孩子不会说话，你要求他说完整句。

改法：  
先用图片、手势、单词。

### 错误3：新表达不够快

孩子递卡后，大人半天不回应。  
孩子会回到尖叫。

改法：  
初期新表达必须马上有效。

### 错误4：问题行为仍然更有效

孩子哭闹马上得手机，说“手机”反而要等。  
那孩子当然继续哭闹。

改法：  
让合适表达比问题行为更有效。

### 错误5：刚学会就频繁拒绝

孩子刚学会“还要”，大人总说“不可以”。

改法：  
初期多满足，后期再教等待和规则。

### 错误6：沟通工具拿不到

图片卡放在柜子里，孩子需要时拿不到。

改法：  
沟通工具必须随手可得。

### 错误7：只在训练室用

家里、学校不用，孩子无法泛化。

改法：  
同一张卡、同一句话，多场景使用。

### 错误8：忽略孩子早期信号

等孩子已经崩溃才教。

改法：  
在皱眉、推开、看门、伸手时就提示表达。

## 二十一、家长版解释话术

### 1. 解释FCT

```text
孩子现在不是没有需求，而是不会用合适方式说出来。
功能沟通训练就是教他用图片、手势或语言表达，比如“我要”“休息”“帮我”。
当这些新表达变得有效，哭闹和打人的必要性就会下降。
```

### 2. 解释为什么要先满足

```text
刚开始孩子学新表达，我们要让他马上发现：这样说有用。
等他稳定会用了，我们再慢慢教等待、轮流和规则。
如果一开始说了也没用，他就会回到原来的哭闹方式。
```

### 3. 解释图片不是倒退

```text
图片不是倒退，而是给孩子一个表达工具。
我们会一边让他用图片，一边配合口语示范。
目标是让他先能表达，再逐步提高到声音、词语或句子。
```

### 4. 解释家长要一致

```text
如果孩子哭闹时能得到东西，但用卡片时得不到，他会选择哭闹。
所以全家要尽量一致：合适表达马上回应，问题行为不直接满足。
```

## 二十五、新人必须记住的10句话

1. 问题行为常常是孩子当前最熟练的沟通方式。  
2. FCT不是压行为，而是教新表达。  
3. 没有功能分析，就容易教错表达。  
4. 替代表达必须比问题行为更容易。  
5. 新表达刚开始必须马上有效。  
6. 图片、手势、AAC不是倒退，而是沟通工具。  
7. 会说话的孩子，崩溃时也可能需要辅助沟通。  
8. 等孩子爆炸后再教，通常太晚。  
9. 家庭、学校、机构要一致回应。  
10. 最终目标是让孩子用安全、清楚、被理解的方式表达自己。

## 二十六、最后给新人训练师的话

孩子每一次哭闹、尖叫、抢夺、逃跑，背后都可能藏着一句没有说出口的话：

> “我想要。”  
> “我不要。”  
> “我太难受。”  
> “我不会。”  
> “请帮帮我。”

FCT就是把这些藏在行为里的话，翻译成别人能听懂的沟通方式。

当孩子第一次不用哭，而是递出“休息卡”；  
第一次不用打人，而是说“帮我”；  
第一次不用尖叫，而是说“太吵了”；  
那一刻，他不是只学会了一个词，  
而是学会了：

> 我的感受可以被理解；  
> 我的需求可以被回应；  
> 我有比崩溃更好的办法。

这就是功能沟通训练真正的价值。
