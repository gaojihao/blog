#ABC行为记录与行为功能分析入门手册

适用对象：自闭症康复训练师、助教、影子老师、家长指导老师、班主任、机构督导。
使用场景：孩子发脾气、尖叫、打人、逃跑、拒绝任务、自伤、抢东西、刻板行为明显影响学习生活时。
核心目的：帮助新人从“孩子不听话”的模糊判断，升级为“看懂行为前后发生了什么，并教会孩子更好的替代方式”。
重要提醒：本手册用于教育康复和行为支持，不替代医生、临床心理师、精神科或行为分析师的专业诊疗。涉及严重自伤、伤人、逃跑、吞食异物、癫痫样表现、长期睡眠饮食异常等情况，应立即上报督导并建议家长寻求专业评估。

#一、先用一句话讲清楚ABC

ABC不是英文字母练习。

一句大白话：

ABC就是观察孩子“行为前发生了什么、孩子做了什么、行为后得到了什么”。

三个字母分别是：

字母 英文 大白话
A Antecedent 行为前发生了什么
B Behavior 孩子具体做了什么
C Consequence 行为后发生了什么

最简单的例子：

A 行为前 B 行为 C 行为后
妈妈让孩子关手机写作业 孩子尖叫、摔笔、躺地 妈妈说“算了，今天不写了”

这个行为可能的功能是：

逃避写作业。

再看一个例子：

A 行为前 B 行为 C 行为后
货架上有薯片，家长说不买 孩子哭闹、拉家长、坐地上 家长买了薯片

这个行为可能的功能是：

获得想要的东西。

#二、行为功能是什么？

一句大白话：

行为功能就是这个行为帮孩子“得到了什么”或“逃开了什么”。

孩子反复出现的行为，通常不是随机的。它往往在帮孩子解决一个问题。

常见四大功能：

  1. 获得物品或活动：想要手机、零食、玩具、出去玩；
  2. 逃避任务或环境：不想写、不想洗澡、不想上课、不想去吵的地方；
  3. 获得关注：想让大人看他、追他、骂他、抱他；
  4. 获得感觉刺激或减少感觉不适：跳、转、拍手、咬物、捂耳、逃离声音。

训练师要牢牢记住：

不先判断功能，就急着处理行为，等于没看地图就开车。

#三、为什么不能只说“问题行为”？

“问题行为”这个词太粗了。

同样是尖叫，功能可能完全不同。

同样表现：尖叫 行为前 行为后 可能功能
情况1 老师拿出作业 老师暂停作业 逃避任务
情况2 看见饼干但拿不到 大人给饼干 获得物品
情况3 大人在聊天没理他 大人立刻转头说他 获得关注
情况4 教室突然放音乐 大人带他离开 逃避声音刺激

所以训练师不能看到尖叫就统一处理。
你要问：

这次尖叫前发生了什么？尖叫后孩子得到了什么？

#四、ABC记录的标准写法

#1. A：行为前发生了什么

A不是写“上午”或“教室”。
A要写具体触发事件。

常见A包括：

  • 大人提出要求；
  • 大人拒绝孩子；
  • 喜欢的东西被拿走；
  • 活动从喜欢切换到不喜欢；
  • 等待时间太长；
  • 环境太吵；
  • 人太多；
  • 任务太难；
  • 没有人关注；
  • 同伴靠近或抢东西；
  • 身体不舒服；
  • 睡眠不足。

错误写法：

A：上课。

正确写法：

A：老师收走小车，并说“坐下做配对”。

#2. B：孩子具体做了什么

B必须写得像摄像头拍到的一样。

不要写评价词：

  • 不乖；
  • 闹；
  • 崩溃；
  • 不配合；
  • 故意；
  • 发疯。

要写可观察行为：

  • 尖叫；
  • 哭；
  • 推桌子；
  • 咬手；
  • 打老师手臂；
  • 跑到门口；
  • 趴在地上；
  • 把卡片扔到地上；
  • 重复说“不要不要”。

错误写法：

B：孩子不听话，乱闹。

正确写法：

B：孩子尖叫约30秒，双手推开桌子，把3张卡片扫到地上，然后趴在地上。

#3. C：行为后发生了什么

C要写所有人的反应和孩子得到的结果。

常见C包括：

  • 大人安慰；
  • 大人批评；
  • 大人抱起；
  • 任务停止；
  • 给了玩具；
  • 给了手机；
  • 带离现场；
  • 同伴退开；
  • 老师追过去；
  • 家长妥协；
  • 大家都看他；
  • 孩子独处；
  • 环境变安静。

错误写法：

C：好了。

正确写法：

C：老师把任务拿走，让孩子去垫子休息2分钟。孩子停止尖叫。

#六、四大行为功能详解

#功能一：获得物品或活动

#1. 本质

一句大白话：

孩子发现“我这样做，大人就会给我想要的东西”。

常见想要的东西：

  • 手机；
  • 零食；
  • 玩具；
  • 出门；
  • 抱抱；
  • 看动画；
  • 坐电梯；
  • 玩水；
  • 重复某个活动。

#2. 典型ABC

A B C
孩子看到手机，家长说不能玩 孩子尖叫、抢手机、打家长 家长把手机给他，让他安静

初步功能：获得手机。

#3. 训练策略

核心原则：

不让问题行为直接换到东西，要教孩子用合适方式要。

#SOP：教替代请求

第一步：提前准备替代表达。

根据孩子能力选择：

  • 眼神；
  • 指物；
  • 递图片；
  • 说“要”；
  • 说“我要手机”；
  • 说“我想玩5分钟”。

第二步：制造小机会。

把孩子想要的东西放在看得见、够不着的位置。

第三步：等待1–3秒。

看孩子是否看你、伸手、发声。

第四步：提示合适表达。

你说：

“说：要。”
“指这个。”
“给我‘我要’卡。”

第五步：孩子一表达，马上给一小部分。

例如：

“你说要了，给你玩1分钟。”

第六步：逐渐增加要求。

从“要”到“我要手机”，再到“我想玩5分钟”。

#4. 千万别这样做

  • 孩子一尖叫就给；
  • 孩子抢成功后不处理；
  • 一开始要求他说很长句；
  • 完全不给任何替代机会；
  • 在孩子爆炸时讲大道理。

#5. B计划

如果孩子已经爆发：

  1. 先保证安全；
  2. 物品不直接给；
  3. 等孩子声音或动作下降一点；
  4. 给最简单替代表达：指一下、递卡、说一个“要”；
  5. 表达后给少量或延迟满足。

例如：

“你想要手机。先给妈妈‘手机卡’，然后看1分钟。”

#功能二：逃避任务或环境

#1. 本质

一句大白话：

孩子发现“我这样做，讨厌的事情就不用做了”。

常见逃避对象：

  • 写作业；
  • 桌面任务；
  • 洗澡；
  • 刷牙；
  • 剪头发；
  • 上学；
  • 排队；
  • 等待；
  • 嘈杂环境；
  • 失败感；
  • 被纠正。

#2. 典型ABC

A B C
老师说“做配对” 孩子把卡片扔地上、躺地 老师取消任务，让孩子自由玩

初步功能：逃避任务。

#3. 训练策略

核心原则：

任务可以变简单、变短、可请求休息，但不能让问题行为直接取消全部要求。

#SOP：任务降阶+休息请求

第一步：把任务变小。

不要说：

“把这20张都做完。”

改成：

“做2个，休息。”

第二步：用先后板。

写或画：

先配对2张 → 后玩小车。

第三步：教“休息”表达。

可以用：

  • 休息卡；
  • 举手；
  • 说“休息”；
  • 说“我想休息”。

第四步：孩子用合适方式请求，给短休息。

例如：

“你说休息，可以休息1分钟。”

第五步：休息结束后回到任务。

这是关键。休息不是永远逃掉，而是恢复后继续一点点。

#4. 千万别这样做

  • 孩子一哭就完全取消任务；
  • 任务太难还硬压；
  • 不给休息方式；
  • 一边骂一边要求做；
  • 孩子崩溃时继续加任务。

#5. B计划

如果孩子已经拒绝严重：

  1. 降低任务难度；
  2. 从“做10个”改成“碰一下/放1个”；
  3. 完成最小任务后立刻休息；
  4. 下一次从更低难度开始预防。

例如：

“好，现在只放一个。放完休息。”

#功能三：获得关注

#1. 本质

一句大白话：

孩子发现“我这样做，大人马上看我、说我、追我”。

注意：
关注不一定是表扬。批评、追赶、训斥，有时对孩子来说也是关注。

常见关注型行为：

  • 故意扔东西看大人反应；
  • 在大人聊天时尖叫；
  • 跑开让大人追；
  • 不停打断；
  • 做奇怪动作逗别人；
  • 轻轻打人后看对方表情。

#2. 典型ABC

A B C
妈妈在和别人说话，没有看孩子 孩子把杯子推到地上 妈妈立刻转头大声说“你怎么又这样”并收拾

初步功能：获得关注。

#3. 训练策略

核心原则：

问题行为少给关注，合适行为多给关注。

这不是冷漠，而是把关注给到正确的方式。

#SOP：预防性关注+教合适打断

第一步：提前给关注。

不要等孩子闹了才看他。
每隔几分钟主动关注一次：

“你在搭积木，我看到了。”
“你安静等妈妈，真好。”

第二步：教合适打断。

给孩子一个方式：

  • 轻拍妈妈手臂；
  • 递“找妈妈”卡；
  • 说“妈妈看我”；
  • 举手。

第三步：练习。

妈妈假装聊天，孩子轻拍手臂。
妈妈马上转头：

“你这样叫妈妈，妈妈就知道了。”

第四步：问题行为时减少戏剧化反应。

例如孩子扔小物：

  • 保持低反应；
  • 简短说：“杯子放桌上。”
  • 引导他重新放好；
  • 放好后给关注。

#4. 千万别这样做

  • 大喊大叫；
  • 追着孩子跑；
  • 长篇批评；
  • 孩子一闹全家围过来；
  • 平时安静时完全不理他。

#5. B计划

如果孩子通过危险行为要关注：

  1. 安全优先，立即阻止危险；
  2. 语言保持短；
  3. 不进行长篇训斥；
  4. 危险结束后教替代方式；
  5. 增加平时的正向关注。

#功能四：感觉刺激或感觉逃避

#1. 本质

一句大白话：

孩子通过某些动作让身体舒服一点，或者逃离让身体难受的刺激。

两种类型：

#感觉寻求

孩子想要更多身体感觉。

表现：

  • 跳；
  • 转；
  • 拍手；
  • 撞；
  • 咬；
  • 闻；
  • 盯光；
  • 听重复声音。

#感觉逃避

孩子想远离不舒服的刺激。

表现：

  • 捂耳；
  • 逃离；
  • 拒绝剪发；
  • 拒绝刷牙；
  • 不穿某些衣服；
  • 进入商场崩溃；
  • 吃饭只吃固定口感。

#2. 典型ABC

#感觉寻求例子

A B C
在走廊等待，无任务 孩子原地跳、拍手 身体获得运动感觉，没人打断

初步功能：获得感觉刺激。

#感觉逃避例子

A B C
教室突然很吵 孩子捂耳尖叫，跑到门口 老师带他离开教室

初步功能：逃避噪音。

#3. 训练策略

核心原则：

不一定消灭感觉行为，而是让它安全、合适、可控。

#SOP：感觉寻求的处理

第一步：判断危险。

  • 马路边跳：必须停止并替代；
  • 教室后排轻微摆手：可能不需要立刻打断；
  • 撞墙撞人：必须干预。

第二步：安排合法出口。

例如：

  • 跳床10下；
  • 推墙10秒；
  • 搬重物；
  • 捏球；
  • 深压抱枕；
  • 走感统路线。

第三步：建立规则。

“红区不跳，绿区可以跳。”
“先走到门口，再跳10下。”

第四步：教替代动作。

如果不能跳，可以：

  • 双手按压；
  • 脚趾抓地；
  • 手插兜握拳；
  • 捏减压球。

#SOP：感觉逃避的处理

第一步：承认不舒服。

“这里太吵了，你不舒服。”

第二步：给工具。

  • 降噪耳机；
  • 帽子；
  • 太阳镜；
  • 安静角；
  • 休息卡；
  • 提前预告。

第三步：逐步适应。

不要一下把孩子丢进最刺激环境。
从低强度、短时间开始。

第四步：教表达。

“太吵了。”
“我要出去。”
“我要耳机。”

#4. 千万别这样做

  • 所有拍手跳跃都禁止；
  • 孩子捂耳还逼他忍；
  • 公开羞辱：“别人都看你了”；
  • 感觉过载时继续上任务；
  • 用惩罚处理身体不适。

#5. B计划

如果感觉行为影响课堂：

  1. 先安排短感觉休息；
  2. 回来做一个很短任务；
  3. 再给感觉活动；
  4. 逐渐增加任务时间。

例如：

“跳10下，回来放2个积木，再跳10下。”

#七、如何判断行为功能：五步法

#第一步：收集至少3–5次ABC

不要只凭一次判断。

如果每次都发生在任务出现后，可能是逃避。
如果每次都发生在想要东西拿不到时，可能是获得物品。
如果每次都发生在没人理他时，可能是关注。
如果没有明显外部诱因，且行为本身带来身体舒服，可能是感觉。

#第二步：找共同点

问自己：

  • 行为前总出现什么？
  • 行为后总得到什么？
  • 什么场景最容易发生？
  • 什么人面前最容易发生？
  • 什么时间最容易发生？
  • 什么情况几乎不发生？

#第三步：提出功能假设

用“可能”而不是“肯定”。

示例：

根据3次ABC记录，孩子的躺地行为多发生在桌面任务开始后，行为后任务通常暂停或降低难度，因此初步假设功能为逃避任务。

#第四步：设计验证

如果你认为是逃避任务，可以试：

  • 降低任务难度；
  • 给先后板；
  • 教休息请求；
  • 任务完成一点后再休息。

如果行为明显下降，说明假设可能正确。

#第五步：持续记录

行为支持不是一次完成。
要持续看：

  • 频率有没有下降；
  • 持续时间有没有缩短；
  • 强度有没有降低；
  • 替代行为有没有增加；
  • 是否能泛化到家里和学校。

#八、行为干预的四件套

一个完整的行为支持计划，不能只写“不要让他哭”。

必须包含四件事：

  1. 前事调整：行为发生前怎么预防；
  2. 替代行为教学：教孩子用什么新方式；
  3. 后果策略：行为发生后大人怎么一致处理；
  4. 数据记录：怎么判断有没有变好。

#九、前事调整：在爆炸前拆掉炸弹

前事调整大白话：

不等孩子炸了再灭火，而是提前减少触发点。

#1. 如果孩子逃避任务

前事调整：

  • 降低任务难度；
  • 减少数量；
  • 给选择；
  • 用先后板；
  • 穿插简单任务；
  • 先强化再教学；
  • 允许短休息。

例子：

“先做2个，再玩车。”

#2. 如果孩子要东西

前事调整：

  • 提前说明规则；
  • 使用等待卡；
  • 给选择；
  • 把高动机物品作为教学机会；
  • 不让孩子随意抢到。

例子：

“手机在这里。你可以说‘我要手机’，然后玩2分钟。”

#3. 如果孩子要关注

前事调整：

  • 定时关注；
  • 给等待任务；
  • 教打断方式；
  • 在孩子安静时多互动；
  • 避免长时间忽视。

例子：

“妈妈先讲电话，你拿这张卡，想找妈妈就轻拍我。”

#4. 如果孩子感觉过载

前事调整：

  • 提前预告；
  • 降低声音；
  • 避开高峰期；
  • 准备耳机；
  • 安排休息区；
  • 缩短停留时间。

例子：

“超市10分钟，戴耳机。买完牛奶就出来。”

#十、替代行为教学：给孩子一条新路

只禁止旧行为，不教新行为，孩子会很痛苦。

#替代行为必须满足3个条件

  1. 孩子做得到；
  2. 比问题行为更省力；
  3. 能快速得到类似结果。

例如，孩子不会说话，你却要求他说：

“妈妈我现在非常想休息一下。”

这不现实。

可以先教：

  • 递休息卡；
  • 指“休息”；
  • 说“休”；
  • 举手。

#常见替代行为表

问题行为 可能功能 替代行为
尖叫要手机 获得物品 递手机卡/说“手机”
打人逃避作业 逃避任务 递休息卡/说“休息”
扔东西引人看 获得关注 轻拍大人/说“看我”
冲出教室 逃避环境 递“出去”卡/请求安静角
抢同伴玩具 获得物品 说“轮到我”/递交换物
捂耳尖叫 逃避噪音 说“太吵”/拿耳机
撞墙 感觉寻求 推墙/抱枕深压/跳床

#十一、后果策略：行为后大人怎么做

后果策略不是惩罚。
它的核心是：

问题行为不要再有效，替代行为马上有效。

#1. 对获得物品型

问题行为后：

  • 不立刻给完整物品；
  • 等孩子稍微平静；
  • 提示合适请求；
  • 合适请求后给少量或限时。

#2. 对逃避任务型

问题行为后：

  • 不完全取消任务；
  • 任务降到最小；
  • 完成一点点；
  • 再给休息。

#3. 对关注型

问题行为后：

  • 危险行为先阻止；
  • 非危险轻微行为降低反应；
  • 合适寻求关注时立刻回应；
  • 平时主动给关注。

#4. 对感觉型

问题行为后：

  • 如果安全，可允许短暂调节；
  • 如果危险，立刻替代为安全感觉活动；
  • 教“我需要休息/耳机/跳一跳”。

#十二、差别强化：新手也要懂的核心技术

差别强化大白话：

你想让哪个行为变多，就把好结果给哪个行为；你不想让哪个行为变多，就不要让它继续得到好处。

#1. 强化替代行为

孩子不用打人，而是说“休息”。
马上给休息。

#2. 强化不相容行为

不相容行为就是两个行为不能同时做。

例如：

  • 双手拿球，就不能打人；
  • 坐椅子，就不能躺地；
  • 嘴巴吹泡泡，就不能尖叫。

#3. 强化低频率

如果孩子一天尖叫20次，先目标降到15次，再降到10次。
不要一开始要求0次。

#4. 强化其他行为

在一段时间里没有出现问题行为，就给强化。

例如：

“你安静等了1分钟，给你贴纸。”

#十三、消退：为什么不能乱用

消退大白话：

以前问题行为能得到好处，现在不再给这个好处。

例如：

  • 以前哭就给手机；
  • 现在哭不给，只有说“手机”才给。

#消退的风险

刚开始行为可能变得更严重。
这叫消退爆发。大白话：

旧办法突然不灵了，孩子会更用力试一试。

比如以前尖叫30秒就有手机，现在没有，他可能尖叫5分钟、扔东西、打人。

#新人注意

不要单独乱用消退。必须同时做到：

  1. 有安全计划；
  2. 有替代行为；
  3. 家长和老师一致；
  4. 督导知情；
  5. 数据记录；
  6. 行为危险可控。

涉及自伤、攻击、逃跑时，不建议新人自行使用消退策略。

#十四、危机行为现场处理SOP

危机行为包括：

  • 自伤；
  • 攻击；
  • 逃跑；
  • 扔重物；
  • 砸玻璃；
  • 冲马路；
  • 长时间崩溃。

#第一步：安全

  • 移开危险物；
  • 拉开其他孩子；
  • 保护头部；
  • 堵住危险出口；
  • 必要时求助同事；
  • 不围观。

#第二步：降刺激

  • 降低声音；
  • 减少语言；
  • 关掉音乐；
  • 减少人群；
  • 给空间；
  • 不强行对视。

#第三步：短句支持

只说短句：

“安全。”
“我在。”
“先停。”
“坐这里。”
“可以休息。”

不要说：

“你为什么这样？”
“你再这样我就不要你了。”
“你看看别人都在看你。”

#第四步：等待下降

当孩子从10分降到6分，再做选择：

“喝水还是坐垫子？”
“耳机还是休息卡？”
“这里坐，还是那里坐？”

#第五步：事后记录和复盘

孩子完全恢复后，才复盘。
先记录：

  • 时间；
  • 地点；
  • 前因；
  • 行为；
  • 持续时间；
  • 是否受伤；
  • 谁参与处理；
  • 后续建议。

#十五、常见场景案例分析

#案例1：孩子一做桌面任务就躺地

ABC

A B C
老师拿出配对卡，说“坐下做” 孩子躺地、闭眼、踢腿 老师说“那先休息吧”，任务取消

初步功能

逃避任务。

支持方案

  1. 任务从10题降到2题；
  2. 使用先后板：先2题，后小车；
  3. 教“休息”卡；

ABC

A B C
看到薯片，妈妈说不买 孩子坐地哭叫 妈妈怕尴尬，买了薯片

获得物品。

支持方案

  1. 出门前写购物清单；
  2. 规定“今天可以买1样”；
  3. 教选择：“饼干还是牛奶”；
  4. 哭闹时不立即买；
  5. 平静后让孩子用“我要这个”表达，再按规则购买或放入下次清单；

ABC

A B C
妈妈接电话，没看孩子 孩子尖叫、拉妈妈衣服 妈妈挂电话骂孩子

初步功能

获得关注。

支持方案

  1. 电话前给孩子任务：拼图/贴纸；
  2. 给“找妈妈”卡；
  3. 教轻拍妈妈手；
  4. 孩子合适打断，妈妈回应一句：“我看到你了，等1分钟。”
  5. 尖叫时减少语言反应；
  6. 安静等待30秒就强化。

#案例4：孩子出门边走边跳、拍手

ABC

A B C
出门走路，环境开阔或兴奋 孩子跳、拍手 身体获得感觉，家长反复喊“别跳”

初步功能

感觉寻求，也可能包含兴奋表达。

支持方案

  1. 区分危险场景和安全场景；
  2. 建立红区绿区：马路电梯不跳，公园空地可跳；
  3. 路上说：“走20步，跳10下”；
  4. 给替代动作：捏球、手按压、背小包；
  5. 不用羞辱语言。

#案例5：孩子听到吹风机就尖叫逃跑

ABC

A B C
家长打开吹风机 孩子捂耳尖叫、跑出浴室 家长关掉吹风机

初步功能

逃避声音刺激。

支持方案

  1. 先承认：“声音大，不舒服”；
  2. 用毛巾擦干替代一部分；
  3. 准备耳罩；
  4. 吹风机先在远处开1秒，马上关；
  5. 强化孩子能待住；
  6. 慢慢增加距离和时间；
  7. 教“太吵了”“我要停一下”。

#十七、数据怎么判断行为有没有改善

不要只问：

“孩子今天乖不乖？”

要看数据。

#1. 频率下降

以前每天10次，现在每天4次。

#2. 持续时间缩短

以前每次30分钟，现在每次5分钟。

#3. 强度降低

以前打人、砸东西，现在只是皱眉、说不要。

#4. 恢复更快

以前需要大人哄很久，现在能去安静角3分钟恢复。

#5. 替代行为增加

以前用尖叫要东西，现在能递卡或说“要”。

#6. 泛化成功

不只在机构有效，在家、学校、外出也逐渐有效。

#十八、家长版解释话术

#1. 解释ABC

我们不是只看孩子发脾气，而是看发脾气前后发生什么。
如果每次一写作业就发脾气，最后作业取消，孩子就可能学会“发脾气可以不用写”。
所以我们要教他一个更好的方式，比如说“休息一下”，而不是靠哭闹解决。

#2. 解释不是纵容

接住情绪不等于满足所有要求。
我们可以理解孩子难受，但规则还是要有。
区别是：我们不让哭闹直接成功，而是让合适表达成功。

#3. 解释为什么要记录

记录不是为了挑孩子毛病，而是为了找规律。
只要找到规律，我们就能提前预防，而不是每次等他爆发后再处理。

#4. 解释替代行为

如果孩子现在只会用尖叫表达，我们不能只说“不许叫”。
我们要给他一个新工具，比如图片、手势或一个简单词。
新工具有效了，旧行为才会慢慢少。

#十九、新人常见错误

#错误1：记录太模糊

错误：

孩子闹了。

正确：

老师说“收玩具”后，孩子尖叫约2分钟，并把小车扔到地上。

#错误2：只记录B，不记录A和C

只写孩子做了什么，无法判断功能。
ABC必须一起写。

#错误3：凭一次行为下结论

一次只能提供线索。
至少看3–5次规律。

#错误4：只想消灭行为，不教替代方式

孩子需要新工具。
否则他只会换一种问题行为。

#错误5：大人处理不一致

妈妈坚持，奶奶妥协；老师坚持，家长妥协。
孩子会学会找最容易妥协的人。

#错误6：任务太难还怪孩子逃避

逃避行为很多时候说明任务设计不合理。
先检查任务难度和强化是否足够。

#错误7：忽略感觉和身体原因

孩子可能不是故意闹，而是太吵、太痛、太累、太饿、便秘、经期不适。

#错误8：危险行为还慢慢分析

冲马路、自伤、攻击时，先安全，再分析。

#错误9：把消退当万能

消退可能引发更大爆发。新人不能独立乱用。

#错误10：没有复盘

行为计划不是写完就结束。
每周至少看一次数据，必要时调整。

#二十二、最后给新人训练师的话

问题行为不是孩子的“坏”，而是孩子现在手里唯一会用的工具。
你的任务不是把这个工具硬抢走，而是教他一个更安全、更清楚、更被社会接受的新工具。

记住这句话：

先看懂，再介入；先预防，再处理；先教新路，再堵旧路。
