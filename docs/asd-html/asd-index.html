#自闭症康复训练师成长资料库

这是一套面向自闭症康复训练师的系统学习资料库。
目标不是只培养“会陪玩”的老师，而是培养能从新人逐步成长为独立训练师、个案负责人、高级训练师、督导，最终具备专家级系统思维的专业人员。

新人会“跟着做”，高手会“自己设计”，专家会“看系统、带团队、控风险、建体系”。

#二、第一阶段：新人入门，先看懂孩子

适合：0–30天新人、助教、刚入行训练师。

  1. 自闭症康复训练师新人入门手册
  2. 儿童发展里程碑速查表：训练师版
  3. 自闭症个案接案访谈表与首次观察记录模板
  4. 职业伦理、安全边界与专业规范手册
  5. 康复训练师30-90-180天成长路径

学习目标：

  • 会观察；
  • 会写具体行为；
  • 会找孩子兴趣；
  • 会守安全边界；
  • 不乱承诺、不越界。

#三、第二阶段：基础教学技术

适合：1–3个月新人、基础训练师。

  1. 强化物使用与代币系统实操手册
  2. 提示与提示撤销实操手册
  3. DTT回合式结构化教学实操手册
  4. NET自然情境教学实操手册
  5. 数据记录与进展报告实操手册

学习目标：

  • 会下短指令；
  • 会强化；
  • 会提示和撤销；
  • 会做基础DTT；
  • 会做基础NET；
  • 会记录数据。

#四、第三阶段：沟通、社交与行为

适合：3–6个月训练师、开始独立带基础个案的老师。

  1. 功能沟通训练FCT实操手册
  2. 沟通层级与AAC辅助沟通实操手册
  3. 社交游戏与共同注意训练实操手册
  4. ABC行为记录与行为功能分析入门手册
  5. 情绪与挑战性行为干预实操手册

学习目标：

  • 会教请求、拒绝、帮我、休息；
  • 会使用图片、沟通板、AAC思路；
  • 会训练共同注意和轮流；
  • 会做ABC分析；
  • 会处理常见情绪行为。

#五、第四阶段：生活、感觉、学校、家庭

适合：独立训练师、家长指导老师、影子老师。

  1. 生活自理与独立生活技能训练手册
  2. 感觉处理与感觉调节支持手册
  3. 融合教育与学校支持实操手册
  4. 家长培训与家庭泛化实操手册

学习目标：

  • 会教洗手、穿衣、如厕、整理；
  • 会识别感觉敏感和感觉寻求；
  • 会做学校支持；
  • 会把机构技能泛化到家庭。

#六、第五阶段：评估、IEP与个案管理

适合：个案协作训练师、个案负责人。

  1. 个案报告与IEP目标撰写实操手册
  2. 标准化评估工具导览与目标转化手册
  3. 个案概念化与综合方案设计手册
  4. 新人训练师典型案例库

学习目标：

  • 会写SMART目标；
  • 会写初评和阶段报告；
  • 会把评估转成IEP；
  • 会做个案概念化；
  • 会用案例训练分析思维。

#七、第六阶段：青春期、共患、危机和转衔

适合：高级训练师、个案负责人、督导。

  1. 青春期支持、职业转衔与成年准备手册
  2. 共患问题识别、红旗信号与转介手册
  3. 复杂行为危机管理与安全计划手册
  4. 高级案例督导库

学习目标：

  • 会支持青春期身体、情绪和安全；
  • 会识别ADHD、焦虑、睡眠、便秘、癫痫等红旗；
  • 会制定高风险安全计划；
  • 会分析复杂案例。

#八、第七阶段：专家级循证、督导、协作和体系建设

适合：高级训练师、督导、机构教学主管、课程负责人。

  1. 从新人到专家：ASD康复训练师能力地图与缺口分析
  2. 新人到专家级完整文档路线图
  3. 循证实践与伪科学辨别手册
  4. 研究阅读与持续学习手册
  5. 家校医社多方协作手册
  6. 督导带教与训练师质量控制手册
  7. 机构课程体系建设与运营质控手册
  8. 训练师能力等级评估与考核标准
  9. 专家级阅读清单与资源地图

学习目标：

  • 会辨别循证与伪科学；
  • 会阅读研究和指南；
  • 会做多方协作；
  • 会带教新人；
  • 会做机构质控；
  • 会建立训练师等级考核；
  • 会持续更新专业体系。

#十一、最后提醒

这套资料现在已经覆盖从新人到专家级的主要路径。

但要真正成为专家，还必须配合：

真实个案实践
+ 高质量督导
+ 视频复盘
+ 数据追踪
+ 家长沟通
+ 学校协作
+ 医学/心理转介意识
+ 持续阅读
+ 伦理边界

资料是地图，实践和督导才是路。
