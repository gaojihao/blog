#融合教育与学校支持实操手册

适用对象：自闭症康复训练师、影子老师、班主任、特教老师、家长、学校支持人员。
使用场景：孩子入园、入学、课堂适应、课间活动、午餐午休、作业、考试、同伴互动、校园行为支持。
核心目的：帮助新人把康复训练和学校真实场景连接起来，让孩子能在普通班、融合班或特教支持环境中更稳定地参与学习生活。
重要提醒：融合教育不是把孩子“放进去就算完成”，而是需要目标、支持、调整和协作。

#一、先用一句话讲清楚融合教育

一句大白话：

融合教育就是让孩子在尽可能合适的支持下，和同龄人一起学习和生活。

融合不是：

  • 把孩子送进普通班就结束；
  • 让孩子自己硬撑；
  • 让老师和同学“习惯就好”；
  • 让孩子每天都在失败和崩溃里硬扛。

融合应该是：

  • 有准备；
  • 有支持；
  • 有调整；
  • 有目标；
  • 有协作；
  • 有记录；
  • 有逐步退位计划。

#二、为什么学校支持特别重要？

孩子在学校面对的不是一对一训练，而是：

  • 集体指令；
  • 时间变化；
  • 噪音；
  • 同伴互动；
  • 规则；
  • 作业；
  • 排队；
  • 午餐午休；
  • 考试；
  • 被评价。

如果没有适当支持，孩子可能：

  • 坐不住；
  • 逃跑；
  • 崩溃；
  • 被同伴排斥；
  • 学习掉队；
  • 出现问题行为；
  • 拒学。

学校支持的核心目标是：

让孩子能参与，而不是只是“在场”。

#三、融合教育的三层支持

#1. 环境支持

让环境更容易理解和承受。

包括：

  • 视觉日程；
  • 固定座位；
  • 安静角；
  • 规则卡；
  • 课间安排；
  • 过渡预告；
  • 降噪耳机；
  • 任务清单。

#2. 教学支持

让任务更容易学会。

包括：

  • 拆步；
  • 提示；
  • 示范；
  • 代币；
  • 分层作业；
  • 口语/图片辅助；
  • 同伴帮助。

#3. 行为支持

让问题行为不再是唯一办法。

包括：

  • FCT功能沟通；
  • 情绪调节；
  • 休息卡；
  • 先后板；
  • 安全计划；
  • 数据记录。

#四、入园入学前要准备什么

#1. 了解孩子当前能力

至少要知道：

  • 语言理解；
  • 主动表达；
  • 叫名反应；
  • 模仿；
  • 轮流等待；
  • 如厕；
  • 饮食；
  • 感觉特点；
  • 问题行为；
  • 安全风险；
  • 分离焦虑。

#2. 了解学校环境

  • 班级人数；
  • 是否有融合班；
  • 老师支持程度；
  • 课间和午休安排；
  • 厕所位置；
  • 安静区域；
  • 校园边界；
  • 特殊活动。

#3. 准备支持材料

  • 视觉日程；
  • 规则卡；
  • 休息卡；
  • 先后板；
  • 沟通板；
  • 代币板；
  • 紧急联系人卡；
  • 行为支持计划。

#五、入学目标怎么定

不要只问“能不能跟上”。

要分成四类：

#1. 参与目标

孩子能否：

  • 坐进教室；
  • 听到铃声后回到座位；
  • 跟随集体指令；
  • 参加活动；
  • 跟上过渡。

#2. 学习目标

孩子能否：

  • 完成桌面任务；
  • 认识常见材料；
  • 识别自己的名字；
  • 做简单作业；
  • 理解课堂规则。

#3. 社交目标

孩子能否：

  • 回应老师；
  • 和同伴轮流；
  • 加入游戏；
  • 请求帮助；
  • 分享兴趣。

#4. 自理安全目标

孩子能否：

  • 如厕；
  • 吃饭；
  • 穿脱衣；
  • 收拾书包；
  • 课间安全；
  • 跟随安全规则。

#六、影子老师怎么做

影子老师不是替孩子做所有事。

一句大白话：

影子老师是“脚手架”，不是“代替孩子的大脑”。

#1. 影子老师的任务

  • 帮孩子理解规则；
  • 减少焦虑；
  • 提示独立完成；
  • 保护安全；
  • 帮老师和家长沟通；
  • 记录支持效果。

#2. 不该做什么

  • 替孩子回答一切；
  • 替孩子写完作业；
  • 一直坐太近；
  • 一直提醒；
  • 代替老师管理全部行为；
  • 让孩子对影子老师形成依赖。

#3. 退位原则

影子老师应该逐步减少帮助：

  • 从身体提示到手势提示；
  • 从手势提示到视觉提示；
  • 从近距离到远距离；
  • 从每步帮助到关键点帮助；
  • 从全程陪到局部陪。

#七、课堂结构支持

#1. 进教室

  • 固定进门流程；
  • 提前看视觉日程；
  • 先坐哪里；
  • 书包放哪里；
  • 第一件事做什么。

#2. 上课中

  • 材料清楚；
  • 指令短；
  • 任务分小；
  • 有完成标记；
  • 有短休息。

#3. 课间

  • 课间不是失控时间；
  • 要教轮流和边界；
  • 选择安全活动；
  • 避免高冲突区域；
  • 可以安排安静角或感觉活动。

#4. 午餐午休

  • 午餐流程图；
  • 固定座位；
  • 吃完收拾；
  • 午休前预告；
  • 安静过渡；
  • 短休息工具。

#八、课堂常见问题怎么处理

#1. 坐不住

可能原因：

  • 任务太长；
  • 座位太难受；
  • 感觉过载；
  • 不会等待；
  • 没有动机。

支持：

  • 缩短任务；
  • 给短休息；
  • 座位调整；
  • 提供感觉工具；
  • 用代币和自然强化。

#2. 叫名不应

支持：

  • 从近到远；
  • 从安静到嘈杂；
  • 配合强化物；
  • 不要连续叫很多遍；
  • 叫名后给好结果。

#3. 发脾气或躺地

支持：

  • 先判断功能；
  • 提供休息卡；
  • 降低任务难度；
  • 保持规则一致；
  • 事后教替代沟通。

#4. 逃跑

支持：

  • 安全边界；
  • 门口监控；
  • 过渡预告；
  • 离开卡；
  • 安静角；
  • 风险个案上报。

#5. 同伴冲突

支持：

  • 练轮流；
  • 练交换；
  • 练说“轮到我”；
  • 练求助老师；
  • 选择友好同伴。

#2. 过渡卡

示例：

  • 还有1分钟；
  • 收拾；
  • 下一节；
  • 轮到你；
  • 休息；
  • 回座位。

#3. 规则卡

示例：

  • 手不打人；
  • 脚走路；
  • 嘴巴说；
  • 排队等；
  • 问老师；
  • 先做后玩。

#4. 安全卡

  • 我迷路了；
  • 我要找老师；
  • 我不舒服；
  • 太吵了；
  • 我要休息。

#十、午餐午休支持

#1. 午餐

  • 提前洗手；
  • 固定位置；
  • 固定餐具；
  • 先少量；
  • 允许请求更多或不要；
  • 吃完收拾。

#2. 午休

  • 睡前预告；
  • 视觉流程；
  • 允许安抚物；
  • 安静角；
  • 不把午休当惩罚；
  • 不强迫孩子马上睡着。

#3. 常见问题

  • 不躺床；
  • 不能安静；
  • 找同伴说话；
  • 起床困难；
  • 噪音敏感。

支持：

  • 先坐安静；
  • 再躺；
  • 再闭眼；
  • 逐步延长安静时间。

#十一、作业和考试支持

#1. 作业支持

  • 分段；
  • 减量；
  • 计时器；
  • 完成勾选；
  • 视觉清单；
  • 先易后难；
  • 完成后强化。

#2. 考试支持

  • 考前预告；
  • 座位安排；
  • 降低干扰；
  • 理解题目格式；
  • 允许适度延时；
  • 必要时按学校规定做合理调整。

#3. 家庭作业沟通

  • 家长别一次盯太多；
  • 每次只抓一小目标；
  • 记录孩子是不会、不会开始，还是会但不愿意；
  • 不把作业变成冲突战场。

#十二、同伴支持怎么做

#1. 选择同伴

优先找：

  • 温和；
  • 能等待；
  • 愿意示范；
  • 不抢；
  • 不嘲笑。

#2. 同伴可做什么

  • 轮流；
  • 示范；
  • 共同游戏；
  • 简单提问；
  • 邀请；
  • 一起完成任务。

#3. 不要让同伴做什么

  • 不是小老师全责；
  • 不要让同伴管理孩子行为；
  • 不要让同伴暴露孩子困难；
  • 不要让同伴成为“监工”。

#4. 同伴训练小目标

  • 我可以跟你玩吗？
  • 轮到你了。
  • 给我一下。
  • 一起做。
  • 你先，我后。

#十三、校园行为支持计划

#1. 行为目标要具体

不要写：

减少捣乱。
提高配合度。

要写：

当老师要求收材料时，儿童能在视觉提示下将材料放入盒中，10次中完成8次，连续3天达到。

#2. 计划应包括

  • 前因预防；
  • 替代行为；
  • 后果策略；
  • 危机处理；
  • 记录方式；
  • 家校一致话术。

#3. 复盘频率

  • 每天简单反馈；
  • 每周一次数据回看；
  • 每月一次目标调整。

#十四、与班主任沟通怎么做

#1. 核心原则

  • 说事实，不说标签；
  • 说具体行为，不说“性格”；
  • 说可执行的支持，不说空话；
  • 说孩子的优势，不只说困难。

#2. 沟通模板

老师您好，这个孩子在安静、结构清晰的环境里表现会更稳定。
他目前最需要的是：提前预告、短指令、视觉日程和短休息。
如果他开始烦躁，建议先让他用休息卡或安静角，而不是连续催促。
我们会同步在机构和家里保持一致。

#3. 不要这样说

  • “他就是不听话。”
  • “你们老师太严了。”
  • “只要你照我说的做就行。”

#4. 应该这样说

  • “这可能和任务切换有关。”
  • “我们先看前因。”
  • “这个支持在学校也可以试。”
  • “我们一起记录一周看看。”

#十五、学校安全与走失预防

#必要措施

  • 认识教室和厕所位置；
  • 认识老师和办公室；
  • 记住姓名和联系方式；
  • 设定校园边界；
  • 练求助句；
  • 需要时使用定位或家校联络。

#求助脚本

我迷路了。
我要找老师。
请帮我联系妈妈。

#风险个案

如果有冲门、冲马路、离校走失史，应制定单独安全计划。

#十六、融合教育中的目标怎么定

#参与目标

  • 进入教室；
  • 回到座位；
  • 参加小组；
  • 完成过渡；
  • 坐住短时间。

#学习目标

  • 跟随1–2步指令；
  • 完成工作篮；
  • 交作业；
  • 认识学具；
  • 回答问题。

#社交目标

  • 回应名字；
  • 请求帮助；
  • 轮流；
  • 邀请同伴；
  • 分享展示。

#自理目标

  • 如厕；
  • 吃饭；
  • 穿脱衣；
  • 整理书包；
  • 准备学习用品。

#十七、学校目标怎么写？

#错误写法

适应学校。
提升课堂表现。

#正确模板

在________课堂场景中，当________发生时，儿童能通过________完成________，在____次机会中独立完成____次，连续____天达到。

#示例

过渡目标

在铃声响起后，儿童能在视觉提示下从课间回到座位，5次中完成4次，连续3天达到。

课堂指令目标

在小组课中，当老师说“把书打开”时，儿童能在手势提示下完成动作，10次中完成8次，连续3节课达到。

同伴互动目标

在课间游戏中，儿童能在成人提示下与同伴完成至少2轮轮流，连续3次活动达到。

#十九、家长版解释话术

#1. 解释为什么需要学校支持

孩子在学校面对的刺激很多，不是只靠他自己硬适应就能解决。
我们会把训练里学到的东西搬到学校里，让他在真实环境中也能稳定参与。

#2. 解释影子老师不是“盯人”

影子老师不是替孩子做，而是帮他慢慢学会自己做。
好的影子老师最终是逐渐退位，让孩子越来越独立。

#3. 解释为什么要视觉支持

很多孩子听语言容易漏掉，视觉信息更稳定。
所以日程表、规则卡、休息卡会帮助他更好地理解学校。

#4. 解释为什么要家校一致

如果学校和家里的规则不一致，孩子会更混乱。
我们尽量统一关键规则，比如先做后玩、休息卡、过渡预告、安静角。

#二十、新人常见错误

#错误1：把融合理解成“放进去”

改法：
融合是有支持、有调整、有目标的参与。

#错误2：影子老师陪太紧

改法：
逐步退位，保留关键支持。

#错误3：只看学业，忽略参与

改法：
先看孩子是否能在班级里稳定参与。

#错误4：课间和午休不管

改法：
这两个环节往往最容易出问题。

#错误5：只处理课堂，不处理同伴

改法：
把同伴互动当作训练目标。

#错误6：不记录

改法：
每周看一次参与、行为和支持效果。

#错误7：没有安全计划

改法：
走失、冲门、攻击、崩溃都要预案。

#错误8：家校不同步

改法：
统一视觉卡和话术。

#二十四、新人必须记住的10句话

  1. 融合不是放进去就完事。
  2. 孩子在学校需要支持，不是只靠忍耐。
  3. 影子老师是脚手架，不是替身。
  4. 课堂、课间、午餐、午休都要支持。
  5. 视觉工具能显著降低混乱。
  6. 安全和过渡是高风险点。
  7. 同伴支持要设计，不是碰运气。
  8. 家校一致比单点努力更重要。
  9. 数据记录能帮助及时调整。
  10. 融合的目标是参与、学习和尊严。

#二十五、最后给新人训练师的话

学校对很多ASD孩子来说，是复杂、快节奏、充满变化的地方。
如果没有支持，孩子很容易在这里反复失败。

你的工作不是让孩子“硬撑过去”，
而是帮助学校和家庭一起搭桥：

  • 让规则更清楚；
  • 让过渡更可预测；
  • 让任务更可完成；
  • 让同伴互动更友好；
  • 让孩子越来越能独立参与。

真正好的融合，不是把孩子塞进环境，
而是让环境也学会照顾孩子。
