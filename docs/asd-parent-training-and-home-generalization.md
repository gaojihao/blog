# 家长培训与家庭泛化实操手册

> 适用对象：自闭症康复训练师、个案负责人、督导、家长指导老师。  
> 使用场景：课后反馈、家庭作业、家长焦虑沟通、家庭训练计划、机构技能泛化到家里、家校一致支持。  
> 核心目的：帮助新人从“只会教孩子”升级为“会教家长”，让训练成果真正回到家庭生活里。  
> 重要提醒：家长培训不是批评家长，而是赋能家长；不是把训练压力全部丢给家庭，而是给家长能做到的小步骤。

---

## 一、先用一句话讲清楚家长培训

一句大白话：

> 家长培训就是把老师会的关键方法，变成家长在家能做、愿意做、做得成的小动作。

如果孩子只在机构会：

- 说“要”；
- 等待；
- 轮流；
- 收玩具；
- 用休息卡；

但回家完全不用，那训练效果就很有限。

家庭泛化的核心是：

> 技能不只在训练室出现，也能在家里、学校、超市、公园出现。

---

## 二、为什么家长培训这么重要？

训练师一周可能只见孩子几小时。  
家长每天都在孩子生活里。

家里有最多自然机会：

- 吃饭；
- 穿鞋；
- 洗手；
- 玩玩具；
- 洗澡；
- 出门；
- 睡前；
- 情绪爆发；
- 请求帮助；
- 兄弟姐妹互动。

如果家长会抓这些机会，孩子进步会更稳。  
如果家长不理解方法，可能无意中强化问题行为。

---

## 三、家长培训的核心原则

### 原则1：少讲理论，多给动作

家长最需要的是：

> 我今晚回家该怎么做？

不是一大堆术语。

### 原则2：每次只布置一个小任务

不要一次给家长10个目标。

更好：

> 本周只练“还要”。每天泡泡游戏5次。

### 原则3：先肯定，再建议

家长长期焦虑，很容易防御。  
反馈要先看见努力。

### 原则4：家庭作业要嵌入生活

不要让家长每天像上课一样坐1小时。  
要用生活场景：吃饭、穿鞋、洗手、玩具打不开。

### 原则5：可执行比完美重要

家长能稳定做60分，比偶尔做100分更有用。

## 四、课后反馈三明治法

### 结构

1. 具体亮点；
2. 主要困难；
3. 家庭任务。

### 示例

```text
今天孩子在泡泡游戏中主动看老师3次，这是很好的沟通信号。
比较困难的是从自由玩切换到桌面任务时会尖叫，大约持续2分钟。
这周家里只练一个任务：泡泡停下来时，等他看您或说“还要”，再继续吹。每天5次就够。
```

### 不要这样反馈

- “今天还行。”
- “他不配合。”
- “回家多练。”
- “家长要严格一点。”

这些都不具体，家长不知道怎么做。

## 五、如何解释专业术语

### 1. 强化

家长版：

> 孩子用好方法表达后，我们马上让这个方法有用。

### 2. 提示

家长版：

> 孩子不会时，我们扶一下，但不是替他做。

### 3. 泛化

家长版：

> 不只在机构会，回家、学校、外面也会。

### 4. ABC记录

家长版：

> 看看孩子闹之前发生了什么，闹之后得到了什么。

### 5. 替代行为

家长版：

> 不只是说“别闹”，而是教他用什么新办法。

## 六、家庭作业怎么设计

### 好的家庭作业特点

- 时间短；
- 场景真实；
- 目标清楚；
- 家长会做；
- 有记录方式；
- 孩子容易成功。

### 不好的家庭作业

```text
每天训练语言、认知、社交、自理各30分钟。
```

家长很难坚持。

### 好的家庭作业

```text
每天晚饭时练5次“还要”。
孩子想继续吃时，家长等2秒，提示“还要”，孩子说/指/递卡后马上给一小口。
```

## 七、家庭训练任务示例

### 1. 请求训练

场景：泡泡、零食、玩具。

家长做法：

1. 先给一次；
2. 停下来；
3. 等2秒；
4. 提示“还要”；
5. 孩子表达后马上给。

### 2. 休息表达

场景：写作业或桌面任务。

家长做法：

1. 任务少一点；
2. 准备休息卡；
3. 孩子烦躁时提示递卡；
4. 递卡后休息1分钟；
5. 回来做一个小任务。

### 3. 洗手步骤

场景：饭前。

家长做法：

1. 贴步骤图；
2. 本周只练“擦手”；
3. 前面家长辅助；
4. 最后让孩子独立擦手；
5. 完成后吃饭。

### 4. 轮流

场景：滚球。

家长做法：

1. 家长滚给孩子；
2. 孩子滚回来；
3. 说“你一次，我一次”；
4. 先做3轮；
5. 成功就结束。

## 八、家长焦虑怎么回应

### 情况1：家长问“孩子能不能好？”

话术：

```text
我理解您很想知道未来会怎样。
我们不轻易承诺结果，但可以把眼前目标做清楚。
比如先让孩子学会表达“我要、不要、帮我、休息”，这些能力会直接改善生活。
```

### 情况2：家长觉得进步太慢

话术：

```text
自闭症干预常常是小步累积。
我们可以一起看数据：以前需要哭闹10分钟，现在能用休息卡2次，这就是进步。
```

### 情况3：家长自责

话术：

```text
这不是您造成的。
您现在愿意学习和配合，就是孩子最重要的支持。
我们不回头责备，只往前找方法。
```

### 情况4：家长执行困难

话术：

```text
我们先不要求每天做很多。
这周只做一个场景：吃水果时练“还要”，每天3次也可以。
```

## 九、家长不配合怎么办

先不要简单说家长“不配合”。

可能原因：

- 听不懂；
- 太忙；
- 太焦虑；
- 家里老人不支持；
- 任务太难；
- 看不到效果；
- 不认可方法；
- 没有被尊重。

### 处理SOP

1. 先问困难；
2. 把任务减半；
3. 现场示范；
4. 让家长练一遍；
5. 给一句话版本；
6. 下次只检查一个任务。

## 十、家长培训现场示范四步

### 第一步：老师做，家长看

老师示范完整过程。

### 第二步：老师讲关键点

只讲3个关键点。

例如：

- 等2秒；
- 提示“还要”；
- 说了马上给。

### 第三步：家长做，老师辅助

家长实际操作，老师旁边提示。

### 第四步：家长独立做，老师反馈

反馈要具体：

> “您刚才等待很好。”  
> “下次提示可以再短一点。”

### 模板

```text

练习场景：
家长要做：
孩子目标反应：
成功后怎么回应：
每天练几次：
记录方式：
```

### 示例

```text
本周家庭目标：请求“还要”
练习场景：吃水果
家长要做：先给一小块，停下来等2秒
孩子目标反应：看家长/指/说“还要”
成功后回应：马上给下一小块
每天练几次：5次
记录方式：打勾
```

## 十二、家庭数据怎么简化

家长不一定能做复杂数据表。

可以用最简单的：

| 日期 | 练了吗 | 成功几次 | 备注 |
|---|---|---:|---|
| 周一 | 是/否 |  |  |
| 周二 | 是/否 |  |  |
| 周三 | 是/否 |  |  |

或者：

```text
今天孩子主动说/指/递卡：□0次 □1次 □2次 □3次以上
```

## 十三、家庭中的一致性

### 常见不一致

- 妈妈坚持，奶奶马上给；
- 爸爸批评，妈妈安慰；
- 机构用休息卡，家里不用；
- 学校要求排队，家里任意插队。

### 解决办法

只统一最关键的3条：

1. 哭闹不直接得到手机；
2. 说/递卡“休息”可以休息；
3. 出门前先预告流程。

不要求全家马上完美一致，从最影响行为的规则开始。

## 十四、家庭作息和视觉日程

### 适合家庭的视觉日程

```text
起床 → 洗脸刷牙 → 早餐 → 上学 → 回家 → 吃饭 → 洗澡 → 睡觉
```

### 晚间流程

```text
吃饭 → 玩10分钟 → 洗澡 → 刷牙 → 故事 → 睡觉
```

### 使用方法

- 每完成一步打勾；
- 变化提前说明；
- 不要用日程威胁；
- 让孩子参与移动图片。

## 十五、家长如何处理发脾气

### 三句话

> “我知道你想要。”  
> “哭不能拿。”  
> “你可以说/指：我要。”

### 三动作

1. 保安全；
2. 少说话；
3. 等降温后提示表达。

### 规则

- 问题行为不直接成功；
- 合适表达马上有效；
- 可以降难度，但不完全放弃规则。

## 十九、新人常见错误

### 错误1：反馈太笼统

改法：  
说具体数据和具体动作。

### 错误2：一次布置太多

改法：  
每周只给1个核心家庭任务。

### 错误3：只批评家长

改法：  
先肯定，再给建议。

### 错误4：不示范

改法：  
老师做一遍，家长做一遍。

### 错误5：家庭作业脱离生活

改法：  
嵌入吃饭、洗手、穿鞋、玩具等场景。

### 错误6：不检查执行困难

改法：  
问家长“哪一步最难做？”

### 错误7：家长不会记录

改法：  
用打勾表，不用复杂表。

### 错误8：忽略家庭成员差异

改法：  
先统一最关键规则。

## 二十三、新人必须记住的10句话

1. 家长培训不是批评家长，而是赋能家长。  
2. 家庭作业越简单，越可能坚持。  
3. 每次只抓一个目标。  
4. 反馈要具体到行为和数据。  
5. 老师要示范，家长要练习。  
6. 家庭训练要嵌入生活，不要变成额外负担。  
7. 先解决家长最痛的场景。  
8. 家长焦虑时，需要方向，不需要指责。  
9. 家校机构一致，孩子才不混乱。  
10. 真正的泛化，是孩子在真实生活里也能用。

---

## 二十四、最后给新人训练师的话

你教孩子很重要，  
但你教会家长更重要。

因为家长不是旁观者，  
他们是孩子生活中最长期、最稳定、最关键的支持者。

好的家长培训，不是让家长变成专业治疗师，  
而是让家长在每天真实生活中多做几个正确的小动作：

- 多等2秒；
- 少替孩子说；
- 先提示表达；
- 表达后马上回应；
- 情绪爆发时少说话；
- 用视觉流程降低混乱。

这些小动作累积起来，就是孩子真正的生活改变。
