# 新人训练师典型案例库

> 适用对象：自闭症康复训练师、助教、督导、家长指导老师。  
> 使用场景：新人培训、案例讨论、督导复盘、家长说明、行为分析练习、IEP目标设计练习。  
> 核心目的：通过典型场景帮助新人练习“看现象、找功能、定目标、给SOP、避坑”。  
> 重要提醒：以下案例均为教学型化名案例，不代表某个真实儿童；实际个案需结合评估和督导。

---

## 案例使用方法

每个案例按五步学习：

1. 看外在行为；
2. 猜可能功能；
3. 找关键观察点；
4. 设计替代技能；
5. 写家庭/课堂SOP。

不要看到案例就直接套方案。  
同样的行为，功能可能不同。

## 案例1：一收手机就尖叫

### 现象

6岁男孩，每次家长收手机时尖叫、哭闹、躺地，持续10–20分钟。

### 可能功能

获得手机，或逃避手机结束后的任务。

### ABC示例

| A | B | C |
|---|---|---|
| 妈妈说“手机结束” | 尖叫、躺地 | 妈妈又给5分钟 |

### 干预目标

- 使用“再玩1分钟”卡；
- 学习计时器结束；
- 学习替代活动。

### SOP

1. 玩前设定计时器；
2. 出示“手机5分钟”；
3. 剩1分钟预告；
4. 结束时提示递“再玩1分钟”或“完成”；
5. 递卡后只延长一次；
6. 后面切换到替代活动。

### 避坑

- 不要尖叫后无限延长；
- 不要突然抢；
- 不要没有预告。

## 案例2：一上桌面就躺地

### 现象

4岁女孩，自由玩很好，一看到桌面卡片就躺地、闭眼、踢腿。

### 可能功能

逃避任务，或任务过难、强化不足。

### 观察重点

- 任务是否太多；
- 是否知道做完有什么；
- 是否有先后板；
- 是否有失败经验。

### 干预目标

- 完成1–2个简单任务；
- 使用休息卡；
- 建立先做后玩。

### SOP

1. 先玩1分钟建立动机；
2. 出示先后板：先放1个，后泡泡；
3. 只做最简单的1个；
4. 完成马上强化；
5. 逐步增加到2个、3个。

### 避坑

- 不要一开始就做10题；
- 不要躺地后直接取消所有任务；
- 不要在地上讲道理。

## 案例3：出门边走边跳和拍手

### 现象

12岁女孩，出门兴奋或人多时边走边跳、拍手。

### 可能功能

感觉调节、兴奋表达、焦虑释放。

### 支持目标

- 红区绿区规则；
- 替代感觉动作；
- 安全行走。

### SOP

1. 出门前画路线；
2. 红区：马路、电梯、楼梯不跳；
3. 绿区：公园空地可以跳10下；
4. 路上说：“走20步，再跳10下”；
5. 商场内用捏球或手按压替代。

### 避坑

- 不要羞辱“别人看你”；
- 不要全部禁止；
- 不要忽略安全。

## 案例4：孩子只转车轮

### 现象

3岁男孩拿到小车只转轮子，不推车，不和老师互动。

### 可能功能

视觉/感觉寻求；游戏发展停留在感官探索阶段。

### 目标

- 接受成人加入；
- 从转轮子扩展到推车；
- 共同注意。

### SOP

1. 先模仿他转轮子；
2. 老师拿另一辆车一起转；
3. 加入“停”；
4. 轻轻推一下车：“车开了”；
5. 车停住等孩子看；
6. 孩子看或发声，车继续。

### 避坑

- 不要马上抢车；
- 不要强行禁止转轮；
- 不要直接考颜色。

## 案例5：孩子会背很多词但不会表达需求

### 现象

5岁男孩会背数字、字母、广告词，但想喝水时只哭或拉大人。

### 问题本质

词汇多不等于功能沟通好。

### 目标

- 用“我要水”请求；
- 用“帮我”求助；
- 用“不要”拒绝。

### SOP

1. 把水杯放在看得见够不着处；
2. 孩子看水时提示“水”；
3. 逐步扩展“我要水”；
4. 说/指后马上给水；
5. 在不同场景练。

### 避坑

- 不要只继续教认字；
- 不要以为会背就语言很好。

## 案例6：孩子在商场捂耳尖叫

### 现象

7岁孩子进商场后捂耳、尖叫、拉妈妈离开。

### 可能功能

逃避噪音和人群刺激。

### 目标

- 表达“太吵”；
- 使用耳机；
- 短时间适应商场。

### SOP

1. 出门前预告；
2. 准备耳机；
3. 先在人少时进入5分钟；
4. 孩子捂耳时提示“太吵”；
5. 说/指后戴耳机或离开安静区；
6. 逐步增加停留时间。

### 避坑

- 不要逼他忍；
- 不要在人多时第一次训练；
- 不要忘记工具。

## 案例7：孩子总打同伴抢玩具

### 现象

6岁孩子看到同伴玩喜欢的车，直接打同伴并抢走。

### 可能功能

获得玩具；不会轮流和请求。

### 目标

- 说“轮到我”；
- 用计时器轮流；
- 交换玩具。

### SOP

1. 准备轮流卡和计时器；
2. 同伴先玩30秒；
3. 孩子提示说“轮到我”；
4. 轮到孩子玩30秒；
5. 逐渐增加等待时间；
6. 打人后不直接获得长期占有。

### 避坑

- 不要打了就让他玩很久；
- 不要只骂“不许打”；
- 要教新句子。

## 案例8：孩子只会仿说

### 现象

孩子问什么都重复最后一句。老师问“你要水吗？”孩子说“要水吗”。

### 可能原因

语言理解弱，表达模板少，仿说作为沟通尝试。

### 目标

- 从仿说转为功能表达；
- 使用选择和短句。

### SOP

1. 少问复杂问题；
2. 直接给模型：“我要水”；
3. 孩子仿说后，如果功能合适就给水；
4. 后续用选择：“水还是饼干”；
5. 再逐步教回答。

### 避坑

- 不要一直追问；
- 不要把仿说全当坏行为；
- 要把它变成表达。

## 案例9：孩子洗头崩溃

### 现象

每次洗头尖叫、挣扎、推开家长。

### 可能原因

触觉敏感、怕水进眼睛、怕不可预测。

### 目标

- 接受洗头步骤；
- 表达“停一下”；
- 缩短崩溃时间。

### SOP

1. 平静时看洗头步骤图；
2. 先练摸水；
3. 再练湿毛巾碰头发；
4. 再短时间冲头发；
5. 提供“停一下”卡；
6. 每完成一步强化。

### 避坑

- 不要突然花洒冲头；
- 不要强压；
- 不要边洗边骂。

## 案例10：青春期女孩经期前情绪爆发

### 现象

13岁女孩月经前几天明显易怒、哭闹、拒绝上学。

### 可能原因

经前不适、疼痛、睡眠变化、情绪调节困难。

### 目标

- 记录周期；
- 表达疼痛和情绪；
- 使用月经包；
- 经期支持计划。

### SOP

1. 记录月经周期和情绪；
2. 准备月经包；
3. 教“肚子痛”“我要休息”；
4. 经前降低任务压力；
5. 明显疼痛建议就医；
6. 学校同步支持。

### 避坑

- 不要说“她就是脾气差”；
- 不要忽略疼痛；
- 不要公开讨论隐私。

## 案例11：孩子被纠正就崩溃

### 现象

8岁孩子做错题，老师一说“不对”，孩子撕纸、哭、拒绝继续。

### 可能原因

失败耐受低、完美主义、纠错经验差。

### 目标

- 接受“错了可以改”；
- 使用帮助请求；
- 完成改错一步。

### SOP

1. 平静时讲“错了可以改”；
2. 老师故意示范自己做错再改；
3. 孩子错时不说刺耳“不对”；
4. 说“再试一次”；
5. 给强提示让他改成功；
6. 强化“你改好了”。

### 避坑

- 不要批评智力；
- 不要连续失败；
- 不要把改错变惩罚。

## 案例12：孩子上课总跑门口

### 现象

5岁孩子上课5分钟后跑到门口，拉门。

### 可能功能

逃避任务、想出去、感觉过载、追逐关注。

### 目标

- 使用“出去/休息”卡；
- 坐满3分钟；
- 安全边界。

### SOP

1. 门口贴红色停止标志；
2. 课程缩短为3分钟一组；
3. 准备休息卡；
4. 孩子看门时提示递卡；
5. 递卡后去指定休息区；
6. 回来完成一个小任务。

### 避坑

- 不要追着跑成游戏；
- 不要忽略安全；
- 不要一直增加任务。

## 新人必须记住的10句话

1. 同样行为，功能可能不同。  
2. 案例不是拿来硬套的，是拿来练思路的。  
3. 先写事实，再猜原因。  
4. 没有ABC，就不要急着定方案。  
5. 每个问题行为都要配替代技能。  
6. 感觉和身体原因必须考虑。  
7. 家庭建议要具体到动作。  
8. 危险行为先安全再分析。  
9. 数据能验证你的假设。  
10. 好训练师像侦探，也像教练。

## 最后给新人训练师的话

案例学习的价值，不是记住答案，  
而是训练你面对真实孩子时的思考顺序。

当你能从“他又闹了”变成：

> 前面发生了什么？  
> 他想得到什么或逃开什么？  
> 我能教他什么新办法？  
> 我要怎么记录是否有效？

你就开始真正进入专业训练师的思维方式。
