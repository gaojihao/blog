# 生活自理与独立生活技能训练手册

> 适用对象：自闭症康复训练师、家长指导老师、影子老师、融合教育老师、特教老师。  
> 使用场景：儿童或青少年在吃饭、穿衣、洗手、刷牙、如厕、洗澡、整理物品、月经管理、公共安全、家务、独立生活方面需要支持。  
> 核心目的：帮助训练师把生活技能拆成可教、可练、可记录、可泛化的小步骤，让孩子从“被照顾”逐步走向“参与照顾自己”。  
> 重要提醒：生活自理训练涉及身体接触、隐私、青春期卫生和安全边界，必须遵守儿童尊严、隐私保护、家长知情同意和机构伦理规范。

---

## 一、先用一句话讲清楚生活自理训练

一句大白话：

> 生活自理训练就是把吃、穿、洗、上厕所、整理、安全这些每天都要做的事，拆小后一步一步教给孩子。

康复训练的最终目标不是孩子在桌面上答对多少卡片，而是：

- 饿了会吃；
- 渴了会喝；
- 脏了会洗；
- 想上厕所会表达；
- 冷了会穿衣；
- 东西会整理；
- 危险会躲开；
- 不舒服会求助；
- 长大后能尽可能独立生活。

---

## 二、为什么生活技能这么重要？

很多家长和新人训练师容易把重点放在：

- 认颜色；
- 识字；
- 数数；
- 背诗；
- 做桌面题。

这些当然可以学，但生活自理是地基。

一个孩子如果不会表达上厕所、不会洗手、不会穿衣、不会等红灯、不会求助，他在学校、家庭和社会中都会遇到巨大困难。

生活技能的意义是：

> 少一点依赖，多一点尊严；少一点替代，多一点参与。

---

## 三、生活技能训练的核心原则

### 原则1：不要一口吃成胖子

生活技能看似简单，其实步骤很多。

比如洗手包含：

1. 走到水池；
2. 打开水；
3. 冲湿手；
4. 挤洗手液；
5. 搓手；
6. 冲干净；
7. 关水；
8. 擦手；
9. 扔纸巾或挂毛巾。

孩子不会洗手，可能只是卡在其中某一步。  
训练师要拆步骤，而不是只说“他不会洗手”。

### 原则2：先让孩子参与一小步

如果孩子完全不会穿衣，不要一开始要求全套独立。  
可以先让他：

- 把手伸进袖子；
- 拉一下裤子；
- 把鞋放正；
- 拉最后一下拉链。

只要参与，就是开始。

### 原则3：用视觉步骤

很多ASD孩子看图比听话更稳定。

可以用：

- 洗手步骤图；
- 刷牙步骤图；
- 穿衣流程图；
- 如厕流程图；
- 晚间流程表；
- 书包清单。

### 原则4：固定地点、固定顺序

生活技能要稳定，先固定：

- 固定牙刷位置；
- 固定毛巾位置；
- 固定穿衣顺序；
- 固定如厕流程；
- 固定收纳盒。

孩子稳定后，再慢慢泛化。

### 原则5：少说话，多提示环境

不要全程语言轰炸。

错误：

> “快点洗手，先开水，不是这样，拿洗手液，搓啊，你怎么又忘了？”

正确：

> 指步骤图：“第一步，开水。”  
> 等待。  
> 必要时手势提示。

### 原则6：身体提示要逐步撤销

生活技能常需要身体辅助，但不能永远抓着孩子做。

从：

```text
握手完成 → 扶手腕 → 轻碰手肘 → 指图片 → 独立
```

### 原则7：尊重隐私和身体边界

尤其是：

- 如厕；
- 洗澡；
- 换衣；
- 月经管理；
- 青春期卫生。

训练师必须遵守机构规范，不随意触碰隐私部位，不单独进行高隐私训练，必要时由同性照护者或家长执行。

## 四、任务分析：生活技能训练的核心工具

任务分析大白话：

> 把一件大事拆成一小步一小步。

例如“刷牙”不是一个动作，而是一串动作：

1. 拿牙杯；
2. 拿牙刷；
3. 打开牙膏；
4. 挤牙膏；
5. 刷上牙；
6. 刷下牙；
7. 刷里面；
8. 漱口；
9. 冲牙刷；
10. 放回原位。

训练师要记录：

- 哪些步骤独立；
- 哪些步骤需要提示；
- 哪些步骤拒绝；
- 哪些步骤有感觉困难。

---

## 五、链式教学：从哪一步开始教？

生活技能常用链式教学。

### 1. 正向链

从第一步开始教。

适合：

- 孩子有一定注意力；
- 步骤少；
- 开头容易。

例子：洗手从“打开水”开始教。

### 2. 反向链

大人完成前面步骤，让孩子完成最后一步。  
孩子成功后，再往前多做一步。

适合：

- 孩子动机弱；
- 容易挫败；
- 想让孩子马上体验完成感。

例子：穿鞋。

- 第一次：大人帮到最后，只让孩子拉最后一下魔术贴；
- 第二次：孩子把脚伸进去+拉魔术贴；
- 后面逐渐增加步骤。

### 3. 全任务教学

每次完整练全流程，不会的步骤给提示。

适合：

- 孩子能力较好；
- 需要建立完整流程；
- 家庭日常练习。

## 六、吃饭训练

### 1. 训练目标

根据孩子能力分级：

- 坐在餐桌；
- 接受餐具；
- 用手拿食物；
- 用勺子；
- 用叉子；
- 用筷子；
- 咀嚼；
- 不边吃边跑；
- 吃完收餐具；
- 表达“还要/不要/饱了”。

### 2. 常见问题

- 挑食；
- 只吃固定品牌；
- 不坐餐桌；
- 只让大人喂；
- 吞咽慢；
- 含饭；
- 用手抓；
- 拒绝新食物；
- 吃饭时看手机。

### 3. 坐餐桌SOP

1. 从短时间开始，比如坐30秒；
2. 桌上放孩子能接受的食物；
3. 使用先后板：“先坐好，后玩车”；
4. 坐好马上强化；
5. 逐渐延长到1分钟、3分钟、5分钟；
6. 不一开始要求吃很多。

### 4. 用勺SOP

1. 选择容易舀的食物，如酸奶、稠粥；
2. 大人先示范；
3. 孩子握勺，大人轻扶；
4. 只辅助到嘴边；
5. 成功一口就强化；
6. 逐渐减少手部辅助。

### 5. 表达训练

吃饭是很好的FCT场景。

教：

- 还要；
- 不要；
- 喝水；
- 饱了；
- 太烫；
- 帮我；
- 我要____。

### 6. 挑食提醒

严重挑食、吞咽困难、体重下降、长期便秘、呕吐，应建议家长咨询医生、营养师或进食治疗专业人员。

## 七、穿脱衣训练

### 1. 穿衣任务分析

以穿上衣为例：

1. 找到衣服；
2. 分清正反；
3. 找到领口；
4. 套头；
5. 伸一只手；
6. 伸另一只手；
7. 拉下衣服；
8. 整理衣摆。

### 2. 脱衣任务分析

1. 拉起衣摆；
2. 一只手退出袖子；
3. 另一只手退出袖子；
4. 从头上脱下；
5. 放入指定位置。

### 3. SOP

1. 选择宽松、容易穿的衣服；
2. 固定顺序；
3. 使用镜子或步骤图；
4. 先让孩子完成最后一步；
5. 每成功一步就反馈；
6. 逐渐撤身体辅助。

### 4. 感觉问题处理

有些孩子不穿某些衣服，不一定是任性。

可能原因：

- 标签扎；
- 布料痒；
- 领口紧；
- 袜子缝线不舒服；
- 温度敏感。

策略：

- 剪标签；
- 选择柔软衣物；
- 提前预告换季；
- 给两个可接受选择；
- 逐步适应新衣服。

## 八、穿鞋训练

### 1. 任务分析

1. 找到鞋；
2. 分左右；
3. 松开魔术贴/鞋带；
4. 把脚伸进去；
5. 拉鞋舌；
6. 脚踩到底；
7. 粘魔术贴/系鞋带；
8. 站起来检查。

### 2. 新手建议

先选：

- 魔术贴鞋；
- 宽口鞋；
- 有明显左右标记；
- 不用鞋带的鞋。

### 3. SOP

1. 鞋放固定位置；
2. 在鞋内贴左右图标；
3. 大人帮前半步，孩子完成最后拉魔术贴；
4. 逐渐让孩子多做一步；
5. 穿好后自然强化：出门。

### 4. 话术

> “先穿鞋，后出门。”  
> “脚进去。”  
> “拉一下。”  
> “好了，出门。”

## 九、洗手训练

### 1. 任务分析

1. 走到洗手池；
2. 打开水龙头；
3. 冲湿双手；
4. 挤洗手液；
5. 搓手心；
6. 搓手背；
7. 冲干净；
8. 关水；
9. 擦手；
10. 纸巾丢垃圾桶/毛巾挂好。

### 2. SOP

1. 贴洗手步骤图；
2. 固定饭前、如厕后、外出回家练；
3. 成人少说话，用图提示；
4. 不会的步骤给最小提示；
5. 完成后自然进入下一活动。

### 3. 常见困难

| 困难 | 可能原因 | 策略 |
|---|---|---|
| 怕水 | 触觉敏感 | 水流小一点，先碰水 |
| 不会搓 | 动作计划弱 | 身体提示后撤销 |
| 忘关水 | 步骤记忆弱 | 关水图片放大 |
| 不擦手 | 不理解完成 | 擦手作为最后一步固定 |

## 十、刷牙训练

### 1. 任务分析

1. 拿杯子；
2. 拿牙刷；
3. 打开牙膏；
4. 挤牙膏；
5. 刷门牙；
6. 刷左边；
7. 刷右边；
8. 刷里面；
9. 漱口；
10. 冲牙刷；
11. 放回原位。

### 2. 常见困难

- 拒绝牙刷入口；
- 牙膏味道不能接受；
- 不会吐水；
- 只咬牙刷；
- 刷几下就跑；
- 牙龈敏感。

### 3. 脱敏SOP

1. 先接受牙刷在手里；
2. 接受牙刷碰嘴唇；
3. 接受碰门牙1秒；
4. 接受刷2下；
5. 慢慢增加；
6. 换孩子能接受的牙膏；
7. 用计时器或刷牙歌。

### 4. 注意

如果孩子刷牙明显疼痛、牙龈出血、口臭严重、拒绝突然加重，建议家长看牙医。

## 十一、如厕训练

### 1. 训练前准备条件

不是所有孩子都适合立刻开始如厕训练。

可观察：

- 能保持尿布一段时间干；
- 有固定排便时间；
- 会表达不舒服；
- 能坐马桶短时间；
- 能理解简单指令；
- 家庭能坚持记录。

### 2. 如厕任务分析

1. 感觉到尿意/便意；
2. 表达厕所；
3. 走到厕所；
4. 脱裤子；
5. 坐马桶；
6. 排尿/排便；
7. 擦拭；
8. 提裤子；
9. 冲水；
10. 洗手。

### 3. 记录表

| 时间 | 是否尿/便 | 在哪里 | 是否有提示 | 备注 |
|---|---|---|---|---|
| 7:00 |  |  |  |  |
| 9:00 |  |  |  |  |
| 11:00 |  |  |  |  |
| 13:00 |  |  |  |  |
| 15:00 |  |  |  |  |
| 17:00 |  |  |  |  |

### 4. 定时如厕SOP

1. 先记录3–7天排尿排便规律；
2. 找出高概率时间；
3. 提前5–10分钟带去厕所；
4. 坐马桶时间短，1–3分钟；
5. 成功后强强化；
6. 不成功也平静结束；
7. 逐渐教表达“厕所”。

### 5. 重要提醒

便秘、憋便、排便痛会严重影响如厕训练。  
如果孩子长期便秘、排便哭、肛裂、憋便，应建议家长先咨询医生。

### 6. 隐私伦理

- 如厕训练优先由家长或同性照护者执行；
- 机构必须有明确规范；
- 不公开讨论孩子隐私；
- 不拍摄隐私部位；
- 保护孩子身体边界。

## 十二、洗澡和个人卫生

### 1. 常见困难

- 怕水淋头；
- 怕洗发水；
- 不喜欢脱衣服；
- 不喜欢擦身体；
- 不知道洗哪些部位；
- 青春期体味管理困难。

### 2. 洗澡任务分析

1. 准备衣物和毛巾；
2. 脱衣；
3. 进入浴室；
4. 调水温；
5. 冲身体；
6. 打沐浴露；
7. 洗身体；
8. 冲干净；
9. 洗头；
10. 擦干；
11. 穿衣；
12. 放好脏衣服。

### 3. 支持策略

- 用视觉步骤；
- 固定顺序；
- 水温稳定；
- 花洒水压小；
- 洗头前预告；
- 用干毛巾遮眼；
- 允许孩子选择沐浴露味道；
- 青少年用隐私保护清单。

### 4. 隐私重点

青春期后，应尽量培养孩子自己洗隐私部位。  
成人指导应使用图示、语言或同性家长支持，避免不必要身体接触。

## 十三、月经管理，适用于女孩

### 1. 为什么要提前教

不要等初潮来了才教。  
很多ASD女孩对血、疼痛、身体变化会害怕，需要提前预告和演练。

### 2. 需要教的内容

- 月经是什么；
- 血从哪里来；
- 不是受伤；
- 卫生巾怎么用；
- 多久换一次；
- 用过的卫生巾怎么包；
- 丢在哪里；
- 手要洗干净；
- 肚子痛怎么表达；
- 可以找谁帮忙。

### 3. 月经包

准备：

- 卫生巾；
- 小内裤；
- 湿巾；
- 塑料袋；
- 备用裤子；
- 简单步骤卡；
- 紧急联系人卡。

### 4. 步骤卡示例

```text
1. 去厕所
2. 脱裤子
3. 撕开卫生巾
4. 贴在内裤上
5. 穿好裤子
6. 用纸包旧卫生巾
7. 扔垃圾桶
8. 洗手
```

### 5. 安全边界

必须教：

- 隐私部位别人不能随便看或碰；
- 换卫生巾在厕所或私人空间；
- 如果需要帮助，找妈妈、女老师或指定照护者；
- 不跟陌生人讨论隐私；
- 不让别人拍照。

## 十四、整理书包和物品管理

### 1. 任务分析

1. 看清单；
2. 放课本；
3. 放作业本；
4. 放文具盒；
5. 放水杯；
6. 放纸巾；
7. 拉上拉链；
8. 放到固定位置。

### 2. 工具

- 书包清单；
- 图片标签；
- 文件袋颜色分类；
- 固定收纳盒；
- 完成打勾表。

### 3. SOP

1. 先减少物品数量；
2. 每个物品有固定位置；
3. 孩子按清单放一个勾一个；
4. 大人只提示清单，不直接替他装；
5. 每天固定时间整理。

## 十五、家务参与训练

家务不是惩罚，而是独立生活训练。

### 适合任务

低龄：

- 把玩具放盒子；
- 把衣服放篮子；
- 擦桌子；
- 扔垃圾；
- 摆碗筷。

学龄：

- 叠衣服；
- 整理书桌；
- 扫地；
- 收快递；
- 简单洗杯子。

青少年：

- 简单做饭；
- 洗衣机操作；
- 购物清单；
- 房间整理；
- 管理个人物品。

### SOP

1. 选一个短任务；
2. 示范；
3. 任务分析；
4. 用清单；
5. 完成后自然强化；
6. 固定为日常责任。

## 十六、公共安全训练

### 1. 必教安全技能

- 停；
- 等；
- 红灯停绿灯走；
- 不冲马路；
- 不跟陌生人走；
- 迷路找谁；
- 记住姓名和电话；
- 不随便吃别人给的东西；
- 电梯安全；
- 水边安全；
- 火和热水危险。

### 2. 过马路SOP

1. 到路口停下；
2. 看红绿灯；
3. 红灯停；
4. 绿灯看左右；
5. 拉手或按规则走；
6. 到对面后强化。

### 3. 走失求助卡

卡片内容：

```text
我叫：____
我需要帮助。
请联系我的家人：____
电话：____
```

### 4. 安全句子

必须练：

> “我迷路了。”  
> “我要找妈妈。”  
> “不要碰我。”  
> “我不跟你走。”  
> “我要找警察/工作人员。”

- 简单找零；

## 十八、生活技能目标怎么写？

### 错误写法

```text
提高自理能力。
学会刷牙。
能自己穿衣。
```

### 正确模板

```text
在________场景中，儿童能按照________步骤，在________提示下完成________，连续____天达到。
```

### 示例1：洗手

```text
在饭前洗手场景中，儿童能根据洗手步骤图完成打开水、冲手、搓手、关水、擦手5个步骤，其中身体提示不超过1次，连续5天达到。
```

### 示例2：穿鞋

```text
在出门前，儿童能在手势提示下完成把脚伸入鞋子并粘好魔术贴，连续5次机会中完成4次。
```

### 示例3：如厕表达

```text
在白天活动中，当儿童有尿意时，能通过说“厕所”或递厕所卡表达，并在成人带领下进入厕所，连续两周每天至少成功3次。
```

## 二十、家长版解释话术

### 1. 解释为什么不直接替孩子做

```text
我们当然可以很快替孩子穿好鞋、洗好手，但这样孩子永远没有练习机会。
我们现在的目标不是快，而是让他每天多参与一步。
```

### 2. 解释任务分析

```text
孩子不会刷牙，不代表每一步都不会。
我们会拆成拿牙刷、挤牙膏、刷门牙、漱口等小步骤，先找到他能做哪一步，再教下一步。
```

### 3. 解释自理比学业也重要

```text
识字和算数重要，但生活自理是孩子每天都要用的能力。
会表达上厕所、会洗手、会穿衣，会直接影响孩子上学、社交和未来独立。
```

### 4. 解释家里怎么配合

```text
每天只选一个生活技能，不要全都练。
比如这周只练洗手里的“擦手”一步。
孩子能参与一步，就先强化这一步，再慢慢往前加。
```

## 二十一、新人常见错误

### 错误1：嫌慢，直接替孩子做

改法：  
至少让孩子完成最后一步。

### 错误2：目标太大

改法：  
从一个步骤开始。

### 错误3：只用语言催

改法：  
用图片、清单、位置提示。

### 错误4：没有固定流程

改法：  
固定时间、地点、顺序。

### 错误5：忽略感觉问题

改法：  
观察衣物、牙刷、水温、声音、气味是否让孩子不适。

### 错误6：身体提示不撤

改法：  
从握手到轻触，再到视觉提示。

### 错误7：不保护隐私

改法：  
如厕、洗澡、月经管理必须遵守伦理边界。

### 错误8：不记录

改法：  
记录每一步是否独立。

### 错误9：只在机构练

改法：  
生活技能必须在家里真实场景练。

### 错误10：忽视医学问题

改法：  
如厕、进食、刷牙困难严重时，考虑便秘、牙痛、吞咽等医学因素。

## 二十五、新人必须记住的10句话

1. 生活自理是康复的核心，不是附属内容。  
2. 不会一件事，往往是卡在其中一小步。  
3. 先让孩子参与最后一步，也是一种进步。  
4. 视觉步骤比反复催促更有效。  
5. 固定流程能降低孩子焦虑。  
6. 身体提示要温和、必要、可撤销。  
7. 如厕、洗澡、月经管理必须保护隐私。  
8. 感觉敏感会严重影响自理训练。  
9. 生活技能必须在真实生活中泛化。  
10. 少替代，多等待；少责备，多拆步。

## 二十六、最后给新人训练师的话

生活自理训练看起来没有“认字”“说话”那么显眼，  
但它是孩子走向独立的每一块砖。

第一次自己擦手，  
第一次自己拉上鞋，  
第一次说“厕所”，  
第一次把卫生巾放进包里，  
第一次知道红灯要停，  
这些都不是小事。

它们意味着孩子正在一点点拥有照顾自己的能力，  
也意味着家庭未来会少一点焦虑，孩子会多一点尊严。

真正好的康复，不只是让孩子在课堂上表现好，  
而是让他在真实生活里更安全、更清楚、更独立。
