## 四、学习建议

### 新人第1个月

只抓三件事：

1. 看懂孩子；
2. 会用强化和提示；
3. 会做简单记录。

### 新人第2–3个月

重点练：

1. DTT；
2. NET；
3. FCT；
4. ABC记录；
5. 家长简短反馈。

### 新人第4–6个月

重点练：

1. 目标设计；
2. 行为支持；
3. 生活自理；
4. 感觉调节；
5. 家校泛化。

### 6个月以后

开始学习：

1. IEP报告；
2. 个案管理；
3. 青春期和转衔；
4. 复杂案例；
5. 督导思维。

## 五、训练师核心提醒

请始终记住：

> 先看懂，再介入；  
> 先安全，再教学；  
> 先建立关系，再提要求；  
> 先教新路，再堵旧路；  
> 先小步成功，再逐步提高；  
> 先真实生活，再课堂表现。

孩子不是一组症状，  
孩子是一个正在努力理解世界的人。  
训练师的价值，就是用专业、温和、稳定的方法，帮他一点点打开通往生活的门。
