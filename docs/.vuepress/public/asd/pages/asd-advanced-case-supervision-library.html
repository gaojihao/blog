#高级案例督导库

适用对象：高级训练师、督导、个案负责人、机构内训人员。
使用场景：复杂案例讨论、督导培训、高风险个案复盘、综合方案设计训练。
核心目的：通过复杂案例训练专家级思维：多维概念化、风险识别、方案整合、数据验证和团队协作。
重要提醒：案例为教学化名案例，真实个案必须结合评估、督导和转介。

#案例学习方法

每个案例按八步讨论：

  1. 表面问题是什么；
  2. 可能有哪些功能；
  3. 还需要收集什么信息；
  4. 有哪些红旗信号；
  5. 如何做安全计划；
  6. 如何设计综合方案；
  7. 如何分工协作；
  8. 用什么数据验证。

#案例1：频繁撞头的4岁儿童

#情况

4岁男孩，无语言。每当被要求停止玩水时撞头，有时撞地板。家长说最近便秘严重，晚上睡不好。

#多维分析

可能因素：

  • 逃避停止玩水；
  • 不会表达“还要”；
  • 感觉寻求；
  • 便秘疼痛；
  • 睡眠不足；
  • 撞头后家长立刻恢复玩水。

#红旗

  • 自伤；
  • 便秘；
  • 睡眠差。

#方案

  1. 立即安全计划，软垫、成人分工；
  2. 建议家长就医处理便秘和睡眠；
  3. 玩水前计时器和预告；
  4. 教“还要”“休息”“结束”图片；
  5. 撞头不直接恢复玩水，递卡后可获得短延迟；
  6. 记录撞头次数、强度、排便和睡眠。

#案例2：8岁孩子拒学

#情况

8岁男孩，语言流畅，最近早上哭闹不去学校，晚上反复问“明天要上学吗”。学校反馈他经常被同学说“怪”。

#多维分析

可能因素：

  • 焦虑；
  • 校园欺凌；
  • 社交理解困难；
  • 学校环境压力；
  • 睡前预期焦虑。

#红旗

  • 持续拒学；
  • 同伴欺负；
  • 焦虑明显。

#方案

  1. 学校调查欺凌；
  2. 建立安全老师；
  3. 分阶段返校；
  4. 社交脚本和求助句；
  5. 睡前视觉日程；
  6. 必要时心理专业支持；
  7. 数据记录上学成功率和焦虑评分。

#案例3：12岁女孩月经期攻击增加

#情况

12岁女孩，语言少，月经前后打人和尖叫明显增加，伴随捂肚子和拒绝坐下。

#多维分析

可能因素：

  • 痛经；
  • 不会表达疼痛；
  • 月经用品不适；
  • 情绪调节弱；
  • 任务要求未调整。

#红旗

  • 周期性疼痛疑似；
  • 青春期隐私和安全需求。

#方案

  1. 月经周期和行为记录；
  2. 教“肚子痛”“休息”“换卫生巾”；
  3. 准备月经包；
  4. 经期降低任务难度；
  5. 建议妇科或儿科咨询疼痛；
  6. 学校隐私支持；
  7. 记录攻击次数和疼痛表达。

#案例4：高功能青少年沉迷网络并被骗钱

#情况

15岁男孩，能正常对话，但社交判断弱。给网友转账，多次被要求保密。

#多维分析

  • 社交边界弱；
  • 网络安全缺失；
  • 渴望朋友；
  • 金钱管理弱；
  • 家长监管规则不清。

#风险

  • 经济被骗；
  • 个人信息泄露；
  • 线下见面风险。

#方案

  1. 网络安全规则卡；
  2. 三圈关系；
  3. 金钱转账规则：大额需家长确认；
  4. 情景演练“我要告诉家长”；
  5. 建立健康社交替代；
  6. 家长设置安全监管；
  7. 记录违规和求助次数。

#案例5：6岁孩子攻击老师，训练无效

#情况

每次做桌面任务时打老师。老师说已经消退两周，但攻击更严重。

#多维分析

  • 任务可能太难；
  • 消退爆发；
  • 没有替代行为；
  • 强化不足；
  • 新人单独处理高风险；
  • 可能已经形成逃避功能。

#风险

  • 攻击升级；
  • 错误使用消退。

#方案

  1. 暂停单纯消退；
  2. 督导介入；
  3. 任务降阶；
  4. 教休息卡和帮我；
  5. 完成最小任务后休息；
  6. 安全距离和材料控制；
  7. 数据记录攻击和替代行为。

#案例6：无语言儿童突然退步

#情况

3岁半孩子原来会说“妈妈、要”，最近一个月不说了，眼神减少，夜间发呆样表现。

#分析

这不是普通训练问题。
需要考虑：

  • 发育退步；
  • 癫痫样表现；
  • 睡眠或医学问题；
  • 环境重大变化。

#方案

  1. 立即建议医学评估；
  2. 记录退步时间线；
  3. 记录发呆视频，需家长自行给医生；
  4. 暂时降低训练要求；
  5. 使用AAC保持沟通；
  6. 跟进医生建议。

#案例7：孩子只在机构表现好，回家崩溃

#情况

5岁孩子机构会用“还要”和“休息”，回家仍然哭闹抢东西。家长说老人一哭就给。

#分析

  • 泛化失败；
  • 家庭后果不一致；
  • 家长任务太复杂；
  • 老人未培训。

#方案

  1. 家庭只选一个目标：还要；
  2. 现场视频示范；
  3. 让老人也参加培训；
  4. 统一规则：哭不直接给，递卡马上给；
  5. 家庭打勾记录；
  6. 每周复盘。

#案例8：10岁孩子课堂上频繁自言自语

#情况

课堂自言自语影响听课，内容多为动画台词。叫名能短暂回应。

#分析

可能因素：

  • 自我调节；
  • 焦虑；
  • 任务太无聊或太难；
  • 语言脚本；
  • 注意力困难。

#方案

  1. 记录发生时间和任务；
  2. 设置“心里说/小声说/休息区说”声音等级；
  3. 给脚本时间；
  4. 任务分段；
  5. 课堂替代：写关键词或捏球；
  6. 如果内容异常恐惧或像回应声音，建议专业评估。

#案例9：青春期女孩被同学利用

#情况

13岁女孩经常给同学买零食，同学说“不买就不是朋友”。女孩不理解被利用。

#分析

  • 朋友概念弱；
  • 金钱边界弱；
  • 渴望归属；
  • 欺凌/利用风险。

#方案

  1. 朋友和利用区别教学；
  2. 金钱规则：每天只花固定金额；
  3. 拒绝脚本：“我不买”；
  4. 求助老师；
  5. 家校沟通；
  6. 同伴安全教育；
  7. 记录事件。

#案例10：孩子严重挑食且便秘

#情况

7岁孩子只吃白米饭和某品牌饼干，便秘，训练中易怒。

#分析

  • 感觉口感问题；
  • 营养风险；
  • 便秘影响情绪；
  • 进食治疗需要。

#方案

  1. 建议儿科/营养/进食专业评估；
  2. 不强迫吃；
  3. 从相似食物逐步暴露；
  4. 记录食物接受阶梯；
  5. 排便记录；
  6. 情绪爆发时考虑身体因素。

#督导提问清单

  1. 你是否只看到了表面行为？
  2. 这个行为可能有什么功能？
  3. 有没有医学或心理红旗？
  4. 家庭后果是否维持了行为？
  5. 学校环境是否触发？
  6. 替代行为是什么？
  7. 数据怎么验证？
  8. 谁需要参与协作？
  9. 安全计划是否足够？
  10. 什么时候复盘？

#专家必须记住的10句话

  1. 高级案例没有单一答案。
  2. 复杂行为常由多个因素共同维持。
  3. 先识别红旗，再设计训练。
  4. 安全计划不是可选项。
  5. 医学因素不能被训练掩盖。
  6. 家庭和学校是行为系统的一部分。
  7. 替代沟通是复杂行为干预核心。
  8. 数据验证假设。
  9. 督导讨论要回到事实和功能。
  10. 专家思维是整合，而不是套公式。

#最后提醒

高级案例训练的目标，不是找到一个“标准答案”，
而是训练你在不确定中保持专业思考：

先安全，后分析；
先事实，后假设；
先沟通，后要求；
先系统，后单点技巧。
