#感觉处理与感觉调节支持手册

适用对象：自闭症康复训练师、家长指导老师、影子老师、融合教育老师、家长。
使用场景：孩子怕声音、怕触觉、怕洗头、怕理发、怕穿衣、爱跳爱撞、爱转圈、挑食、咬物、拍手、捂耳、出门崩溃、在商场或学校明显过载。
核心目的：帮助新人理解“感觉差异”如何影响孩子的情绪、行为、学习和生活，并学会做环境支持、感觉调节和安全替代。
重要提醒：感觉支持不是把孩子“训练成不敏感”，而是让他更舒适、更安全、更能参与生活。

#一、先用一句话讲清楚感觉处理

一句大白话：

感觉处理就是大脑怎么接收和理解声音、光线、触觉、味道、动作和身体位置。

有些孩子不是故意“矫情”，而是：

  • 声音对他太大；
  • 衣服对他太扎；
  • 洗头对他太刺激；
  • 人多对他太乱；
  • 身体需要更多压、跳、转、撞的感觉；
  • 饥饿、疲劳、疼痛也会放大感觉问题。

训练师要理解：

很多所谓“脾气大”的背后，其实是感觉系统在报警。

#二、感觉系统的几个常见维度

#1. 听觉

对声音的敏感程度。

表现：

  • 捂耳；
  • 怕吹风机、吸尘器、马路、广播；
  • 在吵环境崩溃；
  • 需要重复听同一声音；
  • 喜欢制造声音。

#2. 触觉

对皮肤接触、衣服材质、洗澡、剪指甲、理发的反应。

表现：

  • 不喜欢某些衣服、袜子、标签；
  • 不喜欢别人碰；
  • 怕头发被碰；
  • 洗头、刷牙困难；
  • 也可能喜欢挤压、包裹、深压。

#3. 视觉

对光线、闪烁、颜色、场景变化的处理。

表现：

  • 怕强光；
  • 喜欢盯旋转物、灯、风扇；
  • 在视觉杂乱环境里容易乱；
  • 喜欢看细节、排队、摆放。

#4. 味觉/嗅觉

表现：

  • 严重挑食；
  • 对气味敏感；
  • 不愿意尝试新食物；
  • 喜欢闻物品、闻人、闻手。

#5. 前庭觉

和运动、平衡、旋转、晃动有关。

表现：

  • 喜欢跳、转、荡；
  • 怕坐电梯或荡秋千；
  • 走路不稳或特别爱跑；
  • 对姿势变化反应大。

#6. 本体觉

和身体用力、关节压、深压、撞、推、搬有关。

表现：

  • 喜欢撞墙、跳、压、背重包；
  • 喜欢推东西；
  • 身体边界感弱；
  • 需要更多“用力感”来稳住自己。

#三、感觉差异如何影响行为

感觉问题常常表现为：

  • 发脾气；
  • 逃跑；
  • 拒绝；
  • 打人；
  • 躺地；
  • 拍手跳跃；
  • 自言自语；
  • 拒绝穿衣洗头；
  • 吞食异物；
  • 咬物；
  • 摇晃身体。

这些行为可能不是“坏”，而是在做三件事：

  1. 避开不舒服；
  2. 找回身体平衡；
  3. 获得想要的感觉刺激。

比如：

外在行为 可能的感觉意义
捂耳尖叫 声音太大，受不了了
拍手跳跃 身体需要兴奋或调节
撞墙 需要深压和身体反馈
拒绝穿某件衣服 材质/缝线/标签不舒服
挑食 味道、口感、温度太刺激

#四、感觉调节的底层逻辑

一句大白话：

不是让孩子“忍住”，而是先帮他找到更舒服、更安全、更可控的调节方式。

调节方式分两类：

#1. 减少过载

比如：

  • 降低声音；
  • 戴耳机；
  • 减少灯光；
  • 提前预告；
  • 减少人群；
  • 缩短停留时间；
  • 选择柔软衣物；
  • 给安静角。

#2. 提供需要的感觉

比如：

  • 跳床；
  • 推墙；
  • 捏球；
  • 深压抱枕；
  • 背小包；
  • 搬书；
  • 走感统路线；
  • 玩水；
  • 荡秋千，前提安全。

#五、感觉支持三步法

#第一步：观察

看孩子在什么情况下最容易出问题：

  • 在什么声音环境？
  • 在什么衣服/触觉下？
  • 在什么时候最烦躁？
  • 在什么活动后更安静？
  • 哪些动作会让他平静？

#第二步：判断功能

问：

  • 他是在逃避感觉刺激吗？
  • 他是在寻求感觉刺激吗？
  • 他是不是只是太累、太饿、太热、太痛？

#第三步：做支持

  • 先改环境；
  • 再给工具；
  • 再教表达；
  • 最后再练耐受。

#六、常见感觉支持工具

#1. 降噪耳机

适合：

  • 商场；
  • 机场；
  • 理发；
  • 吸尘器；
  • 吹风机；
  • 课间嘈杂环境。

用法：

  • 先在平静时练；
  • 先短时间戴；
  • 再进入轻刺激环境；
  • 戴上后立刻表扬。

#2. 视觉提示

适合：

  • 预告变化；
  • 任务顺序；
  • 安静角规则；
  • 出门流程；
  • 洗头步骤。

#3. 触觉支持

  • 柔软衣物；
  • 去标签；
  • 弹性袜；
  • 宽松裤；
  • 毛巾包裹；
  • 压力毯或抱枕，需安全。

#4. 本体觉支持

  • 推墙；
  • 搬物；
  • 抱枕挤压；
  • 背书包；
  • 爬坡；
  • 深压活动。

#5. 感觉角

一个可控的小空间，里面放：

  • 捏捏球；
  • 抱枕；
  • 软垫；
  • 耳机；
  • 计时器；
  • 情绪卡。

#七、感觉寻求型孩子怎么支持

感觉寻求型孩子喜欢：

  • 跳；
  • 碰撞；
  • 转圈；
  • 摇晃；
  • 咬物；
  • 大动作；
  • 重压。

#支持策略

  1. 给安全出口：跳床、推墙、搬东西、感统路线；
  2. 设规则：红区不撞，绿区可以跳；
  3. 把寻求变成任务奖励：做完2个任务，可以跳10下；
  4. 教替代：想跳时说“我要跳”；想压时说“抱枕”；
  5. 避免过度禁止：不危险的重复动作不必全删。

#例子

孩子上课坐不住、总想站起来。
你可以：

  • 每5分钟安排一次“推墙10秒”；
  • 用重书包作为短时本体觉输入；
  • 在桌面后安排短跳跃；
  • 用计时器帮助等待。

#八、感觉逃避型孩子怎么支持

感觉逃避型孩子常常对：

  • 声音；
  • 光线；
  • 触碰；
  • 气味；
  • 某些食物口感；
  • 洗头、刷牙、理发；
  • 人多地方。

#支持策略

  1. 提前预告：不是突然来；
  2. 降低刺激：声音小、灯柔和、人少；
  3. 给工具：耳机、帽子、毛巾、安静角；
  4. 分步骤暴露：先远看，再靠近，再短时间接触；
  5. 教表达：太吵、太亮、太扎、我要出去；
  6. 不要硬逼：硬逼往往让孩子更敏感。

#例子：洗头

孩子怕洗头。
可以这样拆：

  • 先摸水；
  • 再把水拍在手上；
  • 再淋一点到肩膀；
  • 再淋到头发；
  • 再短时间冲洗；
  • 再慢慢增加。

#九、出门前的感觉预告

很多崩溃发生在“没预告”。

#出门前可以说：

“今天去超市，会有灯和人。”
“你可以戴耳机。”
“先走到门口，再上车。”

  • 在超市待5分钟；
  • 用耳机；
  • 说“太吵”；
  • 选择一个东西就结束。

#十、感觉过载时怎么做

感觉过载通常不是“耍脾气”，而是系统已经满了。

#现场SOP

  1. 降低声音；
  2. 减少人；
  3. 少说话；
  4. 移到安静区；
  5. 给耳机、抱枕或毯子；
  6. 允许少量运动或深压；
  7. 等待恢复；
  8. 恢复后再复盘。

#不要做

  • 继续讲道理；
  • 围着问“为什么”；
  • 让多人同时安慰；
  • 逼他看人；
  • 继续加任务。

#十一、感觉饮食和进食支持

#常见感觉问题表现

  • 只吃极少数食物；
  • 不接受新的口感；
  • 对温度敏感；
  • 不吃混合食物；
  • 只吃脆的/软的/某种颜色；
  • 一闻到味道就拒绝。

#支持原则

  1. 不强迫；
  2. 从孩子能接受的相近食物开始；
  3. 先看、闻、碰，再尝；
  4. 每次只改变一个维度；
  5. 记录哪些口感能接受。

#进阶策略

  • 先接受新食物在盘里；
  • 再用手碰；
  • 再舔；
  • 再咬一小口；
  • 慢慢扩大范围。

#注意

严重挑食、吞咽困难、体重下降、呕吐、便秘要先做医学排查。

#十二、感觉支持与行为支持如何配合

很多行为干预失败，是因为忽略了感觉。

比如：

  • 孩子一进教室就烦躁；
  • 你以为是“不配合”；
  • 其实是教室太吵、太亮、太乱。

这时最优先的不是加任务，而是：

  • 降噪；
  • 调光；
  • 减少干扰；
  • 提供感觉工具；
  • 再回到教学。

#十三、感觉目标怎么写？

#错误写法

改善感觉问题。
让孩子不怕声音。
减少拍手。

#正确写法

在嘈杂环境中，儿童能在成人提示下使用耳机或递出“太吵”卡请求离开，在5次机会中完成4次，连续2周达到。

在洗头场景中，儿童能在步骤图提示下完成从摸水到冲洗头发的3个步骤，身体提示不超过1次，连续5次达到。

在需要身体调节时，儿童能通过推墙、抱枕压一压或捏球选择1种方式安静下来，连续3节课达到。

#十五、家长版解释话术

#1. 解释感觉差异

孩子不是故意挑剔，而是他的大脑对声音、触感、气味或动作的感受和别人不一样。
我们要做的是帮他找到更舒服的方式，而不是一味逼他忍。

#2. 解释为什么要用耳机或安静角

耳机和安静角不是逃避，而是调节工具。
先让孩子有办法稳定下来，他才有可能慢慢适应更复杂的环境。

#3. 解释为什么不能硬逼洗头穿衣

硬逼通常会让孩子更排斥。
我们会先做小步骤适应，比如摸水、短时间冲头、去标签、选柔软衣服，然后慢慢加。

#4. 家庭练习建议

您可以先观察孩子最怕什么、最喜欢什么身体感觉。
怕声音就先准备耳机，喜欢压就用抱枕，喜欢跳就安排短跳跃。
先让孩子稳定，再慢慢练耐受。

#十六、新人常见错误

#错误1：把所有行为都当成不听话

改法：
先问是不是感觉问题。

#错误2：直接强迫适应

改法：
小步暴露，逐步增加。

#错误3：只压制，不替代

改法：
给跳、压、推、捏等合法出口。

#错误4：忽略环境

改法：
调光、降噪、减人、减乱。

#错误5：感觉工具不给孩子随时可用

改法：
工具要可见、可拿、可用。

#错误6：爆发后还继续训练

改法：
先降温，恢复后再练。

#错误7：挑食时只靠哄骗

改法：
用渐进暴露，不强迫。

#错误8：没记录触发因素

改法：
记录何时、何地、什么刺激引发。

#二十、新人必须记住的10句话

  1. 很多情绪问题背后是感觉问题。
  2. 孩子不是故意矫情，而是感受真的不一样。
  3. 先降刺激，再教适应。
  4. 感觉寻求需要安全出口。
  5. 感觉逃避需要工具和预告。
  6. 耳机、安静角、视觉卡都可能是重要支持。
  7. 不要硬逼孩子忍。
  8. 挑食、洗头、理发、穿衣都可能有感觉原因。
  9. 爆发时先调节，恢复后再训练。
  10. 感觉支持的目标，是让孩子更舒适、更安全、更能参与生活。

#二十一、最后给新人训练师的话

感觉问题很容易被误解成“任性”“脾气差”“不服管”。
但如果你愿意认真观察，你会发现：

  • 有些孩子是在躲太吵的世界；
  • 有些孩子是在找身体稳定感；
  • 有些孩子是在告诉你：这个触感、这个声音、这个味道，真的让我难受。

优秀的训练师不是强迫孩子适应所有刺激，
而是帮他一步一步学会：

哪些刺激我可以接受；
哪些刺激我可以避开；
我可以怎样表达不舒服；
我可以怎样安全地调节自己。

这就是感觉处理和感觉调节支持的真正意义。
