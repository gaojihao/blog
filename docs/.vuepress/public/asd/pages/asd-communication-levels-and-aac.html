#沟通层级与AAC辅助沟通实操手册

适用对象：自闭症康复训练师、语言老师、家长指导老师、影子老师、融合教育老师。
使用场景：孩子无语言、语言少、表达不清、仿说多、情绪一来就不会说、需要图片沟通板或电子AAC支持。
核心目的：帮助新人理解沟通发展的层级，学会评估孩子当前沟通水平，并把手势、图片、沟通板、字词、电子设备等工具用到真正有效。
重要提醒：AAC不是放弃说话，而是先保证孩子有一个能表达的办法。

#一、先用一句话讲清楚AAC

AAC全称 Augmentative and Alternative Communication，常译为“辅助与替代沟通”。

一句大白话：

AAC就是当孩子暂时不会说、说不出来、说不清时，给他一个别的办法来表达。

AAC可以是：

  • 手势；
  • 眼神；
  • 指物；
  • 图片卡；
  • 沟通板；
  • 文字；
  • 平板语音按钮；
  • 手机沟通APP；
  • 写字；
  • 打字。

AAC的目标不是“替代孩子一辈子说话”，而是：

先让孩子能表达，再慢慢提高表达质量和语言水平。

#二、为什么要看“沟通层级”？

不是所有孩子都需要从说完整句开始。

很多ASD孩子的沟通能力是分层的：

  • 有的还不会用动作表达；
  • 有的会拉大人；
  • 有的会指物；
  • 有的会递卡；
  • 有的会说单词；
  • 有的能说短句；
  • 有的会聊天，但情绪一来就表达不出来。

训练师如果直接要求所有孩子都说完整句，常常会失败。

所以我们要先看：

这个孩子现在在哪一层？下一层应该怎么上？

#三、沟通发展的八个层级

#第1层：无意图表达

表现：

  • 哭；
  • 叫；
  • 拉大人；
  • 推开；
  • 抢；
  • 跑。

说明：

孩子有需求，但还没有稳定的可理解表达。

#第2层：动作/行为表达

表现：

  • 指物；
  • 递物；
  • 拉手；
  • 摇头点头；
  • 推开不要；
  • 拿东西给大人。

说明：

孩子开始用动作“说话”。

#第3层：声音/单音节

表现：

  • 啊、啊呀、妈、要、开、走；
  • 仿说一两个字；
  • 会重复口头提示。

说明：

孩子开始把声音和意义连起来。

#第4层：单词表达

表现：

  • 要；
  • 水；
  • 车；
  • 不要；
  • 帮；
  • 走；
  • 休息。

说明：

孩子能用一个词表达需求或拒绝。

#第5层：短句表达

表现：

  • 我要水；
  • 帮我打开；
  • 我不要这个；
  • 还要泡泡；
  • 我想休息。

说明：

孩子开始把词组合起来。

#第6层：简单对话

表现：

  • 回答问题；
  • 简单问答；
  • 讲经历；
  • 提问；
  • 评论。

说明：

孩子开始进行来回沟通。

#第7层：复杂对话

表现：

  • 讲原因；
  • 讲计划；
  • 解释想法；
  • 讨论；
  • 自我修正。

说明：

孩子可以更灵活地表达和理解。

#第8层：情境化沟通

表现：

  • 公众场合沟通；
  • 与陌生人问路；
  • 网络交流；
  • 校园和职场沟通；
  • 安全求助。

说明：

孩子能把沟通用到真实生活里。

#四、沟通评估看什么

#1. 理解

孩子听得懂多少？

  • 叫名是否回应？
  • 是否理解常见指令？
  • 是否理解图片？
  • 是否理解“先……后……”？

#2. 表达

孩子能怎么说？

  • 哭；
  • 拉；
  • 指；
  • 递卡；
  • 发声；
  • 单词；
  • 短句；
  • 手写；
  • 打字。

#3. 沟通功能

孩子在表达什么？

  • 要东西；
  • 拒绝；
  • 请求帮助；
  • 请求休息；
  • 分享兴趣；
  • 求关注；
  • 表达不适；
  • 求安全。

#4. 场景

孩子在哪些场景沟通最好？

  • 泡泡；
  • 零食；
  • 玩具；
  • 家里；
  • 学校；
  • 出门；
  • 情绪平稳/爆发时。

#五、无语言孩子怎么开始

#1. 先别急着教“说话”

先看孩子现在会不会：

  • 看你；
  • 伸手；
  • 指；
  • 拉你；
  • 递东西；
  • 模仿；
  • 发声。

#2. 用动机最高的东西

比如：

  • 泡泡；
  • 零食；
  • 小车；
  • 秋千；
  • 音乐；
  • 玩水；
  • 手机屏幕；
  • 喜欢的玩具。

#3. 创造表达机会

  • 东西放高一点；
  • 先关上；
  • 少给一点；
  • 停一下；
  • 等孩子来找。

#4. 先强化任何合适表达

孩子只要有：

  • 看人；
  • 拉你；
  • 递卡；
  • 发声；
  • 指物；

都可以立刻回应。

目标是：

让孩子知道“表达有用”。

#六、手势沟通怎么教

#常见手势

  • 指；
  • 点头；
  • 摇头；
  • 举手；
  • 招手；
  • 拉人；
  • 递物；
  • 双手合十；
  • 比“停”；
  • 比“来”。

#教学SOP

  1. 先选一个功能明确的动作，比如“指泡泡”；
  2. 老师示范给孩子看；
  3. 把强化物停一下；
  4. 等孩子做手势；
  5. 立刻给结果；
  6. 反复练。

#目标

从：

  • 拉大人；

过渡到：

  • 指；
  • 递卡；
  • 手势；
  • 语言。

#七、图片沟通怎么教

#1. 图片沟通适合谁

适合：

  • 无语言；
  • 语言少；
  • 说不清；
  • 情绪一来不会说；
  • 需要稳定表达工具的孩子。

#2. 图片选择原则

  • 清楚；
  • 一张代表一个意思；
  • 大小合适；
  • 常用优先；
  • 容易拿；
  • 贴在孩子能看到的地方。

#3. 初级图片

先准备：

  • 我要；
  • 还要；
  • 不要；
  • 帮我；
  • 休息；
  • 打开；
  • 太吵；
  • 厕所；
  • 喝水；
  • 妈妈；
  • 老师。

#4. 教法

  1. 孩子想要东西；
  2. 老师把对应图片递给他；
  3. 轻提示他拿卡递给老师；
  4. 老师接卡，马上给结果；
  5. 慢慢减少帮助。

#5. 图片不是考试

图片的重点是：

孩子拿来就能得到结果。

不要拿图片反复问答却不给实际反馈。

#八、沟通板怎么设计

#初级沟通板

[我要] [不要] [帮我]
[休息] [打开] [还要]
[厕所] [喝水] [太吵]

#家庭沟通板

[吃] [喝] [玩]
[妈妈] [爸爸] [老师]
[出去] [睡觉] [洗澡]
[手机] [车] [泡泡]

#学校沟通板

[休息] [帮我] [厕所]
[铅笔] [书] [水]
[我不会] [等一下] [完成]

#沟通板使用原则

  • 放在孩子随手能拿的位置；
  • 大人必须示范；
  • 每次使用后都要有反应；
  • 一次不要太多；
  • 定期更新。

#九、电子AAC怎么用

电子AAC可以是：

  • 平板语音按钮；
  • 手机应用；
  • 录音按钮；
  • 文字转语音设备。

#适合对象

  • 图片沟通已经稳定；
  • 能理解选择；
  • 喜欢按按钮；
  • 需要更丰富表达；
  • 书写或打字能力较好。

#教学原则

  1. 从最常用的词开始；
  2. 不要一次塞太多按钮；
  3. 常用词放前面；
  4. 每次按按钮后马上获得结果；
  5. 家校统一设备或词汇。

#适用词汇

  • 我要；
  • 还要；
  • 帮我；
  • 休息；
  • 不要；
  • 打开；
  • 太吵；
  • 回家；
  • 妈妈；
  • 老师。

#十、FCT和AAC怎么结合

FCT是功能沟通训练，AAC是辅助工具。两者可以一起用。

例如：

  • 孩子想要饼干；
  • 用图片卡递给老师；
  • 老师说“饼干”；
  • 给一小块饼干；
  • 逐渐过渡到说词或短句。

或者：

  • 孩子觉得太吵；
  • 递“太吵”卡；
  • 老师立刻带去安静区；
  • 后续再教说“太吵了”。

#十一、仿说怎么处理

仿说大白话：

孩子重复大人刚说的话。

仿说不一定是坏事，它可以是语言发展的中间台阶。

#处理方法

  1. 不急着打断；
  2. 把仿说变成有意义表达；
  3. 用模型句教他怎么用；
  4. 在真实场景中练。

#例子

孩子说：

“要水，要水。”

老师：

“对，水。”

然后马上给水。

如果孩子总是重复问题句，可以给更短、更实用的表达模板。

#十二、如何从单词到短句

#单词阶段

孩子说：

  • 要；
  • 水；
  • 车；
  • 走；
  • 不要；
  • 帮。

#扩展方法

老师把词扩成句：

  • “要” → “我要水”；
  • “车” → “我要车”；
  • “帮” → “帮我打开”；
  • “不要” → “我不要这个”。

#教学SOP

  1. 先接受单词；
  2. 给正确句式模型；
  3. 不要求一次说完整句；
  4. 逐步让孩子模仿；
  5. 最后在真实场景独立用。

#十三、如何从短句到对话

#目标

让孩子不只是要东西，而是能交流。

#训练内容

  • 回答问题；
  • 提问；
  • 评论；
  • 讲经历；
  • 轮流对话；
  • 解释想法。

#例子

评论

孩子看到飞机：

“飞机！”

老师：

“对，飞机飞高高。”

提问

孩子学会问：

“这是什么？”
“谁来了？”
“为什么？”

讲经历

“今天去公园。”
“我玩泡泡。”
“我回家了。”

#十四、情绪状态下的沟通

很多孩子平时能说，一生气就不会说。

#要教的情绪表达

  • 我生气了；
  • 我不要；
  • 太吵了；
  • 我想休息；
  • 我需要帮忙；
  • 我想离开；
  • 我不舒服。

#情绪卡

把表情图和文字放一起：

  • 开心；
  • 生气；
  • 害怕；
  • 累了；
  • 太吵；
  • 休息。

#训练原则

  • 平静时练；
  • 不在爆发时要求完整表达；
  • 先让孩子能用卡，再逐步加语言。

#十五、沟通层级怎么测

#观察表

层级 表现
无意图 哭、叫、拉、抢
动作 指、递、拉人、点头摇头
声音 发声、模仿单音
单词 要、水、车、帮
短句 我要水、帮我打开
对话 回答、提问、评论
场景化 在学校、商场、家里都能用

#判定原则

不要只看“会不会说”。
要看：

  • 会不会主动；
  • 会不会在多场景用；
  • 会不会在情绪变化时用；
  • 会不会被陌生人理解。

#十六、沟通目标怎么写？

#错误写法

提高语言能力。
学会说话。
改善表达。

#正确模板

在________场景中，当________发生时，儿童能通过________方式表达________，在____次机会中独立完成____次，连续____天/节课达到。

#示例

请求目标

在泡泡活动中，当老师停止吹泡泡时，儿童能通过递泡泡卡或说“还要”请求继续，5次机会中独立完成4次，连续3节课达到。

休息目标

在桌面任务中，当儿童感到烦躁时，能递休息卡或说“休息”，在5次机会中完成4次，连续3节课达到。

安全求助目标

在外出场景中，当儿童感到不适或迷路时，能说“我要找妈妈”或递求助卡，连续2周每周至少成功3次。

#十八、家长版解释话术

#1. 解释AAC不是放弃说话

AAC不是不让孩子说话，而是先给他一个能表达的办法。
很多孩子有需求但说不出来，图片或按钮能先让他体验“表达有用”。
当表达动机起来后，再逐步推动语言发展。

#2. 解释为什么孩子需要图片

图片比语言更稳定，尤其是在孩子情绪高、环境乱、压力大的时候。
先用图片让他能沟通，后面再加声音和句子。

#3. 解释为什么大人要一致

如果今天说了“休息”有效，明天又不认卡片，孩子就会失去信任。
全家和学校尽量统一相同的词和图。

#4. 家庭练习建议

家里先只做3–5个最常用的词，比如：我要、不要、帮我、休息、打开。
每次孩子用卡或说词，大人马上给结果。
不要把AAC拿来反复考他。

#十九、新人常见错误

#错误1：一上来给太多词

改法：
先从最常用、最有动机的词开始。

#错误2：只在课上用

改法：
家里、学校、外出都要用。

#错误3：用了AAC但不回应

改法：
孩子一表达，大人要马上反应。

#错误4：总逼孩子说完整句

改法：
先接受卡片、手势、单词，再逐步扩展。

#错误5：把AAC当考试

改法：
AAC是沟通工具，不是测验卷。

#错误6：工具放太远

改法：
孩子随时能拿到。

#错误7：家长不会用

改法：
先教家长3个最常用词。

#错误8：不做泛化

改法：
同一个词在多场景重复练。

#二十三、新人必须记住的10句话

  1. AAC不是放弃说话，而是先保证能表达。
  2. 沟通要看层级，不能一上来就要求完整句。
  3. 最先教最有动机的词。
  4. 图片、手势、语音按钮都可以是沟通工具。
  5. 孩子表达后，大人要马上响应。
  6. 情绪高时先用简单AAC，不要硬逼口语。
  7. 家校要统一工具和词汇。
  8. AAC要放在真实生活里用，不是只在训练室。
  9. 仿说可以是桥梁，不必一棍子打死。
  10. 好的沟通工具会增加表达动机，而不是削弱语言发展。

#二十四、最后给新人训练师的话

很多孩子不是不想说，而是“说”这条路太难走。
AAC的意义，就是先给他一条更容易走的路。

当孩子第一次把“我要”卡递给你，
第一次按下按钮说“帮我”，
第一次在情绪里用“休息”代替尖叫，
你会看到：

沟通不是等待某个神奇时刻才出现，
沟通是我们一步一步搭出来的桥。

先让孩子被理解，
再慢慢让他表达得更完整、更多样、更自然。
这就是AAC最重要的价值。
