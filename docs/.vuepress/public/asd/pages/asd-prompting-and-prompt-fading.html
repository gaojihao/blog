#提示与提示撤销实操手册

适用对象：自闭症康复训练师、助教、影子老师、融合教育老师、家长指导老师。
使用场景：儿童不会回应指令、不会模仿、不会完成任务、需要成人帮助；或者儿童已经会做但总等老师提醒，出现提示依赖。
核心目的：让新人学会“什么时候帮、怎么帮、帮多少、怎么慢慢松手”，既不让孩子反复失败，也不让孩子永远依赖大人。
重要提醒：身体提示必须温和、必要、短暂，并遵守儿童安全、隐私和伦理边界。涉及如厕、洗澡、青春期卫生、身体接触类训练时，必须符合机构规范和家长知情同意。

#一、先用一句话讲清楚提示

一句大白话：

提示就是孩子暂时不会时，大人给一点帮助，让他先做成功。

比如：

  • 孩子不知道把积木放哪里，老师指一下盒子；
  • 孩子不会拍手，老师先示范拍手；
  • 孩子不会说“要”，老师轻声提示“要”；
  • 孩子不会洗手，老师指着水龙头图片；
  • 孩子不会穿鞋，老师轻轻扶一下他的手。

提示像扶孩子骑自行车：

刚开始扶住，是为了让他体验成功；后面慢慢松手，是为了让他自己会。

#二、提示不是替孩子做

新人最容易犯的错是：

看到孩子不会，就立刻替他完成。

这不叫教学，这叫代劳。

#提示和代劳的区别

项目 提示 代劳
目的 帮孩子成功 大人快速完成
孩子参与 有参与 很少参与
是否逐步减少 会减少 一直由大人做
长期结果 孩子独立 孩子依赖

例子：教孩子放积木进盒子。

代劳：

老师直接拿起孩子手里的积木放进盒子，然后说“做完了”。

提示：

老师先说“放进去”，等2秒。
孩子不动，老师指盒子。
还不动，老师轻轻碰孩子手肘帮助启动。
孩子完成后，马上强化。
下一次老师先尝试只指盒子，不再碰手。

#三、为什么提示撤销很重要？

提示撤销大白话：

帮助要越给越少，让孩子从“靠老师”变成“靠自己”。

如果不撤提示，会出现提示依赖。

#什么是提示依赖？

大白话：

孩子明明会，但不等老师提示就不做。

表现：

  • 老师不指，他不拿；
  • 老师不重复，他不回答；
  • 老师不碰手，他不动；
  • 老师不说第一个字，他不说话；
  • 每次都看老师脸色等答案；
  • 家里会做，机构不会做，或反过来。

提示依赖不是孩子懒，常常是大人教出来的。

#四、常见提示类型

#1. 身体提示

大白话：

用身体接触帮助孩子完成动作。

等级从重到轻：

  • 完全身体辅助：握手完成；
  • 部分身体辅助：扶手腕、扶手肘；
  • 轻触提示：轻碰一下手背或肩膀。

适合：

  • 动作技能；
  • 生活自理；
  • 初学模仿；
  • 精细动作。

注意：

  • 必须温和；
  • 不要粗暴拉拽；
  • 不要造成疼痛；
  • 避免不必要身体接触；
  • 尽快撤到更轻提示。

#2. 示范提示

大白话：

老师先做一遍给孩子看。

例子：

  • 老师拍手，说“这样做”；
  • 老师把积木放进盒子；
  • 老师示范用纸巾擦嘴；
  • 老师示范说“我要水”。

适合：

  • 模仿；
  • 游戏；
  • 社交动作；
  • 生活技能。

#3. 手势提示

大白话：

用指、点、摆手、眼神方向提醒孩子。

例子：

  • 指向正确答案；
  • 指向椅子；
  • 指向水杯；
  • 用手势表示“给我”；
  • 用眼神看向目标物。

#4. 语言提示

大白话：

用话提醒孩子。

例子：

  • “放进去”；
  • “说要”；
  • “看这里”；
  • “下一步洗手”；
  • “先写名字”。

语言提示很常用，但也最容易造成依赖。
因为大人太容易一直说。

#5. 视觉提示

大白话：

用图片、文字、符号、流程卡帮助孩子理解。

例子：

  • 视觉日程；
  • 先后板；
  • 洗手步骤图；
  • 选择板；
  • 规则卡；
  • 情绪卡；
  • 工作清单。

视觉提示非常适合ASD儿童，因为很多孩子“看得懂”比“听得懂”更稳定。

#6. 位置提示

大白话：

把正确答案放在更容易选的位置。

例子：

  • 让正确卡片离孩子更近；
  • 把要拿的鞋放在最显眼位置；
  • 把目标物放在孩子惯用手一侧。

注意：

位置提示要尽快撤，否则孩子可能只学会选最近的，不是真的理解。

#7. 线索提示

大白话：

给一点点开头，让孩子接下去。

例子：

  • 老师说：“小狗怎么叫？汪……”
  • 孩子接：“汪。”
  • 老师说：“我要……”
  • 孩子接：“水。”

适合：

  • 语言表达；
  • 回答问题；
  • 熟悉句型。

注意：

不要长期每次都给第一个字，否则孩子会等你开头。

#六、提示使用黄金原则

#原则1：先等，再帮

给指令后，不要马上帮。
先等2–3秒。

SOP：

  1. 给指令：“放进去。”
  2. 等2–3秒；
  3. 孩子动了，立刻强化；
  4. 孩子不动，再给提示。

等待很重要。
很多孩子不是不会，而是处理速度慢。

#原则2：给刚刚够的帮助

不要过度帮助。

如果孩子只需要你指一下，就不要握着他的手。
如果孩子只需要你看一眼目标物，就不要说一大堆。

#原则3：提示后也要强化

孩子在提示下完成，也是完成。
但强化强度可以比独立完成稍弱。

例子：

  • 独立完成：强强化；
  • 手势提示完成：中等强化；
  • 身体提示完成：轻强化或自然完成。

#原则4：独立完成要特别强化

如果孩子从需要提示变成独立完成，必须让他感受到差别。

老师可以说：

“你自己做的！太棒了！”

并给更强强化。

#原则5：提示要计划撤销

每次给提示时，都要想：

下一次我能不能少帮一点？

#原则6：不要重复指令当提示

错误：

“坐下，坐下，坐下，坐下，坐下！”

孩子可能学到：

前几次不用听，反正老师会重复很多遍。

正确：

“坐下。”
等2秒。
指椅子。
必要时轻触提示。

#原则7：避免无意识提示

新人常常不自觉给答案：

  • 眼睛盯着正确答案；
  • 身体偏向正确答案；
  • 声音在正确答案时变高；
  • 每次正确答案都放同一边；
  • 孩子看老师表情猜答案。

训练师要学会控制自己的身体线索。

#七、提示策略一：从多到少

专业上叫“最大到最小提示”。
大白话：

一开始给足帮助，保证孩子成功；之后慢慢减少帮助。

适合：

  • 新技能；
  • 孩子容易失败；
  • 错误后容易崩溃；
  • 生活自理；
  • 动作模仿。

#示例：教“放进去”

第1轮：完全身体提示，帮助放进去。
第2轮：扶手腕。
第3轮：轻碰手肘。
第4轮：指盒子。
第5轮：只说“放进去”。
第6轮：孩子独立完成。

#优点

  • 成功率高；
  • 减少错误；
  • 适合初学。

#风险

  • 如果撤得慢，容易提示依赖。

#八、提示策略二：从少到多

专业上叫“最小到最大提示”。
大白话：

先给孩子机会自己做，不会再逐步加帮助。

适合：

  • 孩子已经学过；
  • 复习任务；
  • 你不确定他会不会；
  • 想测独立能力。

#示例：教“给我”

  1. 老师伸手说：“给我。”
  2. 等2秒。
  3. 孩子不动，老师看向物品。
  4. 还不动，老师指物品。
  5. 还不动，老师轻碰孩子手。
  6. 完成后强化。

#优点

  • 给孩子独立机会；
  • 不容易过度帮助；
  • 能观察真实水平。

#风险

  • 如果孩子错误太多，会挫败；
  • 不适合完全不会的新任务。

#九、提示策略三：时间延迟

大白话：

指令后先等一等，让孩子有机会自己反应；不会再提示。

#固定时间延迟

每次都等固定时间，比如3秒。

#逐步时间延迟

一开始马上提示，后来慢慢增加等待时间。

第1阶段：指令后马上提示
第2阶段：等1秒
第3阶段：等2秒
第4阶段：等3秒

适合：

  • 语言表达；
  • 图片沟通；
  • 日常流程；
  • 已经会但反应慢的孩子。

#示例：教孩子说“要”

  1. 拿泡泡，孩子想要；
  2. 老师看着孩子，等2秒；
  3. 孩子不说，老师提示：“要”；
  4. 孩子说或模仿，马上吹泡泡；
  5. 后续逐渐从提示“要”变成只等待。

#十、提示策略四：错误少教学

大白话：

不让孩子连续错很多次，而是提前给足帮助，让他多体验正确。

适合：

  • 容易因失败发脾气的孩子；
  • 新任务；
  • 语言和配对初学；
  • 需要建立信心的孩子。

示例：教颜色红色。

错误多教学可能是：

老师问“哪个是红色？”孩子乱指，错了很多次。

错误少教学是：

  1. 先只放红色卡片；
  2. 说“红色”，帮助孩子摸红色；
  3. 再加入一个明显不同颜色；
  4. 正确多次后再增加干扰项。

注意：
错误少不是永远不让孩子思考，而是在初学阶段降低错误率。

#十一、提示撤销的四种方法

从：

↓
2碰手肘
↓
3指一下
↓
4不碰
查看原始流程文本 
扶手腕 → 碰手肘 → 指一下 → 不碰

#方法2：减少语言

从：

“拿杯子，放到桌上，放这里，对，就是这里。”

变成：

“杯子。”

再变成：

看向杯子或使用图片。

#方法3：增加等待

从马上提示，变成等1秒、2秒、3秒。

#方法4：转为自然线索

最终让真实环境提示孩子。

例子：

  • 水龙头脏手 → 提示洗手；
  • 饭前餐具 → 提示坐下吃饭；
  • 书包在门口 → 提示准备上学；
  • 门铃响 → 提示开门前问是谁。

#十二、不同技能怎么提示

#技能一：叫名反应

#目标

孩子听到名字后转头、看人或回应。

#SOP

  1. 站在孩子侧前方；
  2. 手里拿强化物；
  3. 叫名字一次；
  4. 等1–2秒；
  5. 不反应，用声音玩具或轻轻移动强化物到眼前；
  6. 孩子一转头，马上强化；
  7. 逐渐减少玩具辅助；
  8. 换距离、换人、换场景。

#避坑

  • 不要连续叫十几遍；
  • 不要叫名后没有任何好事发生；
  • 不要每次都用手掰脸；
  • 不要把叫名变成批评开头。

错误：

“小明！小明！小明！你怎么不听！”

正确：

“小明。”
孩子看你。
“你看妈妈了，泡泡！”

#技能二：模仿动作

#目标

孩子能跟着老师做动作。

#SOP

  1. 面对孩子坐；
  2. 先确保孩子注意到你；
  3. 说：“这样做。”
  4. 老师做动作；
  5. 等2秒；
  6. 不会就给身体或手势提示；
  7. 做对马上强化；
  8. 逐步撤提示。

#提示顺序示例

拍手：

  • 握手拍；
  • 轻碰双手；
  • 老师夸张拍手；
  • 只说“这样做”；
  • 独立模仿。

#避坑

  • 孩子没看你就示范；
  • 动作太复杂；
  • 一次换太多动作；
  • 每次都握手不撤。

#技能三：物品配对

#目标

孩子把相同物品或图片放在一起。

#SOP

  1. 先用完全相同实物；
  2. 桌面只放1个目标，减少干扰；
  3. 老师说：“放一样的。”
  4. 等2秒；
  5. 不会就指目标位置；
  6. 再不会轻扶手完成；
  7. 正确后强化；
  8. 逐渐增加干扰项。

#提示撤销

  • 从身体提示到指点；
  • 从近距离位置提示到正常距离；
  • 从1个干扰项到2–3个干扰项；
  • 从实物配实物到图片配图片。

#技能四：主动请求

#目标

孩子用动作、图片、声音或语言表达想要。

#SOP

  1. 找强动机物；
  2. 放在孩子看得见拿不到的位置；
  3. 等孩子看、伸手或发声；
  4. 根据能力给提示：
    • 指物；
    • 递图片；
    • 模仿“要”；
    • 说“我要+物品”；
  5. 孩子表达后马上给；
  6. 慢慢减少语言提示，只用等待和期待眼神。

#避坑

  • 孩子一伸手就无条件给，错过教学机会；
  • 要求过高，必须说完整句才给；
  • 孩子哭了才提示表达；
  • 每次老师都先问“你要什么”，孩子不主动。

#十四、如何判断提示撤销是否成功

成功不是“孩子做了一次”。
要看稳定性。

#成功指标

  1. 独立完成比例增加；
  2. 身体提示减少；
  3. 老师语言减少；
  4. 换老师仍能做；
  5. 换材料仍能做；
  6. 换场景仍能做；
  7. 家长也能看到类似表现。

#例子

不稳定掌握：

只在王老师面前会，只用红色积木会，只在训练室会。

较稳定掌握：

不同老师、不同盒子、家里和机构都能把物品放进去。

#十五、提示依赖怎么办？

#1. 先找原因

常见原因：

  • 老师提示太快；
  • 老师重复指令太多；
  • 独立完成没有更强奖励；
  • 任务太难，孩子不敢尝试；
  • 孩子习惯看老师脸色；
  • 之前错误被批评太多。

#2. 处理SOP

第一步：增加等待。

指令后等3秒，不马上提示。

第二步：减少语言。

从“拿红色放这里”变成“找一样”。

第三步：强化独立。

孩子自己做，给更强强化。

第四步：使用视觉提示替代成人提示。

让孩子看图，不是看老师。

第五步：老师身体退后。

从坐旁边变成坐远一点，减少控制感。

第六步：故意保持中性表情。

避免孩子通过表情猜答案。

#十六、错误纠正怎么做

孩子做错时，不要骂，也不要让他连续错。

#错误纠正SOP

  1. 保持平静；
  2. 简短说：“再来。”或“这个不是”；
  3. 立即给更强提示；
  4. 帮孩子做对；
  5. 做对后轻强化；
  6. 下一轮降低难度再试。

#示例

老师说：“拿苹果。”
孩子拿了香蕉。

错误处理：

“这个是香蕉。再来。”
老师指苹果。
孩子拿苹果。
“对，苹果。”

不要说：

“你怎么又错了？刚才不是教过了吗？苹果苹果苹果！”

#十七、家长版解释话术

#1. 解释提示

提示就是孩子还不会时，我们给一点帮助。
但这个帮助不是替他做，而是让他在帮助下成功，然后慢慢少帮一点。

#2. 解释为什么不能一直提醒

如果每次都提醒，孩子会学会等大人，而不是自己想下一步。
所以我们会先等几秒，再给刚刚够的帮助。

#3. 解释提示撤销

提示撤销就像教孩子骑车。
一开始扶着，后来扶少一点，再后来只在旁边看着。
最终目标是孩子自己会，不是永远靠我们提醒。

#4. 解释孩子做错怎么办

孩子做错不是故意捣乱，说明他还没真正学会。
我们会少批评、多示范，让他马上看到正确做法。

#十九、不同场景的提示注意事项

#1. 一对一训练室

优势：干扰少，容易控制。

  • 换老师；
  • 换桌子位置；

#2. 家庭

优势：自然机会多。

  • 家长语言要短；
  • 给孩子等待时间；

#3. 学校

优势：真实集体环境。

  • 用同伴示范；
  • 简化语言；

#二十、新人常见错误

#错误1：指令刚说完就帮

孩子没有独立机会。

改法：
等2–3秒。

#错误2：无限重复指令

孩子学会等第5遍才动。

改法：
说一次，等待，提示。

#错误3：身体提示太多

孩子容易被动，甚至抗拒。

改法：
尽快撤到轻触、手势或视觉提示。

#错误4：提示后不记录

看不出是否进步。

改法：
记录I、V、G、M、P。

#错误5：只帮不撤

孩子形成依赖。

改法：
每次问自己：下一轮能不能少帮一点？

#错误6：孩子错了就批评

孩子更怕尝试。

改法：
立即示范正确，降低难度。

#错误7：用眼神偷偷给答案

孩子学会看老师，不看材料。

改法：
控制表情和身体方向。

#错误8：视觉提示用了就不撤

孩子永远看图，不会自然完成。

改法：
从完整图片步骤，逐渐减少到关键词、符号、自然线索。

#错误9：提示和强化不匹配

孩子独立完成和被动完成得到一样奖励。

改法：
独立完成给更强强化。

#错误10：在情绪爆发时教新技能

孩子大脑已经过载。

改法：
先降温，恢复后再练。

#二十三、新人训练师必须记住的10句话

  1. 提示是帮助，不是代劳。
  2. 先等，再帮。
  3. 只给刚刚够的帮助。
  4. 帮了以后要计划撤。
  5. 不要无限重复指令。
  6. 独立完成要更强强化。
  7. 身体提示要温和、短暂、必要。
  8. 视觉提示能减少语言轰炸。
  9. 提示等级必须记录。
  10. 最终目标不是孩子在你提示下会，而是没有你也会。

#二十四、最后给新人训练师的话

提示就像一只扶着孩子过桥的手。
扶得太少，孩子可能摔倒；扶得太久，孩子学不会自己走。

优秀训练师的本事，不是一直帮孩子完成，
而是精准判断：

此刻他需要我帮多少？
下一次我能不能少帮一点？
最后怎样让环境和孩子自己接管？

当孩子从“等老师提醒”变成“我自己会做”，
提示才真正完成了它的使命。
