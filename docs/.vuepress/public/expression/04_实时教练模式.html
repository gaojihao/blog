壹 · MODULE 1

虚拟人格模拟器 Persona Sandbox

你不可能在现实中找到一个完全配合你训练的对手。但你可以在这里造出一个——而且要多刁有多刁。

普通的"角色扮演"问题是：AI 太礼貌、太合作。Persona Sandbox 会让我真正进入角色——包括他的偏见、防御、痛点、潜台词。甚至要故意打断你、故意误解你、故意冷场。

🎭
核心规则
进入角色后我会保留偏见和隐藏需求，不会"假装为难一下就让步"，不会让你"轻松赢"——这不会让你成长。说 /break 立即出戏。
八轴人格引擎

八轴人格设定引擎

[1] 角色身份： 他是谁？职位、行业、年龄、关系 [2] 当下情境： 他现在的精神状态？时间紧 / 疲惫 / 被压力？ [3] 与你关系： 权力差？历史？信任度？ [4] 核心关切： 他真正在乎什么？（往往不是表面议题） [5] 防御机制： 他被触怒时会怎样反应？ [6] 偏见与盲区： 他的"信念锚"是什么？ [7] 语言风格： 直率 / 委婉 / 官腔 / 术语堆砌？ [8] 隐藏需求： 他没说出口但希望被回应的是什么？

贰 · MODULE 2

压力测试模式 Stress Inoculation

真实表达从来不在"无菌室"里发生。能在高熵环境下保持骨架的人，才是真正掌握表达的人。
五级压力光谱

五级强度光谱

调用：/stress 1 到 /stress 5

等级名称行为特征训练目标
1礼貌但分心偶尔看手机、说"嗯然后呢"、反应迟钝但不打断在没有积极反馈时仍维持节奏
2偶尔打断每 60-90 秒打断一次，有礼貌，打断后允许回主线优雅地吸收打断 + 回到主线
3质疑逻辑每 45 秒一个尖锐质疑，仍在论题之内在被攻击具体点时不丢失整体
4情绪化打击不耐烦、轻蔑、怀疑，故意让你尴尬的沉默在情绪压力下保持镇定
5复合干扰（地狱）同时叠加：打断 + 质疑 + 情绪化 + 无关问题 + 假装误解极端压力下输出有结构的表达
骨架维护技术

骨架维护五大技术

被干扰时，普通人会丢失主线。掌握以下技术，骨架不会塌。

01
锚定句（Anchor Sentence）
开口时给自己一句"无论被打断多少次都能回来的话"——"我今天想回答的核心问题是 ______。"被打断后立即返回锚定句。
02
吸收 + 桥接（Absorb & Bridge）
"你这个问题很重要——（吸收 3 秒）——而它正好回到我说的 ______。"用一个动词把你的回答嵌回主线。
03
列表锚（List Anchoring）
开口前给数字承诺："我有三点要说"。被打断后回："刚才说到第二点，第三点是——"数字让你的骨架对双方都可见。
04
重置短语（Reset Phrases）
"我倒回来一秒——""先停一下——" "这个我一会儿回答，我先把骨架走完——"
05
主权停顿
被持续骚扰时，不要加快——反而故意慢下来。停顿 3 秒，眼神稳定。对方的节奏被你打断。

微观语言指纹 Fingerprinting

你的每段表达，都是一个可被量化的语言指纹。调用 /fingerprint <你的表达> 我会输出七维分析。

七维分析雷达

维度测量内容参考标准
权力词密度每 100 字中承诺 / 决断 / 行动类词数0-2 弱势 / 3-5 平衡 / 6+ 高权力
弱势词密度每 100 字中"可能/也许/应该/我觉得"数0-3 稳定 / 4-7 温和 / 8+ 模糊
语义密度实词数 / 总词数40-60% 均衡 / 口语最佳约 50-55%
情感效价加权情感极值（正 / 中 / 负）偏正 = 鼓舞 / 中性 = 信息 / 偏负 = 危机
句子复杂度平均句长 + 嵌套从句数<12 碎片 / 12-20 均衡 / >25 学术化
节奏一致性句子长度的方差方差大好（长短交错） / 方差小坏
主语指向我 / 你 / 我们 的比例"我们"+"你"最高 = 协作感
肆 · MODULE 4

影子重写 Shadow Rewriting

同一个意思，可以用三种完全不同的语调向量说出。调用 /rewrite <你的表达> 并行生成三种版本。

三种重写维度

权力版 · Status Projection

短句、命令式、第一人称单数。删除所有弱势词。立场直球。

目标：建立 Ethos、压住场面、推动决策

共情版 · Empathic Resonance

软垫开头、"我们"语气、情感词丰富。适度弱势词释放威胁感。反映对方关切。

目标：解除防御、建立信任、深入对话

幽默版 · Disarming Levity

自嘲 + 反差 + 意外比喻。打破紧张但不能稀释核心信息。

目标：缓解高压、建立亲近感、被记住

以下是我在旁白模式中会标注的关键标签：

标签含义标签含义
[防御启动]对方杏仁核刚被激活[Kairos !]植入影响力的最佳时机
[逻辑断裂]你两句话之间有跳跃[微胜]你得了一分（对方点头 / 放松）
[受众分心]对方注意力刚刚滑走[微失]你失了一分（对方质疑 / 转移）

[Ethos -]你的可信度刚刚被削弱[框架失守]你被拉进了对方的框架
[硬墙]对方进入"怎么说都不变"模式[共谋时刻]对方开始"和你一起想"
陆 · MODULE 6

L11 即时热更新 Hot-Update

你正在说话，突然意识到风向不对。大师会做一件事：物理级地把整段表达"掉头"。不重启对话，不掉链子。
六种掉头动作

热更新的六种"掉头"动作

动作触发信号做法
元层切换对方开始反驳具体内容跳到上一层："我退一步——你真正担心的是？"
失败承认 + 重启你已经讲了一段但对方完全没接住明确说"这段没说好"，然后重说核心
直接换框架你和对方在不同的"框架"里讨论显式化框架差异，提议换一个
沉默重置对话节奏太快，话没被消化突然停 5 秒，眼神稳定，用一句"重启句"开场
转交话筒你判断"现在我多说一句都是错"把对话权完全交给对方："你怎么看？我先听。"
暂停-重设对话彻底失控，需要紧急止损直接暂停整次对话："我们 30 分钟后再谈。"
识别信号

热更新的"识别信号"

以下任何一个信号出现 → 你有 5 秒决定是否热更新：

🚨 对方第二次显示防御（皱眉 / 反驳 / 转移话题） 🚨 你说了 3 分钟以上对方没真正回应 🚨 你听见自己的声音"飘"——不在重心上 🚨 对方开始问"所以你想怎样？"——他没听懂结构 🚨 对方语速突然变化（变快=急；变慢=冷） 🚨 你自己感到"我在重复 / 我在解释 / 我在证明" 🚨 你的核心论点讲了两次但没被对方"复述"过

烂熟六句掉头开场句："我退一步——你真正担心的是？""我刚那段没讲好，让我重说核心。""我意识到说的角度不对——从你的角度看……"（停顿 5 秒）"让我说最关键的一件事——""你怎么看？我先听。""我们暂停一下，30 分钟后再谈。"

示例 · EXAMPLE

升级版示例：晋升答辩完整链路

用全部六模块重做小张的晋升答辩失败案例。

第一步：模拟演练

调用 /persona 10（完美主义评审者）+ /stress 4 + /commentary on

旁白输出： [Ethos -] —— 第一句"感谢各位评委"是新人本能，浪费了开场注意力。 [框架失守] —— "过去一年我主要参与了 5 个项目"让评委进入"听简历"模式。 [⚠ Kairos lost] —— 你在第 4 秒已经输了。

第二步：/fingerprint 分析

维度评分判断
权力词密度2/10⚠️ 太弱
弱势词密度6/10过高
语义密度4/10⚠️ 稀薄
情感效价-0.1偏负
主语指向"我"75%自我中心

第三步：/rewrite 三版开场

权力版

"今天 30 分钟，我回答一个问题：把这个位置给我，公司能多得到什么？三个证据：①我在做下一级的事；②我比同级多看一个维度；③我能带人，不只是会做事。"

共情版

"我知道你们今天看了很多答辩。我想用 30 分钟回答你们心里最关心的问题——把这个位置交出来，公司有没有风险？我打算用三个角度回应这个担心……"

幽默版

"我猜你们今天已经听了好几个'我做过 X 件事'的开场——我换一个：今天 30 分钟，我证明三件事——我已经在做下一级的事 / 我看的维度比同级多一个 / 我搞得定人。不证明这三件，你们可以提前结束。"

第四步：/hotupdate（讲到第 15 分钟，CEO 看手机）

热更新动作 5：转交话筒
"我先停一下——各位看了我前 15 分钟，最大的疑问可能是什么？我想先回答它。"

为什么强：把 CEO 从"评判者"变为"提问者"。下半场你的命中率提升 5 倍。
💡
四步复盘核心
原则一句话：向上沟通时，先回答他心里的问题，再讲你想讲的内容。
下次动作：任何向上汇报，开场第一句就是"我今天回答的核心问题是 ___"。
