# 强化物使用与代币系统实操手册

> 适用对象：自闭症康复训练师、助教、家长指导老师、影子老师、融合教育老师。  
> 使用场景：儿童学习动机低、配合度低、不会等待、任务持续时间短、需要建立课堂规则、需要减少零食依赖时。  
> 核心目的：让新人理解“强化不是贿赂”，并学会科学选择强化物、及时强化、逐步撤销、建立代币系统，让孩子从“为了奖励才做”慢慢走向“理解规则、享受互动、能完成任务”。  
> 重要提醒：强化系统是教学工具，不是控制孩子的手段。任何强化都应尊重儿童安全、健康、尊严和个体差异。

---

## 一、先用一句话讲清楚强化

一句大白话：

> 强化就是孩子做出我们希望增加的行为后，马上得到一个好结果，于是他以后更愿意再做这个行为。

例如：

| 孩子做了什么 | 马上发生什么 | 结果 |
|---|---|---|
| 孩子说“要水” | 大人马上给水 | 下次更可能说“要水” |
| 孩子坐好10秒 | 老师马上夸并给小车 | 下次更可能坐好 |
| 孩子递“休息卡” | 老师给休息1分钟 | 下次更可能用卡而不是尖叫 |

强化的核心不是奖励本身，而是告诉孩子：

> 你刚刚这个行为很有用，值得再做一次。

## 二、强化和贿赂有什么区别？

这是新人和家长最容易混淆的地方。

### 1. 贿赂是什么？

贿赂通常发生在问题行为之后。

例子：

```text
孩子尖叫、躺地、不写作业。
家长说：“你别哭了，我给你手机。”
```

孩子学到的是：

> 我哭闹，就能得到手机。

### 2. 强化是什么？

强化发生在合适行为之后。

例子：

```text
孩子说“休息一下”。
家长说：“你用语言说了，可以休息2分钟。”
```

孩子学到的是：

> 我用合适方式表达，大人会听懂。

### 3. 区别表

| 项目 | 贿赂 | 强化 |
|---|---|---|
| 出现时机 | 问题行为后 | 目标行为后 |
| 孩子学到 | 哭闹有效 | 合适行为有效 |
| 大人状态 | 被动妥协 | 主动教学 |
| 长期结果 | 问题行为增加 | 目标行为增加 |

记住一句话：

> 哭闹后给，是贿赂；合适表达后给，是强化。

## 三、为什么ASD训练特别需要强化？

很多自闭症儿童不是“不想学”，而是普通社交奖励对他们不够有吸引力。

普通孩子可能因为：

- 老师夸一句；
- 妈妈笑一下；
- 小朋友喜欢他；
- 完成任务有成就感；
- 得到小红花；

就愿意继续做。

但有些ASD孩子可能暂时不觉得这些有意义。  
所以我们要先找到孩子真正喜欢、真正能激发动机的东西。

训练师要明白：

> 强化物不是纵容，而是通往学习的发动机。

没有发动机，车再好也开不动。

## 四、强化物有哪些类型？

强化物不等于零食。强化物可以很多样。

### 1. 食物强化

例子：

- 小饼干；
- 葡萄干；
- 水果粒；
- 海苔；
- 米饼；
- 小口饮料。

适合：

- 初期动机弱；
- 目标行为刚建立；
- 需要高频练习。

注意：

- 小份量；
- 注意过敏；
- 注意吞咽安全；
- 不要影响正餐；
- 不要作为唯一强化。

### 2. 玩具强化

例子：

- 小车；
- 泡泡；
- 积木；
- 音乐玩具；
- 弹跳球；
- 贴纸；
- 旋转玩具。

适合：

- 游戏能力训练；
- 互动训练；
- 轮流、请求、等待。

### 3. 活动强化

例子：

- 荡秋千；
- 跳床；
- 追逐游戏；
- 玩水；
- 吹泡泡；
- 看30秒动画；
- 和老师玩挠痒痒。

适合：

- 感觉寻求明显；
- 喜欢运动；
- 需要提升互动乐趣。

### 4. 社交强化

例子：

- 表扬；
- 击掌；
- 微笑；
- 拥抱，前提是孩子喜欢且边界允许；
- 夸张表情；
- 老师的兴奋回应。

注意：

有些孩子一开始不喜欢别人夸张表扬。  
如果孩子对大声表扬捂耳、躲开、皱眉，那表扬可能不是强化，而是刺激。

### 5. 感觉强化

例子：

- 深压；
- 捏球；
- 跳；
- 推墙；
- 摇晃；
- 旋转；
- 听固定声音。

适合：

- 感觉需求明显的孩子；
- 坐不住；
- 需要身体调节。

### 6. 权利强化

例子：

- 选择第一个玩具；
- 当小老师；
- 选择座位；
- 选择先做哪个；
- 帮老师拿材料。

适合：

- 年龄较大；
- 对控制感需求高；
- 已经能理解规则。

### 7. 代币强化

代币本身不是最终奖励，而是“积分”。

例子：

- 星星；
- 小贴纸；
- 印章；
- 积分；
- 笑脸；
- 拼图块；
- 小火车车厢。

孩子集齐一定数量后，兑换真正想要的东西。

## 五、怎么找强化物：偏好评估SOP

一句大白话：

> 不要猜孩子喜欢什么，要让孩子用行动告诉你。

### 方法一：自由观察法

适合刚接触孩子时。

步骤：

1. 准备5–8种物品；
2. 摆在孩子看得到的地方；
3. 不急着指导；
4. 观察孩子主动拿什么；
5. 记录玩多久、是否重复选择、被拿走是否想要。

记录表：

| 物品/活动 | 是否主动选择 | 持续时间 | 被拿走后的反应 | 强化潜力1–5 |
|---|---|---|---|---|
| 泡泡 |  |  |  |  |
| 小车 |  |  |  |  |
| 球 |  |  |  |  |
| 积木 |  |  |  |  |
| 音乐玩具 |  |  |  |  |
| 零食 |  |  |  |  |

### 方法二：二选一法

适合孩子能做选择时。

步骤：

1. 左右手各拿一个物品；
2. 说：“你要哪个？”
3. 孩子选一个；
4. 让他玩10–20秒；
5. 换组合继续测试；
6. 记录选择频率。

注意：

- 左右位置要换，避免孩子只选某一边；
- 不要用语言暗示；
- 每次只给短时间，保持动机。

### 方法三：多选法

适合能力较好的孩子。

步骤：

1. 摆出5个物品；
2. 让孩子选一个；
3. 选中后短暂使用；
4. 拿走选过的，再让他从剩下的选；
5. 排出偏好顺序。

### 方法四：家长访谈法

问家长：

```text
孩子在家最喜欢什么？
什么东西被拿走时最舍不得？
什么活动可以让他安静下来？
什么东西能让他愿意配合？
哪些东西以前喜欢，现在不喜欢了？
```

注意：  
家长说喜欢，不一定在课堂有效。还要现场验证。

## 六、强化物使用的黄金原则

### 原则1：及时

孩子做对后，最好1秒内强化。

错误：

```text
孩子看老师一眼。
老师过了10秒才说“真棒”。
```

孩子可能不知道自己刚刚哪里做对了。

正确：

```text
孩子一看老师，老师马上说：“你看老师了！”并吹泡泡。
```

### 原则2：具体

不要只说“真棒”。  
要说清楚棒在哪里。

例子：

- “你坐好了。”
- “你说‘要’了。”
- “你等了10秒。”
- “你没有抢，用手指了。”
- “你把车给老师了。”

### 原则3：匹配

行为越难，强化越强。  
行为越简单，强化可以轻一点。

例子：

- 第一次说“妈妈”：强强化；
- 已经会做的配对：普通表扬即可；
- 忍住没打人，用休息卡：强强化；
- 坐好2秒：对初学者强强化，对高能力孩子可能只需表扬。

### 原则4：少量多次

尤其是食物和高动机玩具。

不要一次给一大把饼干。  
给一小口，保持学习机会。

### 原则5：先高频，后稀释

刚学新行为时，每次做对都强化。  
稳定后，逐渐变成隔几次强化一次。

### 原则6：强化目标行为，不强化问题行为

孩子哭闹时不要直接给最终奖励。  
等他出现哪怕一点合适行为，再给。

### 原则7：不断更新

强化物会失效。

今天喜欢泡泡，明天可能不喜欢。  
训练师要经常评估和轮换。

## 七、强化时老师怎么说、怎么看、怎么做

### 1. 眼神

老师不要像监考一样盯着孩子。  
要自然、温和、带期待。

孩子做对时：

- 眼睛看向孩子；
- 表情有变化；
- 身体微微靠近；
- 马上给强化。

### 2. 语气

语气要短、清楚、有情绪。

例子：

> “对，放进去了！”  
> “你说要了，给你！”  
> “你等到了，开车！”  
> “你用休息卡了，可以休息。”

不要机械地说：

> “很好。下一个。”

### 3. 手的位置

强化物要在老师可控范围内。  
不要一整盒都放到孩子面前，否则孩子容易抢或失去教学机会。

正确：

- 强化物放老师身边；
- 孩子看得到但拿不到；
- 做对后老师递给他；
- 用完再收回来或换下一轮。

### 4. 身体距离

如果孩子喜欢互动，老师可以靠近。  
如果孩子敏感，老师要保持距离，避免强化变成压力。

## 八、不同学习阶段怎么强化

### 阶段1：刚建立行为

特点：

- 孩子不会；
- 需要很多提示；
- 容易逃避；
- 动机不稳定。

强化策略：

- 每次正确都强化；
- 强化要强；
- 任务要短；
- 成功率要高；
- 以食物、玩具、活动为主。

例子：

> 孩子第一次递“要”卡，马上给最喜欢的泡泡。

### 阶段2：行为开始稳定

特点：

- 孩子在提示下能做；
- 偶尔能独立；
- 还需要动力维持。

强化策略：

- 独立完成给更强强化；
- 提示完成给普通强化；
- 开始加入社交表扬；
- 逐渐增加任务数量。

例子：

> 独立说“我要车”给玩30秒；提示后说，给玩10秒。

### 阶段3：行为已经掌握

特点：

- 大多数时候能独立；
- 可以换环境、换人；
- 不需要每次都给实物。

强化策略：

- 间歇强化；
- 自然强化；
- 社交强化；
- 代币强化；
- 泛化到生活。

例子：

> 孩子自己说“我要喝水”，自然结果就是喝到水，不一定再给额外奖励。

## 九、如何避免孩子只为零食学习

### 1. 零食不是不能用，而是不能只会用

食物强化在早期很有效，但如果长期只靠食物，可能出现：

- 没零食就不做；
- 只盯着吃；
- 正餐受影响；
- 家长反感；
- 学习变得机械。

### 2. 食物强化撤退路线

#### 第一步：食物+表扬

> “你说要了，太好了，给你小饼干。”

#### 第二步：食物+活动

> “你做完了，吃一小口，然后开车。”

#### 第三步：活动为主，食物偶尔

> “做完3个，玩泡泡。”

#### 第四步：自然结果

> “你说打开，老师就打开。”

### 3. 把食物变小、变少、变不固定

- 从一整块变成小碎块；
- 从每次给变成几次给；
- 从固定给变成随机惊喜；
- 同时增加玩具、活动、社交强化。

### 4. 用孩子本来想要的东西教学

如果孩子想开门，就教“开门”。  
如果孩子想玩水，就教“水”。  
如果孩子想继续动画，就教“还要”。

这叫自然强化。  
大白话：

> 学到的表达本身就能换来他想要的结果。

## 十、代币系统是什么？

一句大白话：

> 代币就是积分。孩子先赚星星，集齐星星后换真正想要的东西。

例如：

```text
坐好一次 → 得一颗星
完成5颗星 → 玩小车2分钟
```

代币适合教：

- 等待；
- 完成多个任务；
- 延迟满足；
- 课堂规则；
- 小组课参与；
- 自我管理。

代币不适合：

- 完全不理解交换的孩子；
- 当下动机很弱的孩子；
- 严重危机行为现场；
- 没有真正后备强化物时。

后备强化物大白话：

> 最后能拿星星去换的真正奖励。

---

## 十一、什么时候可以开始用代币？

孩子需要具备一些基础：

1. 能理解“先……后……”；
2. 能等几秒到几十秒；
3. 对最终奖励有兴趣；
4. 能看到星星数量变化；
5. 不会因为没马上拿到最终奖励而严重崩溃，或可通过很少数量开始。

如果孩子还不能等，先不要用5颗星。  
可以从1颗星开始：

> 做一个 → 得一颗 → 马上换。

再变成：

> 做两个 → 两颗星 → 换。

## 十二、代币板怎么做？

### 1. 最简单的代币板

```text
我要换：小车

[☆] [☆] [☆]

集满3颗星 → 玩小车2分钟
```

### 2. 低龄儿童版

用孩子喜欢的主题：

- 火车车厢；
- 小鱼进鱼缸；
- 小苹果贴树上；
- 小恐龙回家；
- 拼图块；
- 奥特曼能量格。

### 3. 大龄儿童版

可以用：

- 积分表；
- 任务清单；
- 打勾表；
- 等级表；
- 每日积分兑换表。

### 4. 代币板必须清楚写明

| 内容 | 示例 |
|---|---|
| 赚什么 | 星星 |
| 怎么赚 | 坐好、回答、完成任务 |
| 赚几个 | 3颗或5颗 |
| 换什么 | 玩小车2分钟 |
| 什么时候换 | 集满马上换 |

## 十三、代币系统实施SOP

### 第一步：选后备强化物

问孩子或观察孩子想要什么。

例如：

- 小车；
- 泡泡；
- 贴纸；
- 休息；
- 动画30秒；
- 跳床；
- 选择游戏。

### 第二步：确定代币数量

初学者：1–3个。  
中级：3–5个。  
高级：5–10个或积分制。

原则：

> 孩子第一次用代币，必须很快成功。

### 第三步：明确目标行为

不要说：

> “表现好就给星星。”

要说：

> “坐好给星星。”  
> “回答给星星。”  
> “放进去给星星。”  
> “安静等给星星。”

### 第四步：立刻给代币

孩子做对后，马上贴星星，并说清楚：

> “你坐好了，一颗星。”

### 第五步：集满马上兑换

不要拖。

> “三颗星满了，换小车。”

### 第六步：重新开始下一轮

兑换完，把星星拿掉，开始下一轮。

## 十四、代币系统的完整示例

### 示例1：桌面任务

目标：完成5个配对任务。

```text
每完成1个配对 → 得1颗星
集满5颗星 → 玩泡泡1分钟
```

老师语言：

> “放一样的。”  
> “对，一颗星。”  
> “五颗星满了，泡泡！”

### 示例2：等待训练

目标：等待30秒。

```text
每等10秒 → 得1颗星
3颗星 → 打开音乐玩具
```

老师语言：

> “等一下。”  
> “你等了，星星。”  
> “三颗星，音乐开始。”

### 示例3：小组课规则

目标：举手回答、不抢话、坐椅子。

```text
举手一次 → 1分
坐好3分钟 → 1分
轮到别人时安静听 → 1分
满5分 → 选择一个游戏
```

### 示例4：家庭作业

目标：完成晚间流程。

```text
洗手 → 1星
刷牙 → 1星
换睡衣 → 1星
整理书包 → 1星
上床 → 1星
5星 → 睡前故事一本
```

## 十五、代币系统常见错误

### 错误1：孩子不懂代币就直接用

如果孩子不知道星星能换东西，星星就没有意义。

解决：

- 先用1颗星马上换；
- 老师示范；
- 让孩子亲手贴、亲手换。

### 错误2：代币数量太多

孩子刚开始就要10颗星，等不到就崩溃。

解决：

- 从1颗、2颗、3颗开始；
- 成功后再加。

### 错误3：目标行为太模糊

“乖一点”“表现好”太抽象。

解决：

写具体：

- 坐椅子；
- 手放好；
- 回答问题；
- 走路不跑；
- 用卡片请求。

### 错误4：集满后不兑现

如果孩子集满星星却不能换，代币系统会失去信用。

解决：

- 说到做到；
- 如果奖励不可用，一开始就不要写；
- 换奖励前准备好。

### 错误5：用代币威胁孩子

错误：

> “你再哭，我把星星撕掉。”

这会让代币变成惩罚系统，孩子更焦虑。

更好的做法：

> “手放好可以继续赚星星。”

### 错误6：只扣不奖

有些老师把代币板变成扣分板。  
新人不建议用扣代币，尤其是低龄和情绪不稳的孩子。

### 错误7：长期不撤

孩子已经会了，还每次给星星，会形成依赖。

解决：

- 逐渐增加要求；
- 改成随机给；
- 换成自然强化；
- 引入自我评价。

---

## 十六、代币如何逐步撤销？

撤销不是突然取消。

大白话：

> 不要一下把拐杖抽走，要慢慢让孩子自己走。

### 方法1：增加兑换数量

从：

```text
3颗星换奖励
```

变成：

```text
5颗星换奖励
```

再变成：

```text
完成一整项任务换奖励
```

### 方法2：增加每颗星对应的任务量

从：

```text
做1题得1星
```

变成：

```text
做2题得1星
```

再变成：

```text
做5题得1星
```

### 方法3：从实物奖励转为活动奖励

从零食换：

```text
饼干
```

转为：

```text
泡泡、休息、选游戏、当小老师
```

### 方法4：从外部评价转为自我管理

让大龄孩子自己打勾：

```text
我今天完成了：
[ ] 准时到教室
[ ] 完成作业
[ ] 没有打断别人
[ ] 用语言请求休息
```

### 方法5：改成自然结果

例如：

- 完成穿鞋 → 可以出门；
- 说“打开” → 门打开；
- 写完作业 → 可以收起来；
- 整理好书包 → 明天更容易找到东西。

## 十七、不同目标如何选强化

### 1. 教语言请求

最好的强化是孩子请求的东西本身。

孩子说“泡泡” → 吹泡泡。  
孩子说“水” → 给水。  
孩子说“开” → 打开。

这比额外给饼干更自然。

### 2. 教模仿

强化要快、强、有趣。

孩子模仿拍手 → 老师马上玩夸张拍手游戏。  
孩子模仿推车 → 小车马上冲出去。

### 3. 教等待

等待初期强化要非常快。

等1秒 → 给。  
等3秒 → 给。  
等5秒 → 给。

不要一开始等太久。

### 4. 教生活自理

用自然后果和活动强化。

- 穿好鞋 → 出门；
- 洗完手 → 吃点心；
- 刷完牙 → 选睡前书；
- 整理玩具 → 玩下一个活动。

### 5. 处理问题行为

强化替代行为。

- 不尖叫，递卡 → 得到帮助；
- 不打人，说“休息” → 得到休息；
- 不抢，问“可以玩吗” → 得到轮流机会。

## 十八、强化物饱和怎么办？

强化物饱和大白话：

> 孩子吃够了、玩腻了，这个奖励不香了。

表现：

- 不再看奖励；
- 拿到也不开心；
- 做任务动力下降；
- 开始逃避；
- 只想换别的。

解决方法：

1. 轮换强化物；
2. 每次给少量；
3. 限制训练外随意获得；
4. 引入新活动；
5. 让孩子选择；
6. 暂停使用一段时间；
7. 把强化物和社交互动结合。

例子：

泡泡不香了，可以换成：

- 火箭气球；
- 小风扇；
- 音乐按钮；
- 追逐游戏；
- 弹跳球。

## 十九、强化物被孩子抢怎么办？

### 预防

- 强化物放在透明盒里；
- 看得到，拿不到；
- 一次只拿一点；
- 先教“要”；
- 用图片表示可兑换。

### 现场处理

如果孩子抢：

1. 保持冷静；
2. 不和孩子拔河；
3. 用短句：“手放好。”
4. 等孩子松手或手停；
5. 提示：“说要/指要。”
6. 合适表达后给一小份。

不要：

- 大喊；
- 用力抢回来；
- 孩子抢成功后让他一直玩；
- 把强化物全部暴露在桌上。

## 二十、强化和问题行为的关系

很多问题行为其实是被无意强化出来的。

### 例子1：哭闹强化

```text
孩子哭 → 家长给手机 → 下次更容易哭
```

### 例子2：逃避强化

```text
孩子扔作业 → 老师取消作业 → 下次更容易扔
```

### 例子3：关注强化

```text
孩子尖叫 → 全家围过来 → 下次更容易尖叫
```

训练师要帮助家长看到：

> 不是家长故意教坏孩子，而是孩子的大脑很聪明，会记住哪个方法最快有效。

我们要做的是让更好的方法更快有效。

## 二十一、家长版解释话术

### 1. 解释强化不是贿赂

```text
我们不是用奖励收买孩子。
我们是在孩子做出正确行为后，马上告诉他：这样做有用。
如果孩子哭了我们给手机，那是强化哭。
如果孩子说“手机”后我们给1分钟，那是强化表达。
```

### 2. 解释为什么要小份量

```text
不是舍不得给，而是给太多会让孩子很快满足，后面就没有练习机会。
我们每次给一点点，是为了让他多练几次表达和等待。
```

### 3. 解释为什么要撤零食

```text
零食只是早期发动机。
等孩子学会后，我们会慢慢换成玩具、活动、表扬和自然结果。
最终目标不是让孩子为了吃而学，而是让他真正会表达、会参与。
```

### 4. 解释代币

```text
代币就像积分卡。
孩子先学会完成任务、等待、积累，再兑换喜欢的活动。
它可以帮助孩子从马上要，慢慢过渡到等一等、做完再拿。
```

## 二十五、新人训练师必须记住的10句话

1. 强化不是贿赂，时机决定性质。  
2. 孩子做对后立刻给，才叫强化。  
3. 不要猜孩子喜欢什么，要观察和测试。  
4. 强化物不是只有零食。  
5. 新行为刚开始要高频强化。  
6. 已经会的行为要逐步稀释强化。  
7. 代币是积分，不是威胁工具。  
8. 集满代币必须兑现。  
9. 问题行为常常是被无意强化出来的。  
10. 最终目标是自然强化：孩子因为表达、互动和完成任务本身获得真实好处。

## 二十六、最后给新人训练师的话

强化就像给孩子点亮一盏小灯。

孩子刚开始不知道哪条路能走，你用强化告诉他：

> 这条路是安全的；  
> 这条路大人能听懂；  
> 这条路比哭闹、抢夺、逃跑更有效。

好的训练师不是永远拿着奖励的人，  
而是帮助孩子慢慢发现：

> 我会表达，我能完成，我可以等待，我能参与真实生活。

当孩子不再只为奖励而做，而是因为理解、习惯和成就感而做，强化就真正完成了它的使命。
